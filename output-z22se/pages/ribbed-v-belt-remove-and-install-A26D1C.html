Ribbed V–belt, Remove and Install

  Remove

Mark the running direction on the ribbed V–belt. Use a 3/8" tool to slacken the ribbed V–belt tensioner pulley (anti–clockwise) and take off the ribbed V–belt.

  Install

Ribbed V–belt

Note the running direction and installation position of the ribbed V–belt.

  Adjust

Use a 3/8" tool to slacken the ribbed V–belt at the tensioner pulley.
