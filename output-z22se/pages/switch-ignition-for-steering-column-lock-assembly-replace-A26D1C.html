Switch, Ignition for Steering Column Lock Assembly, Replace

  Install

Driver's air bag from steering wheel – bolts.

Horn harness connectors.

Steering wheel is shown removed for clarity.

  Remove

Steering wheel from hub – four bolts.

Steering column upper and lower shrouds – screws.

Turn ignition key to position "I". Release lock cylinder with 3 mm pin remover then remove key and barrel in direction (A arrowed).

Caution !

Release of the harness plug from switch contact part is only possible if the lock cylinder is at position "I".

  Remove

Harness plug from switch contact part, guide 3 mm pin into release aperture (B, 1) to compress lug (C 4) carefully levering off harness plug with second screwdriver (B, 2) and remove.

Set screw from switch contact part (C, 3). Then pull switch contact part from steering and ignition lock (D, arrowed).

  Install

Insert lock cylinder – if steering lock blocking pin has engaged, press lock stone downwards with screwdriver until retaining pin engages; ensure that lock cylinder still in position "I".

Turn ignition switch to align recess (1) to be opposite lug (2), connect harness plug to switch contact part and insert into steering column lock and fasten with set screws.

Steering column upper and lower shrouds – screws.

Switch wheel to hub – four bolts.

Horn harness connectors.
