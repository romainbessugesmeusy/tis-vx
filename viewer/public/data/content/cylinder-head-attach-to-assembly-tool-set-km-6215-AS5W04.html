Cylinder head, attach to assembly tool set KM-6215

Special service tools required:

Assembly tool set KM-6215, mount KM-113-2

Note:   The cylinder head can only be attached to assembly device KM-6215-1 (1) once all attaching parts have been removed.

  Install

Position cylinder head with timing side facing mount and attach at centre threaded hole (2) with fastening bolts. The lengths of the fastening bolts can be found in the Survey. For Z 22 SE, the second bore must be used.

The threaded hole for the rear fastening bolt for the cylinder head is determined by the length of the cylinder head.
