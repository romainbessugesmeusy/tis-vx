Valve, measure

Special service tools required:

Caliper gauge, micrometer screw

Note:   The engine-specific values can be found in the corresponding "Technical Data".

  Remove

Preconditions: All valves are removed and cleaned.

Ensure that there are no crater-like burns on the valve cone.

  Measure

Determine the valve stem diameter using a micrometer screw. Valve length is measured using a caliper gauge.
