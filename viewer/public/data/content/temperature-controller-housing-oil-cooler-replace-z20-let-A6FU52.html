Temperature Controller Housing, Oil Cooler, Replace (Z20 LET)

  Remove

Important:   A four post hoist must be used to raise the vehicle when removing the engine undertray. The vehicle can be moved to a two post hoist for other operations if required.
1.   Remove engine undertray (1)
•   Raise vehicle
•   Remove 13x bolt, 13x washer



Important:   Observe relevant environmental regulations when disposing of used coolant and lubricants.
2.   Remove oil filter
•   Install Oil Filter Tool KM-726-A
•   Screw fit
– Discard oil filter
3.   Release oil cooler feed and return pipe from oil filter housing
•   Remove banjo bolt from feed pipe (1)
•   Remove banjo bolt from return pipe (2)
•   Remove, then discard 4x sealing washer
•   Remove clip



4.   Remove temperature controller from oil cooler housing
•   Remove Plug, Spring and Piston, then discard Sealing Washer
Note:   Note position for installation.
– Position collecting container
5.   Remove temperature controller housing
•   Remove adapter nut (1) for oil filter, use socket KM-943
•   Remove housing bolt (2)
6.   Remove, discard O ring seal



  Install

7.   Install temperature controller housing
•   Install housing bolt 23 Nm
•   Install new O ring seal
– Lubricate new seal
•   Install adapter nut for oil filter 23 Nm
•   Use Socket KM-943
8.   Secure oil cooler feed and return pipe to oil filter housing
•   Install banjo bolt, feed pipe to temperature controller housing 30 Nm
•   Install banjo bolt, return pipe to temperature controller housing 30 Nm
•   Install 4x new sealing washer
•   Install clip
9.   Install new oil filter
•   Install oil filter tool KM-726-A
•   Screw fit 15 Nm
– Lubricate O ring seal
– Remove collecting container
10.   Install temperature controller to oil cooler housing
•   Install Piston, Spring, new Sealing Washer and Plug 20 Nm + 60° + 30°
Note:   Note position for installation.
11.   Install engine undertray
•   Install 13x bolt, 13x washer
12.   Lower vehicle
13.   Run engine and check for leaks, stop engine
•   Check engine oil level, top up if required
