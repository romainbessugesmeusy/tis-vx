Master Cylinder, Remove and Install

Open bonnet.

  Remove

Front bonnet – see operation "Front Bonnet, Remove and Install" in group A.


  Install

Hose clamps to brake fluid supply hoses (1).

  Remove

Brake fluid supply hoses (1) from master cylinder.


Caution !

Use absorbent material to soak up brake fluid.

Note position of supply hoses to master cylinder.

  Install

Blanking plugs to master cylinder.

  Remove

Brake lines (1) from master cylinder.

Caution !

Use absorbent material to soak up brake fluid.

  Install

Blanking plugs to master cylinder and brake lines.


  Remove

Master cylinder from brake servo – 2 nuts (1).

  Install

Master cylinder to brake servo – 2 nuts (1); use new seal.


Blanking plugs from master cylinder and brake lines.

  Install

Brake lines (1) to master cylinder.


  Remove

Blanking plugs from master cylinder.

  Install

Brake hoses (1) to master cylinder.

  Remove

Hose clamps from brake fluid supply hoses (1).


Bleed brake circuit – see operation "Brake System, Bleed".

Check brake system for leaks – see operation "Brake System, Inspect for Leaks".


  Install

Front bonnet – see operation "Front Bonnet, Remove and Install" in group A.

Close bonnet.
