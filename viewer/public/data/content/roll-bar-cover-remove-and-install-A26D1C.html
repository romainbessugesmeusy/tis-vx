Roll Bar Cover, Remove and Install

  Remove

Soft top or hard top – see Owner's Manual.

Roll bar cover (1) – 3 screws and 3 spacers (2), 3 nuts and washers.

Important:   Note arrangement of all washers.

Do not lose spacers (2).

Do not drop washers into engine compartment.

Weatherstrip from roll bar cover.

  Install

Roll bar cover (1) – 3 screws and 3 spacers (2), 3 nuts and washers.

Weatherstrip to roll bar cover.

Important:   Install washers in arrangement noted on removal.

  Install

Soft top or hard top – see Owner's Manual.
