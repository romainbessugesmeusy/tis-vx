Side Pod, Left Side, Replace (Z 20 LET)

  Remove

1.   Raise vehicle
2.   Remove rear road wheel from left side
•   Remove security bolt
– Use special socket located under front bonnet
•   Remove 4x wheel bolt
3.   Remove wheel arch liner from left side
•   Remove 4x screw, 2x clip
4.   Release side pod from wheel arch
•   Remove, then discard 2x nut (1)
•   Remove 2x washer



5.   Remove side pod from vehicle
•   Remove 2x screw (1)



  Install

6.   Install grille (1) to new side pod
•   Use suitable bonding adhesive



7.   Install side pod to vehicle
•   Install 2x screw (1)



8.   Secure side pod to wheel arch
•   Install 2x washer
•   Install 2x new nut (1)



9.   Install wheel arch liner to left side
•   Install 4x screw, 2x clip
10.   Install rear road wheel to left side
•   Install, then hand tighten 4x bolt, security bolt
11.   Lower vehicle
12.   Secure rear road wheel to left side
•   Secure 4x bolt 90 Nm
•   Secure locking bolt 90 Nm
– Use special socket located under front bonnet
