Oil Feed Pipe, Intermediate, Turbocharger, Replace (Z20 LET)

  Remove

Important:   A four-post hoist must be used to raise the vehicle when removing the engine undertray. The vehicle can be moved to a two-post hoist for other operations if required.
1.   Remove engine undertray
•   Raise vehicle
•   Remove 13x bolt, 13x washer
2.   Remove hose (3) from turbocharger (2) and intercooler (1)
•   Release 2x clip
•   Remove, then discard cable tie



Important:   Observe relevant environmental requirements when disposing of used oil.
3.   Disconnect intermediate pipe (1) from oil feed pipe for turbocharger (3)
•   Position suitable collecting container
•   Disconnect pipe union (2)



4.   Release intermediate pipe (1) from engine cylinder block
•   Remove bolt (2)



5.   Remove intermediate pipe (1) from threaded adaptor on engine cylinder block
•   Disconnect pipe union (2)



  Install

6.   Install new intermediate pipe to threaded adaptor on engine cylinder block
•   Connect pipe union 35 Nm
7.   Secure intermediate pipe to engine cylinder block
•   Install bolt 35 Nm
8.   Connect intermediate pipe to oil feed pipe for turbocharger
•   Connect pipe union 30 Nm
9.   Install hose to turbocharger and intercooler
•   Install new cable tie
•   Secure 2x clip
10.   Install engine undertray
•   Install 13x bolt, 13x washer
11.   Lower vehicle
12.   Check engine oil level and correct as necessary
