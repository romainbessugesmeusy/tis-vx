Camshafts, Replace (Z20 LET)

  Remove

Important:   A four-post hoist must be used to raise the vehicle when removing the engine undertray. The vehicle can be moved to a two-post hoist for other operations if required.
1.   Remove engine undertray
•   Raise vehicle
•   Remove 13x bolt, 13x washer
2.   Lower vehicle
3.   Disconnect battery negative lead
•   Release windscreen wash reservoir, then move to one side
•   Release nut, bolt
4.   Disconnect intercooler hose (1) from connecting pipe for throttle body
•   Release clip (2)
•   Pull off



5.   Raise vehicle to suitable working height
6.   Remove rear road wheel from right side
•   Remove security bolt
– Use special socket located under front bonnet
•   Remove 4x wheel bolt
7.   Remove wheel arch liner, right rear
•   Remove 4x screw, 2x clip
8.   Remove intercooler hose (1) from intercooler
•   Release clip (2)
•   Release fuel vapour line from intercooler hose (1)
– Remove, then discard 4x cable tie (3)



9.   Remove upper front cover (2) for timing belt
•   Remove 2x bolt (1)
•   Pull off



10.   Remove ribbed V-belt
•   Release ribbed V-belt tension
– Using suitable spanner rotate tensioner (1) clockwise, then Install suitable locking pin (2)



11.   Remove crankshaft pulley
•   Apply match marks to crankshaft pulley and crankshaft
•   Remove 4x bolt
– Counter-hold centre bolt on crankshaft sprocket
12.   Remove ribbed V-belt tensioner
•   Remove 2x bolt
13.   Remove lower cover for timing belt
•   Remove 2x bolt (1)



14.   Rotate crankshaft in normal direction of rotation to Top Dead Centre (TDC) on cylinder no. 1, until timing marks (2) and (1) are aligned
15.   Lock camshafts in position
•   Use Locking Tool KM-853
16.   Release tension from timing belt
•   Release bolt, then turn adjusting eccentric clockwise until pointer is located just before left stop
17.   Remove timing belt from pulleys
•   Position timing belt to one side



18.   Lower vehicle
19.   Remove air intake hose with engine breather hose
•   Disconnect air intake hose (1) from mass air flow meter (2) and turbocharger (3)
– Release 2x clip
•   Disconnect engine breather hose from cylinder head cover
– Release clip
– Remove, then discard cable tie



20.   Remove heat shield (1) from support bracket (2) on cylinder head
•   Remove 4x bolt, 4x washer



21.   Remove support bracket (1) for heat shield from cylinder head
•   Remove bolt, 2x washer
•   Remove engine lifting eye (2)
– Remove bolt



22.   Disconnect harness connector (1) from ignition module
23.   Remove ignition module cover (2) from cylinder head
•   Slide off



24.   Remove ignition module from cylinder head
•   Remove 2x bolt (1)



25.   Disconnect harness connector from camshaft sensor (1)
26.   Release wiring harness from cylinder head cover
•   Release from 3x clip
27.   Disconnect 2x breather hose (2) from cylinder head cover (3)
28.   Remove cylinder head cover (3)
•   Remove 10x bolt
•   Remove, then discard 10x 'O' ring
•   Remove, then discard gasket



29.   Remove 2x camshaft pulley (2) and (3) from camshaft
Note:   Note position of locating pin on exhaust pulley.
•   Remove 2x bolt (1), 2x washer
– Counter-hold camshaft at hexagon with open end of spanner (4)
– Second technician required
– Discard 2x bolt



Important:   Note position of bearing caps before disassembling. Ensure bearing caps are released from bearing seats evenly in stages of 1/2 to 1 turn in spiral pattern starting outwards to inwards.
30.   Remove intake camshaft (1)
•   Remove 5x bearing cap (2)
– Remove 10x bolt
•   Remove, then discard front oil seal



Important:   Note position of bearing caps before disassembling. Ensure bearing caps are released from bearing seats evenly in stages of 1/2 to 1 turn in spiral pattern starting outwards to inwards.
31.   Remove exhaust camshaft (1)
•   Remove 5x bearing cap (2)
– Remove 10x bolt
•   Remove, then discard front oil seal



  Install

32.   Install new oil seal to new exhaust camshaft
•   Lubricate new oil seal
Important:   Do not apply sealant to oil duct area. Lubricate journals and lobes of camshaft with lubricating paste. Install bearing caps to previously marked position. Ensure bearing caps are secured to bearing seats evenly in stages of 1/2 to 1 turn in spiral pattern starting inwards to outwards.
33.   Install new exhaust camshaft to cylinder head
•   Install 5x bearing cap
•   Apply sealant to sealing surface of guide bearing (arrowed)
– Install 10x bolt 8 Nm
34.   Install new oil seal to new intake camshaft
•   Lubricate new oil seal
Important:   Lubricate journals and lobes of camshaft with lubricating paste. Install bearing caps to previously marked position. Ensure bearing caps are secured to bearing seats evenly in stages of 1/2 to 1 turn in spiral pattern starting inwards to outwards.
35.   Install new intake camshaft to cylinder head
•   Install 5x bearing cap
•   Apply sealant to sealing surface of guide bearing (arrowed)
– Install 10x bolt 8 Nm



36.   Install 2x camshaft pulley to 2x camshaft
Note:   Note position of locating pin on exhaust pulley.
•   Install 2x new bolt, 2x washer 50 Nm + 60° + 15°
– Counter-hold camshaft at hexagon with open end of spanner
– Second technician required
37.   Install cylinder head cover to cylinder head
•   Install 10 x new 'O' ring, new gasket
•   Install 10x bolt 8 Nm
38.   Reconnect 2x breather hose to cylinder head cover
39.   Secure wiring harness to cylinder head cover
•   Secure to 3x clip
40.   Reconnect harness connector to camshaft sensor
41.   Install ignition module to cylinder head
•   Install 2x bolt 8 Nm
42.   Install ignition module cover to cylinder head
•   Slide on
43.   Reconnect harness connector to ignition module
44.   Install support bracket for heat shield to cylinder head
•   Install bolt, 2x washer
•   Install engine lifting eye (2)
– Install bolt
45.   Install heat shield to support bracket on cylinder head
•   Install 4x bolt, 4x washer
46.   Install air intake hose with engine breather hose
•   Reconnect air intake hose to mass air flow meter and turbocharger
– Secure 2x clip
•   Reconnect engine breather hose to cylinder head cover
– Secure clip
– Install new cable tie
47.   Raise vehicle to suitable working height
48.   Check timing marks are aligned
49.   Lock camshafts in position
•   Use Locking Tool KM-853
50.   Install timing belt to pulleys
51.   Pre-tension timing belt
•   Turn eccentric of tensioner in direction of arrow (anti-clockwise) until pointer is located just before right stop, then secure tensioner bolt
•   Remove Locking Tool KM-853 from camshafts
•   Rotate crankshaft two complete turns in normal direction of rotation to Top Dead Centre (TDC) on cylinder no. 1, until timing marks (ll) and (lll) are aligned
•   Install Locking Tool KM-853 to camshafts



52.   Tension timing belt
•   Release tensioner bolt, then turn eccentric of tensioner in direction of arrow (clockwise) until pointer is located in correct position
Note:   For new belt, pointer (1) for timing belt tensioner aligns with notched mark (3).
Note:   For used belt, pointer (1) for timing belt tensioner aligns with notched mark (2).
53.   Secure tensioner
•   Secure bolt 20 Nm
•   Remove Locking Tool KM-853
54.   Rotate crankshaft two complete turns in normal direction of rotation to (TDC) on cylinder no. 1, until timing marks are aligned. If pointer of tensioner does not align with corresponding notch mark, then repeat adjustment procedure



55.   Install lower cover of camshaft timing belt
•   Install 2x bolt 6 Nm
56.   Install ribbed V-belt tensioner
•   Install 2x bolt 25 Nm
57.   Install crankshaft pulley
•   Align match marks
•   Install 4x bolt 20 Nm
– Counter-hold centre nut on crankshaft sprocket
58.   Install ribbed V-belt
•   Use suitable spanner to rotate tensioner clockwise, then remove locking pin
59.   Install upper cover of camshaft timing belt
•   Push on to secure
•   Install 2x bolt 6 Nm
60.   Install intercooler hose to intercooler
•   Secure clip
•   Install 4x new cable tie
61.   Install liner to wheel arch
•   Install 4x screw, 2x clip
62.   Install rear road wheel to right side
•   Install 4x bolt, security bolt
•   Do not tighten
63.   Lower vehicle
64.   Secure rear road wheel right side
•   Tighten 4x bolt, security bolt 90 Nm
65.   Install intercooler hose to throttle body
•   Secure clip
66.   Reconnect battery negative lead
•   Secure nut and bolt
67.   Install windscreen wash reservoir
•   Push fit
68.   Program all volatile memories
69.   Raise vehicle
70.   Install engine undertray
•   Install 13x bolt, 13x washer
71.   Lower vehicle
