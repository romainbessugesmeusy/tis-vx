Fuel Injectors, Replace (Z20LET)

  Remove

1.   Release screen wash reservoir from support bracket
•   Slide off
– Position screen wash reservoir to one side



2.   Disconnect battery negative lead
•   Release nut and bolt



Important:   Fuel may escape – observe safety regulations and relevant national legislation.
3.   Release fuel pressure from fuel system
•   Remove protective cap
•   Connect test connection hose (2) of Pressure Tester KM-J-34730-91 (1) to fuel rail
•   Release fuel pressure using Pressure Tester KM-J-34730-91 (1)
– Collect any escaping fuel in a suitable container
4.   Remove Pressure Tester KM-J-34730-91 (1)
•   Disconnect test connection hose (2) of Pressure Tester KM-J-34730-91 (1) from fuel rail
•   Install protective cap



5.   Disconnect breather hose (1) from cylinder head cover
•   Release clip (2)
6.   Release breather hose (1) from lifting eye
•   Remove, then discard cable tie (3)



7.   Disconnect vacuum hose (1) from cylinder head cover
•   Release clip (2)



8.   Disconnect vacuum hose (1) from fuel pressure regulator (2)
•   pull off



9.   Disconnect harness connector (1) from throttle position sensor
•   Release integral clip



10.   Release fuel rail assembly (3) from intake manifold
•   Remove 2x bolt (1)
11.   Remove cover (2) from housing for injector harness
•   Release 10x integral clip



12.   Remove 4x injector (2) from fuel rail assembly
•   Remove 4x clip (1)
•   Remove, then discard 8x O ring



  Install

13.   Install 8x new O ring to 4x injector
14.   Install 4x injector to fuel rail assembly
•   Install 4x clip
15.   Secure fuel rail assembly to intake manifold
•   Install, then secure 2x bolt 8 Nm
16.   Install cover to housing for injector harness
•   Secure 10x integral clip
17.   Connect breather hose to cylinder head cover
•   Secure clip
18.   Secure breather hose to lifting eye
•   Install new cable tie
19.   Connect harness connector to throttle position sensor
•   Secure integral clip
20.   Connect vacuum hose to fuel pressure regulator
•   push on
21.   Connect vacuum hose to cylinder head cover
•   Secure clip
22.   Connect battery negative lead
•   Secure nut and bolt
23.   Secure screen wash reservoir to support bracket
•   slide on
24.   program all volatile memories
