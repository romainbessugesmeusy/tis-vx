C-025 - Engine-Compression

Test

Work Order Description

Nominal Value
T001

Mechanic and/or Hydraulic Check
  • Ignition OFF
  • Engine oil temperature is greater than 80 °C (176 °F)
  • Disconnect wiring harness connector from:
    A5 Control Unit - Engine
  • Remove following component:
    Spark plugs
  • Accelerator pedal actuated to full load stop
  • Verify mechanical system functions/components
    Engine-compression

Test okay?
Yes: E001 No: E002


Result

Cause Of Fault
E001 Result: Defective Component
  • Defective component:
    A5 Control Unit - Engine

or

  • Check mechanical functionality of the following components and all attached parts:
    Ignition system, intake manifold, injection valve, combustion chamber (carbon deposit)

Important:

Reset concerned control unit (engine or immobiliser control unit) with diagnostic tester before replacing. Select immobiliser in the diagnostic tester and call up the corresponding test in the menu ADDITIONAL FUNCTIONS. Ensure that both control units are never reset and replaced at the same time.
Repair done

  • Check mechanical functionality of the following components and all attached parts:
    Cylinder-head gasket, induction valve, discharge valve, piston ring, cylinder head
