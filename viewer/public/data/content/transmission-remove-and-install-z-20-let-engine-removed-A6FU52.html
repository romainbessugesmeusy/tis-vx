Transmission, Remove and Install (Z 20 LET) (Engine Removed)

  Remove

1.   Remove turbocharger heat shield (1) from engine and transmission assembly
•   Remove 2x bolt
Important:   When removing transmission observe relevant safety measures and national legislation.
2.   Remove transmission (2) from engine assembly
•   Remove 11x bolt
Note:   Includes removal of support bracket for engine harness.



  Install

Important:   When installing transmission observe safety measures and national legislation.
3.   Install transmission to engine assembly
•   Install 7x bolt (1) 60 Nm symmetrically
Note:   Includes installation of support bracket for engine harness.
•   Install 4x bolt (2) 40 Nm symmetrically



4.   Install turbocharger heat shield to engine and transmission assembly
•   Install 2x bolt
5.   Check oil level and top-up if required
