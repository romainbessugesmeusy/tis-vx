Air Cleaner Element, Replace (Z 20 LET)

  Remove

1.   Disconnect air intake hose from air cleaner housing
•   Release clip (1)
2.   Release air cleaner cover from air cleaner housing
•   Release 2x clip (2)



3.   Raise and support vehicle
4.   Remove rear road wheel from left side
•   Remove security bolt
– Use special socket located under front bonnet
•   Remove 4x wheel bolt
5.   Remove liner from wheel arch
•   Remove 6x screw
6.   Remove air cleaner cover with air cleaner element
•   Remove 2x clip (1)



7.   Remove air cleaner element (1) from air cleaner cover



  Install

8.   Install air cleaner element (1) to air cleaner cover



9.   Install air cleaner cover with air cleaner element
•   Secure 2x clip (1)



10.   Install liner to wheel arch
•   Install 6x screw
11.   Install rear road wheel
•   Install 4x bolt, security bolt
•   Do not tighten
12.   Raise vehicle and remove support
13.   Lower vehicle
14.   Secure rear road wheel
•   Tighten 4x bolt, security bolt 90 Nm
15.   Secure air cleaner cover to air cleaner housing
•   Secure 2x clip (2)
16.   Connect air intake hose to air cleaner housing
•   Secure clip (1)
