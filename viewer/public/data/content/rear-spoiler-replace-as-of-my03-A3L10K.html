Rear Spoiler, Replace (As of MY03)

  Remove

1.   Release mat from load compartment
•   Release velcro fastening
– Position mat to one side
2.   Remove rear spoiler (1) from rear clamshell (2)
•   Remove 3x nut, 3x washer
– Discard 3x nut



  Install

3.   Install rear spoiler to rear clamshell
•   Install, then secure 3x new nut, 3x washer
4.   Secure mat to load compartment
•   Secure velcro fastening
