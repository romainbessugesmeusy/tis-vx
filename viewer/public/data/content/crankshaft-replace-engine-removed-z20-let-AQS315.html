Crankshaft, Replace (Engine Removed) (Z20 LET)

  Remove

Attach engine with Adapter KM-412-8-A to Engine Overhaul Stand KM-412 – remove necessary attaching parts.

Remove transmission from cylinder block.

Remove oil pump – see operation "Oil Pump, Replace".

Remove upper part of oil pan – see operation "Oil Pan, Upper, Replace".

Remove balancer shaft unit – see operation "Balancer Shaft Unit, Replace".

Lock flywheel with KM-652 (1) and remove from crankshaft.

Mark order of con-rod bearing caps (2) – bead (3) points to transmission side.

Remove con-rod bearing cap from con-rod.

Mark the order of the crankshaft bearing caps and remove from cylinder block.

Remove crankshaft from cylinder block.


  Clean

Clean all parts.

  Inspect

Check crankshaft - see operation "Crankshaft, Out-of-round, Check".

  Install

When replacing crankshaft: transfer crankshaft pulse pickup sensor disc (1) with new bolts - tightening torque 13 Nm

Coat new bearing shells with engine oil and insert in cylinder block and bearing caps. Carefully insert crankshaft in cylinder block. The seating of the crankshaft can be corrected by light blows on the crank webs with a rubber hammer. Apply bead of adhesive sealing compound (black) to grooves of the crankshaft rear main bearing cap. Install crankshaft and con-rod bearing caps - note identification.

Crankshaft bearing caps to cylinder block 50 Nm + 45° + 15°

Important:   Use new fastening bolts.

Note:   After assembly of rear main bearing cap, press in adhesive sealing compound (black) from above again until it escapes from the joints.

Con-rod bearing caps to con-rod 35 Nm + 45° + 15°

Important:   Use new fastening bolts.


  Install

Install rear crankshaft oil seal – see operation "Crankshaft Oil Seal, Flywheel Side, Replace".

Install balancer shaft unit – see operation "Balancer Shaft Unit, Replace".

Install upper part of oil pan – see operation "Oil Pan, Upper, Replace".

Install oil pump – see operation "Oil Pump, Replace".

Attach transmission to cylinder block – tightening torque 60 Nm
