Shock Absorber, Replace

  Remove

1.   Chock front road wheels
2.   Raise and support vehicle
3.   Remove rear road wheel
•   Remove 4x wheel bolt, security bolt
– Use special socket located under bonnet
4.   Release shock absorber from chassis
•   Remove bolt, nut, washer
Note:   Support rear suspension at lower wishbone



5.   Release shock absorber from lower wishbone
•   Remove bolt, nut, washer



6.   Remove shock absorber from vehicle
7.   Remove spring from shock absorber
•   Use spring compressor



  Install

8.   Install spring to new shock absorber
•   Use spring compressor



9.   Install shock absorber to vehicle
10.   Secure shock absorber to lower wishbone
•   Install bolt, washer, new nut 45 Nm
11.   Secure shock absorber to chassis
•   Install bolt, washer, new nut, 45 Nm
– Use new nylock nut
12.   Remove support from lower wishbone
13.   Install rear road wheel
•   Install 4x wheel bolt, security bolt 90 Nm
14.   Raise vehicle and remove support
15.   Lower vehicle
16.   Remove chocks from front road wheels
