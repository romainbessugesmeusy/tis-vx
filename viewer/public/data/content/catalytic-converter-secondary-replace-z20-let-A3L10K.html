Catalytic Converter, Secondary, Replace (Z20 LET)

  Remove

1.   Disconnect oxygen sensor for secondary catalytic converter
•   Disconnect harness connector (1)
•   Release integral clip
•   Lower harness through aperture in left rear inner wheel arch



Important:   A four-post hoist must be used to raise the vehicle when removing the engine undertray and exhaust pipe undertray. The vehicle can be moved to a two-post hoist for other operations if required.
2.   Remove engine undertray (1)
•   Raise vehicle
•   Remove 13x bolt, 13x washer
3.   Remove exhaust pipe undertray (2)
•   Remove 12x bolt, 12x washer, 2x clip



4.   Release harness for secondary catalytic converter
•   Release 2x clip from subframe
•   Release clip from rear silencer heatshield
5.   Remove secondary catalytic converter (1)
•   Remove 3x front nut (3), then discard
•   Remove 3x rear nut (2), then discard
– Release rubber mount bracket
•   Release rubber mount from bracket
•   Remove gasket (4) from front joint, then discard
Note:   Illustration shows exhaust removed for clarity.



6.   Remove oxygen sensor (2) from secondary catalytic converter (1)



  Install

7.   Install oxygen sensor to secondary catalytic converter
8.   Install secondary catalytic converter
•   Install new gasket to front joint
•   Install 3x new front nut
– Position bracket for rubber mount
•   Install 3x new rear nut
•   Secure rubber mount to bracket
9.   Install harness for oxygen sensor to rear silencer heatshield
•   Secure clip
•   Install harness for oxygen sensor to subframe
•   Secure 2x clip
•   Raise harness through aperture in left rear inner wheel arch
10.   Secure clamp for intermediate pipe
11.   Install engine undertray
•   Remove 13x bolt, 13x washer
12.   Install exhaust pipe undertray
•   Install 12x bolt, 12x washer, 2x clip
13.   Lower vehicle
14.   Connect oxygen sensor for secondary catalytic converter
•   Connect harness connector
•   Secure integral clip
15.   Program volatile memories
