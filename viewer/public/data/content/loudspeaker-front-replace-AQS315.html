Loudspeaker, Front, Replace

  Remove

1.   Remove tread plate (2)
•   Remove screw, 2x clip
2.   Remove trim panel (1) with loudspeaker
•   Remove 2x clip
•   Disconnect harness connector
– Release integral clip



3.   Release loudspeaker (1) from trim panel



  Install

4.   Secure new loudspeaker (1) to trim panel



5.   Install trim panel (1) with loudspeaker
•   Connect harness connector
– Secure integral clip
•   Install 2x clip
6.   Install tread plate (2)
•   Install screw, 2x clip
