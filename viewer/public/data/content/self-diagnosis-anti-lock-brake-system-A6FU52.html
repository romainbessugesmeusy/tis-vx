Self-diagnosis  

The control unit performs self checks and checks of the ABS system each time the ignition is switched on. The ABS telltale illuminates briefly to indicate these checks are in progress.  

If the ABS telltale does not illuminate when the ignition is switched on, or if the ABS telltale illuminates when the vehicle is driven, there is a fault in the system.  



Conventional braking becomes operative if the ABS malfunctions. The trouble codes can be read using TECH 2.
