Engine Damping Block, Front, Replace (Z20 LET)

  Remove

Important:   A four-post hoist must be used to raise the vehicle when removing the engine undertray. The vehicle can be moved to a two-post hoist for other operations if required.
1.   Remove engine undertray
•   Raise vehicle
•   Remove 13x bolt, 13x washer
Important:   Do not support on oil pan.
2.   Install engine support to engine and transmission assembly
3.   Remove engine damping block with support bracket from transmission and chassis
•   Remove centre bolt, nut (1), then discard
•   Remove 2x bolt (2)



4.   Remove engine damping block (2) from support bracket (3)
•   Apply match marks (1)
•   Use hydraulic press



  Install

5.   Install new engine damping block to support bracket
•   Align match marks
– Transfer match marks to new engine damping block
•   Use hydraulic press
6.   Install engine damping block with support bracket to transmission and chassis
•   Install centre bolt, nut
– Use new bolt, nut
– Do not tighten
•   Install 2x bolt, washer 49 Nm
– Apply locking compound
7.   Remove engine support from engine and transmission assembly
8.   Secure centre bolt, nut for engine damping block 86 Nm
9.   Install engine undertray
•   Install 13x bolt, 13x washer
10.   Lower vehicle
