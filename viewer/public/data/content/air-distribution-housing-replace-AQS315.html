Air Distribution Housing, Replace

  Remove

1.   Release windscreen wash reservoir from support bracket
•   Pull off, then move to one side
2.   Remove air intake pipe (1) from heater assembly and air distribution housing
•   Remove mesh grille (if installed) from air distribution housing
•   Pull off



3.   Remove wiper motor cover (1)
•   Remove 2x screw (2)



4.   Remove wiper motor from support bracket
•   Release connector (1) from wiper motor
•   Release linkage from wiper motor
Note:   Note position of linkage.
– Remove nut (2)
•   Remove 3x bolt (3)



5.   Remove wiper arm from linkage
Note:   Note position of wiper arm.
•   Pull up cover (1)
•   Remove nut (2)



6.   Remove support bracket for bonnet catch and anti-theft switch (1)
•   Remove 2x bolt (2), 4x washer, bracket, then move bonnet catch (3) to one side
– Apply match marks (4)
•   Disconnect harness from anti-theft switch (5)



7.   Release windscreen washer pipe from support bracket then move to one side
•   Release pipe (2) from T piece (3)
– Pull off
•   Release clip (1) from support bracket
– Pull off



8.   Remove support bracket (1) and wiper linkage from bulkhead
•   Remove 3x bolt, grommet



9.   Disconnect bowden cable (1) from lever on air distribution housing
•   Apply match mark (2)
•   Prise off clip (3)



10.   Remove air distribution housing from bulkhead
•   Remove screw (1)
•   Remove clip (2)
Note:   Clip (2) located inside vehicle under lower instrument panel.



  Install

11.   Install new air distribution housing to bulkhead
•   Install clip
•   Install screw
12.   Connect bowden cable to lever on air distribution housing
•   Align match mark
•   Push on clip
13.   Install support bracket and wiper linkage to bulkhead
•   Install 3x bolt, grommet
14.   Secure windscreen washer pipe to support bracket
•   Secure clip to support bracket
– Push on
•   Secure pipe to T piece
– Push on
15.   Install support bracket for bonnet catch and anti-theft switch
•   Install 2x bolt, 4x washer, bracket, bonnet catch
– Align match marks
•   Connect harness to anti-theft switch
16.   Install wiper arm to linkage
•   Install nut
•   Push on cover
17.   Install wiper motor to support bracket
•   Install 3x bolt
– Apply locking compound
•   Install linkage to wiper motor
– Install nut
•   Secure connector to wiper motor
18.   Install wiper motor cover
•   Install 2x screw
19.   Install air intake pipe to heater assembly and air distribution housing
•   Install mesh grille (if installed) to air distribution housing
•   Push on
20.   Secure windscreen wash reservoir to support bracket
•   Slide on
