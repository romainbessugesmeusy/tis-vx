Related Literature Units              

Engine Repair, Using Short Block (Z20 LET)

  Remove

Remove engine and transmission - see operation "Engine with Transmission, Remove and Install".

  Install

Attach Adapter KM-412-8-A to cylinder block - remove attaching parts as required.

Fit engine with transmission to Engine Overhaul Stand KM-412 .

  Remove

Remove transmission from cylinder block.

Remove all cable and hose connections from engine.

Remove all attaching parts from engine - see corresponding operations.

  Install

Emboss engine number in cylinder block (arrow) with numbers punch.

Drive centring sleeves (1) to stop in cylinder block with KM-427 .

Install all attaching parts on engine - see corresponding operations. Required tightening torques - see "Recommended Torque Values".

Attach all cable and hose connections to engine.

Attach transmission to cylinder block.

  Remove

Remove engine from Engine Overhaul Stand KM-412 .

Remove Adapter KM-412-8-A from cylinder block.

  Install

Install engine and transmission - see operation "Engine with Transmission, Remove and Install".
