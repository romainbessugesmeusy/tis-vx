Hydraulic Valve Lifters, Remove and Install (Z 20 LET)

  Remove

Remove camshafts - see operation "Camshaft, Replace (Z 20 LET)".

  Remove

Remove hydraulic valve lifters from cylinder head with KM-845 .

Lay hydraulic valve lifters aside in correct order, to ensure that the hydraulic valve lifters can be allocated to their valves on installation.


  Install

Coat hydraulic valve lifter with engine oil and insert in cylinder head in correct sequence.

Install camshafts - see operation "Camshaft, Replace (Z 20 LET)l".
