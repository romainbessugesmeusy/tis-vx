Fuel Evaporation Tank, Replace

1.   Chock front road wheels
2.   Remove security bolt from right rear road wheel
•   Use special socket located under front bonnet
3.   Release wash reservoir from bracket, then position to one side



4.   Disconnect battery negative lead
•   Release nut



5.   Raise and support vehicle
•   Support vehicle at furthest jacking point to rear
6.   Lower vehicle to midway position
7.   Remove rear road wheel from right side
•   Remove security bolt
– Use special socket located under front bonnet
•   Remove 4x wheel bolt
8.   Remove rear wheelarch liner, right side
•   Remove 6x screw



9.   Release 3x hose (1) from fuel evaporation tank
•   Release 2x clip (2)



10.   Remove fuel evaporation tank from bracket
•   Remove nut, washer, bolt
11.   Install new fuel evaporation tank to bracket
•   Install bolt, washer, nut



12.   Secure 3x hose to fuel evaporation tank
•   Secure 2x new clip
13.   Install rear wheelarch liner, right side
•   Install 6x screw
14.   Install rear wheel to right side
•   Install 4x bolt, security bolt
•   Do not tighten
15.   Remove support from vehicle and lower
16.   Tighten rear road wheel, right side
•   Tighten 4x bolt, security bolt 90 Nm
17.   Remove chock from front road wheels
18.   Connect battery negative lead
•   Secure nut
19.   Secure wash reservoir to bracket
20.   Program all volatile memories
