Instrument Panel, Replace

  Remove

1.   Remove soft top/hardtop
2.   Release electrical harness from wash pump
3.   Release wash reservoir from support bracket, then position to one side



4.   Disconnect battery
•   Disconnect negative lead
– Release nut



5.   Remove 2x plastic insert from fascia panel
6.   Remove radio from installation frame
•   Remove radio display fascia, then store in protective display case
•   Insert 2x extraction tool in radio
– Push each extraction tool outwards, then withdraw radio
•   Disconnect aerial connector and harness connector



7.   Remove installation frame for radio from instrument panel
•   Lightly press top and bottom of installation frame together



Important:   Observe all safety regulations for removal and storage of airbag unit
8.   Remove driver's air bag unit from steering wheel
•   Remove 2x screw
•   Release harness connector



9.   Disconnect horn switch from combination switch assembly
•   Release harness connector
10.   Remove steering wheel from steering column
•   Remove 4x bolt



11.   Remove upper shroud from steering column
•   Remove 6x screw (1)
12.   Remove lower shroud from steering column
•   Remove 3x screw (2), clip, harness connector
– Disconnect trip switch



13.   Remove instrument panel housing
•   Remove 2x bolt, 2x washer, harness connector
14.   Remove lighting switch assembly from instrument panel
•   Release 4x retainer (arrowed)
•   Disconnect 5x harness connector



15.   Remove top fascia from instrument panel
•   Disconnect harness connector from starter switch
•   Remove 4x screw
•   Release harness connector



16.   Disconnect 2x control cable from heater control unit
•   Release clamp
– Remove 2x screw
17.   Disconnect electrical harness from heater control unit
•   Release 4x electrical connector
18.   Release heater control unit from instrument panel
•   Remove 4x screw, 4x washer, 4x nut
19.   Release demist duct from top fascia
•   Remove 20x screw
20.   Remove 2x fascia panel from instrument panel
•   Release 2x clip
21.   Remove instrument centre panel extrusion
•   Remove 4x bolt, 4x washer

  Install

22.   Install instrument centre panel extrusion
•   Install 4x washer, 4x bolt
23.   Install 2x fascia panel to instrument panel
•   Secure 2x clip
24.   Connect demist duct to top fascia
•   Install 20x screw
25.   Install heater control unit to instrument panel
•   Install 4x washer, 4x screw, 4x nut
26.   Connect 2x control cable to heater control unit
•   Install clamp
– Tighten 2x screw
27.   Connect demist duct to top fascia
•   Install 20x screw
28.   Install top fascia to instrument panel
•   Install 4x screw
•   Install harness connector
•   Connect harness connector to starter switch
29.   Install lighting switch assembly to instrument panel
•   Connect 5x electrical connector
30.   Install instrument panel housing
31.   Install lower shroud to steering column
•   Connect trip switch
•   Install harness connector, clip, 3x screw
32.   Install upper shroud to steering column
•   Install 6x screw
33.   Install steering wheel to steering column
•   Tighten 4x bolt
34.   Connect horn switch to combination switch assembly
35.   Install driver's air bag unit to steering wheel
•   Connect harness connector to airbag unit
•   Install 2x screw
36.   Install installation frame for radio to instrument panel
37.   Install radio to installation frame
•   Connect aerial connector and harness connector
•   Install radio display fascia
38.   Install 2x plastic insert to fascia panel
39.   Connect battery
•   Connect negative lead
– Secure nut
40.   Secure wash reservoir to support bracket
41.   Connect harness connector to wash pump
42.   Reprogram all volatile memories
43.   Install soft top/hardtop
