Sensor, Engine Knock, Replace (Z 20 LET)

  Remove

1.   Disconnect battery negative lead
•   Release windscreen wash reservoir, then move to one side
•   Remove nut, bolt
2.   Release harness connector (1) for primary oxygen sensor from support bracket (2)
•   Release clip (5)
•   Remove cable tie, then discard
3.   Release support bracket (2), then move to one side
•   Remove 4x bolt
4.   Release harness connector (3) for knock sensor from support bracket
•   Slide off
•   Remove 2x cable tie, then discard
5.   Disconnect harness connector (4) from knock sensor
•   Release integral clip



Important:   A four-post hoist must be used to raise the vehicle when removing the engine undertray. The vehicle can be moved to a two-post hoist for other operations if required.
6.   Remove engine undertray
•   Raise vehicle
•   Remove 13x bolt, 13x washer
7.   Remove knock sensor (1) from engine cylinder block
•   Remove bolt (2)



  Install

8.   Install knock sensor to engine cylinder block 20 Nm
•   Install bolt
9.   Install engine undertray
•   Install 13x bolt, 13x washer
10.   Lower vehicle
11.   Connect harness connector to knock sensor
•   Secure integral clip
12.   Secure harness connector for knock sensor to support bracket
•   Install 2x new cable tie
•   Slide on
13.   Secure support bracket
•   Install 4x bolt
14.   Secure harness connector for primary oxygen sensor to support bracket
•   Install new cable tie
•   Secure clip
15.   Connect battery negative lead
•   Install nut, bolt
•   Secure windscreen wash reservoir
16.   Program all volatile memories
