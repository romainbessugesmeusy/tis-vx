Auxiliary Coolant Pump, Replace (Z 20 LET)

  Remove

1.   Remove cap from coolant expansion tank
•   Ensure engine is cold
Important:   A four-post hoist must be used to raise the vehicle when removing the engine undertray. The vehicle can be moved to a two-post hoist for other operations if required.
2.   Remove engine undertray
•   Raise vehicle
•   Remove 13x bolt, 13x washer



Important:   Observe relevant environmental regulations when disposing of used coolant and lubricants.
3.   Drain cooling system
•   Position collecting container
•   Disconnect lower coolant hose from radiator feed pipe
•   Release clip (1)
– Remove, then discard cable tie (2)



4.   Disconnect engine harness from auxiliary coolant pump
•   Disconnect harness connector
5.   Lower vehicle
6.   Disconnect hose (4) of distribution housing from auxiliary coolant pump
•   Release clip (3)
7.   Disconnect heater hose (1) from auxiliary coolant pump
•   Release clip (2)
8.   Remove auxiliary coolant pump from support bracket
•   Slide off



  Install

9.   Install auxiliary coolant pump to support bracket
•   Slide on
10.   Connect heater hose to auxiliary coolant pump
•   Secure clip
11.   Connect hose of distribution housing to auxiliary coolant pump
•   Secure clip
12.   Raise vehicle
13.   Connect engine harness to auxiliary coolant pump
•   Connect harness connector
14.   Connect lower coolant hose to radiator feed pipe
•   Install clip
•   Install new cable tie
15.   Install engine undertray
•   Install 13x bolt, 13x washer
16.   Lower vehicle
17.   Fill cooling system
18.   Apply pressure to cooling system 1.0 bar to 1.5 bar
•   Use commercially available tool
19.   Bleed coolant system
•   Open engine bleed valve (1)
•   Open radiator bleed valve (2), left side of radiator



20.   Release pressure from cooling system
21.   Install cap to coolant expansion tank
22.   Check coolant level
•   Top up if required
23.   Run engine until operating temperature is reached
24.   Check coolant level
•   Top up if required
