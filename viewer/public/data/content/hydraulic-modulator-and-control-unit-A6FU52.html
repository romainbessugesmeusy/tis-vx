Description of Components  

Hydraulic Modulator and Control Unit  

The hydraulic modulator and control unit is an assembly, which is mounted in an anti-vibration bracket at the services compartment. A g-sensor is located in the control unit.

The control unit can be replaced separately. The hydraulic modulator is supplied already filled.  

If a fault is recognised that will affect the integrity of the ABS, the control unit switches off the ABS, lights the ABS telltale and stores the appropriate trouble code.
