Switch, Hazard Warning Lamp Replace

1.   Open bonnet
2.   Remove windscreen wash reservoir from mounting bracket
3.   Disconnect battery negative lead
•   Remove nut
4.   Release switch panel assembly from instrument panel
•   Pull off



5.   Remove switch assembly from panel
•   Release 4x clip (arrowed)



6.   Remove front cover from hazard warning lamp switch
•   Pull off
7.   Remove hazard warning lamp switch from housing
•   Release 2x clip, release harness connector
8.   Install hazard warning lamp switch to housing
•   Secure 2x clip, secure harness connector
9.   Install front cover to hazard warning lamp switch
•   Push fit
10.   Install switch assembly to panel
•   Secure 4x clip
11.   Secure switch panel assembly to instrument panel
•   Push fit
12.   Connect battery negative lead
•   Install nut
13.   Install windscreen wash reservoir to mounting bracket
14.   Close bonnet
