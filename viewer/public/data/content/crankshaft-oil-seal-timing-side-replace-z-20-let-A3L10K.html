Crankshaft Oil Seal, Timing Side, Replace (Z 20 LET)

  Remove

Important:   A four-post hoist must be used to raise the vehicle when removing the engine undertray. The vehicle can be moved to a two-post hoist for other operations if required.
1.   Remove engine undertray
•   Raise vehicle
•   Remove 13x bolt, 13x washer
2.   Lower vehicle
3.   Disconnect battery negative lead
•   Release windscreen wash reservoir, then move to one side
•   Remove nut, bolt
4.   Disconnect intercooler hose (1) from throttle body (2)
•   Release clip



5.   Raise vehicle to suitable working height and support
6.   Remove rear road wheel from right side
•   Remove security bolt
– Use special socket located under front bonnet
•   Remove 4x wheel bolt
7.   Remove liner from wheel arch
•   Remove 6x screw
8.   Disconnect intercooler hose (1) from intercooler (2)
•   Release clip
•   Remove, then discard 4x cable tie



9.   Remove upper cover of camshaft timing belt
•   Remove 2x bolt (1)



10.   Remove ribbed V-belt
•   Release ribbed V-belt tension
– Using suitable spanner rotate tensioner (1) clockwise, then Install suitable locking pin (2)



11.   Remove crankshaft pulley
•   Apply match marks to crankshaft pulley and crankshaft
•   Remove 4x bolt, using spanner to counter-hold centre nut on crankshaft sprocket
12.   Remove ribbed V-belt tensioner
•   Remove 2x bolt
13.   Remove lower cover of camshaft timing belt
•   Remove 2x bolt (1)



14.   Rotate crankshaft clockwise to top dead centre (TDC) on cylinder No. 1, until 2x timing mark (3) and (1) are aligned
15.   Lock camshafts in position
•   Use Locking Tool KM-853 (2)
16.   Release tension from camshaft timing belt
•   Release bolt, then turn adjusting eccentric in direction of arrow (clockwise) until pointer (4) is located just before left stop
17.   Remove camshaft timing belt from pulleys, move to one side
•   Do not remove camshaft timing belt from vehicle



18.   Remove crankshaft sprocket (1) from crankshaft
•   Remove bolt
– Counter-hold using KM-662-C



19.   Remove retaining key (1) from crankshaft
•   Prise out using suitable tool
20.   Remove washer (2) from crankshaft
21.   Remove, then discard crankshaft oil seal (3) from crankshaft
•   Prise out using suitable tool



  Install

22.   Install new crankshaft oil seal to crankshaft
•   Use Assembly Sleeve KM-693
23.   Install washer to crankshaft
24.   Install retaining key to crankshaft
25.   Install crankshaft sprocket to crankshaft
•   Install bolt 130 Nm + 45°
– Counter-hold using KM-662-C
26.   Install camshaft timing belt to pulleys
•   Ensure timing marks are aligned
27.   Adjust tension of camshaft timing belt - see operation “Camshaft Timing Belt Tension, Check and Adjust (Z 20 LET)”
28.   Install lower cover of camshaft timing belt
•   Install 2x bolt 6 Nm
29.   Install ribbed V-belt tensioner
•   Install 2x bolt 25 Nm
30.   Install crankshaft pulley
•   Align match marks on crankshaft pulley and crankshaft
•   Install 4x bolt 20 Nm
– Counter-hold centre nut on crankshaft sprocket using spanner
31.   Install ribbed V-belt
•   Using suitable spanner rotate tensioner clockwise, then remove locking pin
32.   Install upper cover of camshaft timing belt
•   Install 2x bolt 6 Nm
33.   Install intercooler hose to intercooler
•   Secure clip
•   Install 4x new cable tie
34.   Install liner to wheel arch
•   Install 6x screw
35.   Install rear road wheel
•   Install 4x bolt, security bolt
•   Do not tighten
36.   Lower vehicle
37.   Secure rear road wheel
•   Tighten 4x bolt, security bolt 90 Nm
38.   Connect intercooler hose to throttle body
•   Secure clip
39.   Connect battery negative lead
•   Install nut, bolt
•   Secure windscreen wash reservoir
40.   Program all volatile memories
41.   Raise vehicle
42.   Install engine undertray
•   Install 13x bolt, 13x washer
43.   Lower vehicle
