Oxygen Sensor, Secondary, Replace (Z20 LET)

  Remove

1.   Disconnect oxygen sensor for secondary catalytic converter
•   Disconnect harness connector (1)
•   Release integral clip



2.   Raise vehicle
3.   Remove exhaust undertray (1)
•   Remove 12x bolt, 12x washer, 2x clip



4.   Release harness for secondary catalytic converter
•   Release 2x clip from subframe
•   Release clip from rear silencer heatshield
5.   Remove oxygen sensor (1) from secondary catalytic converter (2)
•   Use KM-6326



  Install

6.   Install oxygen sensor to secondary catalytic converter
•   Use KM-6326
7.   Install harness for oxygen sensor to rear silencer heatshield
•   Secure clip
•   Install harness for oxygen sensor to subframe
•   Secure 2x clip
8.   Install exhaust pipe undertray
•   Install 12x bolt, 12x washer, 2x clip
9.   Lower vehicle
10.   Connect oxygen sensor for secondary catalytic converter
•   Connect harness connector
•   Secure integral clip
11.   Program volatile memories
