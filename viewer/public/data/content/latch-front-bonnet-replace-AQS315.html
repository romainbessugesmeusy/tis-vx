Latch, Front Bonnet, Replace

  Remove

1.   Release bonnet latch from retaining bracket
•   Remove 2x bolt, 2x washer



2.   Remove bonnet latch
•   Release lock cable from bonnet latch
– Remove 2x clip



  Install

3.   Install new bonnet latch
•   Secure lock cable to bonnet latch
– Install 2x clip



4.   Secure bonnet latch to retaining bracket
•   Install 2x washer, 2x bolt



5.   Align front bonnet
