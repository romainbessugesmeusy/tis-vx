Liner, Front Wheelarch, Remove and Install  

  Remove, Disconnect   

Wheel - bolts.  

For liner on right side only, remove from front compartment:  

 

-  tyre inflator aerosol - release from clips  

-  fuse box - 2 screws  

-  relay box - 2 screws.  

 

Liner - 6 screws.  

  Install, Connect   

Liner - 6 screws.  

For liner on right side only, install to front compartment:  

 

-  relay box - 2 screws  

-  fuse box - 2 screws  

-  tyre inflator aerosol - engage in clips.  

 

Wheel - bolts 90 Nm.
