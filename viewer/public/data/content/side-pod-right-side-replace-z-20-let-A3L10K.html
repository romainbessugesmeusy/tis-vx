Side Pod, Right Side, Replace (Z 20 LET)

  Remove

1.   Release rear road wheel from right side
•   Release 4x bolt
•   Release security bolt
– Use special socket located under front bonnet
2.   Raise vehicle
3.   Remove rear road wheel from right side
•   Remove 4x bolt, security bolt
4.   Remove wheel arch liner from right side
•   Remove 4x screw, 2x clip
5.   Remove insulation pad (1) from wheel arch
•   Pull off



6.   Release side pod from wheel arch
•   Remove, then discard 2x nut (1)
•   Remove 2x washer



7.   Remove side pod from vehicle
•   Remove 2x screw (1)



  Install

8.   Install grille (1) to new side pod
•   Use suitable bonding adhesive



9.   Install side pod to vehicle
•   Install 2x screw (1)



10.   Secure side pod to wheel arch
•   Install 2x washer
•   Install 2x new nut (1)



11.   Install insulation pad (1) to wheel arch
•   Push on



12.   Install wheel arch liner to right side
•   Install 4x screw, 2x clip
13.   Install rear road wheel to right side
•   Install, then hand tighten 4x bolt, security bolt
14.   Lower vehicle
15.   Secure rear road wheel to right side
•   Secure 5x bolt 90 Nm
•   Secure locking bolt 90 Nm
– Use special socket located under front bonnet
