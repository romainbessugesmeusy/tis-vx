Front Spoiler, Replace (As of MYO3)

  Remove

1.   Raise vehicle
2.   Remove front spoiler (1) from clamshell
•   Remove, then discard 3x rivet (2)



  Install

3.   Install front spoiler to clamshell
•   Install 3x new rivet
4.   Lower vehicle
5.   Repeat operation for front spoiler on opposite side
