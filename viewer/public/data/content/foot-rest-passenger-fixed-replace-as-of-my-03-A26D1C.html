Foot Rest, Passenger, Fixed, Replace (as of MY 03)

  Remove

1.   Remove carpet mat, where fitted from footwell
•   Remove 2x screw (1)



2.   Remove foot rest (1) from foot well
•   Raise vehicle to suitable working height
•   Remove 4x bolt (2) from underside of vehicle



  Install

3.   Install new foot rest to foot well
•   Install 4x bolt to underside of vehicle
4.   Lower vehicle
5.   Install carpet mat to footwell
•   Install 2x screw
