Camshaft Pulleys, Replace (Z20 LET)

  Remove

Important:   A four-post hoist must be used to raise the vehicle when removing the engine undertray. The vehicle can be moved to a two-post hoist for other operations if required.
1.   Remove engine undertray
•   Raise vehicle
•   Remove 13x bolt, 13x washer
2.   Lower vehicle
3.   Disconnect battery negative lead
•   Release windscreen wash reservoir, then move to one side
•   Release nut, bolt
4.   Disconnect intercooler hose (1) from connecting pipe for throttle body
•   Release clip (2)
•   Pull off



5.   Raise vehicle to suitable working height
6.   Remove rear road wheel from right side
•   Remove security bolt
– Use special socket located under front bonnet
•   Remove 4x wheel bolt
7.   Remove wheel arch liner, right rear
•   Remove 4x screw, 2x clip
8.   Remove intercooler hose (1) from intercooler
•   Release clip (2)
•   Release fuel vapour line from intercooler hose (1)
– Remove, then discard 4x cable tie (3)



9.   Remove upper front cover (2) for timing belt
•   Remove 2x bolt (1)
•   Pull off



10.   Remove ribbed V-belt
•   Release ribbed V-belt tension
– Using suitable spanner rotate tensioner (1) clockwise, then Install suitable locking pin (2)



11.   Remove crankshaft pulley
•   Apply match marks to crankshaft pulley and crankshaft
•   Remove 4x bolt
– Counter-hold centre bolt on crankshaft sprocket
12.   Remove ribbed V-belt tensioner
•   Remove 2x bolt
13.   Remove lower cover for timing belt
•   Remove 2x bolt (1)



14.   Rotate crankshaft in normal direction of rotation to Top Dead Centre (TDC) on cylinder no. 1, until timing marks (2) and (1) are aligned
15.   Lock camshafts in position
•   Use Locking Tool KM-853
16.   Release tension from timing belt
•   Release bolt, then turn adjusting eccentric clockwise until pointer is located just before left stop
17.   Remove timing belt from pulleys
•   Position timing belt to one side



18.   Lower vehicle
19.   Disconnect harness connector (1) from ignition module
20.   Remove ignition module cover (2) from cylinder head
•   Slide off



21.   Remove ignition module from cylinder head
•   Remove 2x bolt (1)



22.   Disconnect harness connector from camshaft sensor (1)
23.   Disconnect 3x breather hose (2) from cylinder head cover (3)
24.   Remove cylinder head cover
•   Remove 10x bolt
•   Remove, then discard 10x 'O' ring
•   Remove, then discard gasket



25.   Remove 2x camshaft pulley (2) and (3) from camshaft
Note:   Note position of locating pin on exhaust pulley.
•   Remove 2x bolt (1), 2x washer
– Counter-hold camshaft at hexagon with open end of spanner (4)
– Second technician required
– Discard 2x bolt



  Install

26.   Install 2x new camshaft pulley to 2x camshaft
Note:   Note position of locating pin on exhaust pulley.
•   Install 2x new bolt, 2x washer 50 Nm + 60° + 15°
– Counter-hold camshaft at hexagon with open end of spanner
– Second technician required
27.   Install cylinder head cover to cylinder head
•   Install 10 x new 'O' ring, new gasket
•   Install 10x bolt 8 Nm
28.   Connect 3x breather hose to cylinder head cover
29.   Connect harness connector to camshaft sensor
30.   Install ignition module to cylinder head
•   Install 2x bolt 8 Nm
31.   Install ignition module cover to cylinder head
•   Slide on
32.   Connect harness connector to ignition module
33.   Raise vehicle to suitable working height
34.   Check timing marks are aligned
35.   Lock camshafts in position
•   Use Locking Tool KM-853
36.   Install timing belt to pulleys
37.   Pre-tension timing belt
•   Turn eccentric of tensioner in direction of arrow (anti-clockwise) until pointer is located just before right stop, then secure tensioner bolt
•   Remove Locking Tool KM-853 from camshafts
•   Rotate crankshaft two complete turns in normal direction of rotation to Top Dead Centre (TDC) on cylinder no. 1, until timing marks (ll) and (lll) are aligned
•   Install Locking Tool KM-853 to camshafts



38.   Tension timing belt
•   Release tensioner bolt, then turn eccentric of tensioner in direction of arrow (clockwise) until pointer is located in correct position
Note:   For new belt, pointer (1) for timing belt tensioner aligns with notched mark (3).
Note:   For used belt, pointer (1) for timing belt tensioner aligns with notched mark (2).
39.   Secure tensioner
•   Secure bolt 20 Nm
•   Remove Locking Tool KM-853
40.   Rotate crankshaft two complete turns in normal direction of rotation to (TDC) on cylinder no. 1, until timing marks are aligned. If pointer of tensioner does not align with corresponding notch mark, then repeat adjustment procedure



41.   Install lower cover of camshaft timing belt
•   Install 2x bolt 6 Nm
42.   Install ribbed V-belt tensioner
•   Install 2x bolt 25 Nm
43.   Install crankshaft pulley
•   Align match marks
•   Install 4x bolt 20 Nm
– Counter-hold centre nut on crankshaft sprocket
44.   Install ribbed V-belt
•   Use suitable spanner to rotate tensioner clockwise, then remove locking pin
45.   Install upper cover of camshaft timing belt
•   Push on to secure
•   Install 2x bolt 6 Nm
46.   Install intercooler hose to intercooler
•   Secure clip
•   Install 4x new cable tie
47.   Install liner to wheel arch
•   Install 4x screw, 2x clip
48.   Install rear road wheel to right side
•   Install 4x bolt, security bolt
•   Do not tighten
49.   Lower vehicle
50.   Secure rear road wheel right side
•   Tighten 4x bolt, security bolt 90 Nm
51.   Install intercooler hose to throttle body
•   Secure clip
52.   Connect battery negative lead
•   Secure nut and bolt
53.   Install windscreen wash reservoir
•   Push fit
54.   Program all volatile memories
55.   Raise vehicle
56.   Install engine undertray
•   Install 13x bolt, 13x washer
57.   Lower vehicle
