Upper Coolant Hose, Engine, Replace (Z 22 SE)

  Remove

1.   Drain coolant system – see operation “Coolant System, Drain, Fill and Bleed”
2.   Lower vehicle
3.   Release upper coolant hose from thermostat housing
•   Release, then remove clip
4.   Raise vehicle
5.   Remove fuel tank - see operation “Fuel Tank, Remove and Install”
6.   Remove upper coolant hose (2) from coolant pipe
•   Release, then remove clip (1)



  Install

7.   Transfer 2x clip to new upper coolant hose
8.   Install new upper coolant hose to coolant pipe
•   Install clip
9.   Install fuel tank - see operation “Fuel Tank, Remove and Install”
10.   Lower vehicle
11.   Secure upper coolant hose to thermostat housing
•   Install clip
12.   Fill cooling system - see operation “Cooling System, Drain, Fill and Bleed”
