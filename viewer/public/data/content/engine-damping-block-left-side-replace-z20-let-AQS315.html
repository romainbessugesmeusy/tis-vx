Engine Damping Block, Left Side, Replace (Z20 LET)

  Remove

Important:   A four-post hoist must be used to raise the vehicle when removing the engine undertray. The vehicle can be moved to a two-post hoist for other operations if required.
1.   Remove engine undertray
•   Raise vehicle
•   Remove 13x bolt, 13x washer
2.   Release engine damping block from chassis
•   Remove nut (1), then discard



3.   Lower vehicle to suitable working height
Important:   Do not support on oil pan.
4.   Install engine support to engine and transmission assembly
5.   Remove rear road wheel from right side
•   Remove security bolt
– Use special socket located under front bonnet
•   Remove 4x wheel bolt
6.   Remove wheel arch liner from left rear
•   Remove 4x screw, 2x clip
7.   Release coolant reservoir tank (1) from support bracket
•   Remove clip (2)
– Slide off
•   Disconnect coolant level sensor
•   Move coolant reservoir tank to one side



8.   Release mounting bracket (1) from engine damping block
•   Remove nut (2), then discard



9.   Remove mounting bracket (1) from engine damping block and transmission
•   Remove 3x bolt (2)



10.   Remove engine damping block (1) from chassis



  Install

11.   Install new engine damping block (1) to chassis
Note:   Ensure locating lug (2) on engine damping block locates in hole in chassis (3).



12.   Install mounting bracket to transmission and engine damping block
Note:   Ensure locating lug (1) on engine damping block locates in hole in mounting bracket (2).
Note:   Illustration shows parts removed for clarity.
•   Install 3x bolt 48 Nm
– Apply locking compound
•   Install nut 48 Nm
– Use new nut



13.   Secure coolant reservoir tank to support bracket
•   Connect coolant level sensor
•   Install clip
– Slide on
14.   Install wheel arch liner to left rear
•   Install 4x screw, 2x clip
15.   Install rear road wheel to left side
•   Install 4x bolt, security bolt
•   Do not tighten
16.   Raise vehicle
17.   Remove engine support from engine and transmission assembly
18.   Secure engine damping block to chassis
•   Install nut 48 Nm
– Use new nut
19.   Install engine undertray
•   Install 13x bolt, 13x washer
20.   Lower vehicle
21.   Secure rear road wheel to left side
•   Tighten 4x bolt, security bolt 90 Nm
