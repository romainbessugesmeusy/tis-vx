Intermediate Pipe, Replace (Z 20 LET)

  Remove

Important:   A four-post hoist must be used to raise the vehicle when removing exhaust pipe undertray. The vehicle can be moved to a two-post hoist for other operations if required.
1.   Remove exhaust pipe undertray (1)
•   Raise vehicle
•   Remove 12x bolt, 12x washer, 2x clip



2.   Release intermediate pipe and heatshield from rear silencer
•   Release clamp (1) from silencer



3.   Release intermediate pipe and heatshield from secondary catalytic converter
•   Remove, then discard 3x nut (1)



4.   Remove intermediate pipe and heatshield from vehicle
5.   Remove heatshield from intermediate exhaust pipe
•   Remove 5x clip (1)
•   Remove 3x bolt, 3x washer (2)



  Install

6.   Install heatshield to intermediate exhaust pipe
•   Install 3x bolt, 3x washer, 5x clip
7.   Install intermediate pipe and heatshield to vehicle
•   Install clamp
– Push fit
•   Position bracket for rubber mount
•   Install 3x new nut
•   Secure clamp for intermediate pipe
•   Secure 3x new nut
8.   Install exhaust pipe undertray
•   Install 12x bolt, 12x washer, 2x clip
9.   Lower vehicle
