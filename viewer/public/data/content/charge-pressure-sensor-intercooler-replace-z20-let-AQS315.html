Charge Pressure Sensor, Intercooler, Replace (Z20 LET)

  Remove

1.   Release rear road wheel from right side
•   Release security bolt
– Use special socket located under front bonnet
•   Release 4x wheel bolt
2.   Raise vehicle
3.   Remove rear road wheel from right side
•   Remove 4x bolt, security bolt
4.   Remove wheel arch liner from right side
•   Remove 4x screw, 2x clip
5.   Disconnect harness connector (1) from charge pressure sensor (2)
•   Release integral clip
6.   Remove charge pressure sensor (2) from intercooler
•   Remove bolt (3), washer



  Install

7.   Install charge pressure sensor to intercooler
•   Install bolt, washer
8.   Connect harness connector to charge pressure sensor
•   Secure integral clip
9.   Install wheel arch liner to right side
•   Install 4x screw, 2x clip
10.   Install rear road wheel to right side
•   Install 4x bolt, security bolt
•   Do not tighten
11.   Lower vehicle
12.   Secure rear road wheel to right side
•   Tighten 4x bolt, security bolt 90 Nm
