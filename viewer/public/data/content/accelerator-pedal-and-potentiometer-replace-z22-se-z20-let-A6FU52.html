Accelerator Pedal and Potentiometer, Replace (Z22 SE, Z20 LET)

  Remove

1.   Remove soft top or hard top
2.   Disconnect battery negative lead
•   Release windscreen wash reservoir, then move to one side
•   Release nut, bolt
3.   Disconnect harness (1) from potentiometer
•   Pull out clip (2)



4.   Remove accelerator pedal and potentiometer from bulkhead
•   Remove 3x bolt (1), 3x nut
Note:   Second technician required to counterhold nuts from inside vehicle.



5.   Remove pad from accelerator pedal
•   Remove 2x bolt (1), 2x nut, 2x washer, then discard 2x nut



  Install

6.   Install pad to new accelerator pedal
•   Install 2x bolt, 2x new nut, 2x washer
7.   Install accelerator pedal and potentiometer to bulkhead
•   Install 3x bolt, 3x nut
Note:   Second technician required to counterhold nuts from inside vehicle.
8.   Connect harness to potentiometer
•   Push in clip
9.   Connect battery negative lead
•   Secure nut, bolt
10.   Install windscreen wash reservoir
•   Push fit
11.   Program all volatile memories
12.   Install soft top or hard top
