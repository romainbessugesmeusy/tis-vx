Switch, Fog Lamp, Remove and Install

1.   Open bonnet
2.   Remove windscreen wash reservoir from mounting bracket
3.   Disconnect battery negative lead
•   Remove nut
4.   Release switch panel assembly from instrument panel
•   Pull off



5.   Remove switch unit assembly from panel
•   Release 4x clip



6.   Remove front cover from fog lamp switch
•   Pull off
7.   Remove fog lamp switch
•   Release 2x clip, release harness connector
8.   Install fog lamp switch
•   Secure 2x clip, secure harness connector
9.   Install front cover to fog lamp switch
•   Push fit
10.   Install switch to housing
•   Secure 4x clip
11.   Secure fog lamp switch to instrument surround
•   Push fit
