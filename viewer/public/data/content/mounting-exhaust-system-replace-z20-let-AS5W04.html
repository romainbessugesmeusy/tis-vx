Mounting, Exhaust System, Replace (Z20 LET)

  Remove

1.   Raise vehicle
2.   Remove exhaust undertray (1)
•   Remove 12x bolt, 12x washer, 2x clip



3.   Release intermediate pipe with heatshield (1) from secondary catalytic convertor (2)
•   Remove 3x nut (3), then discard
•   Release mounting bracket for secondary catalytic convertor (4) from stud on intermediate pipe



4.   Remove intermediate pipe with heatshield from rear silencer
•   Release clamp (1)



5.   Release support bracket from rear silencer
•   Remove 2x nut (1), then discard
6.   Remove flexible support for rear silencer and support bracket from subframe
•   Remove 2x bolt (2)
•   Remove clamp
•   Pull off flexible support for rear silencer from support bracket



  Install

7.   Install new flexible support for rear silencer and new support bracket to subframe
•   Push on flexible support for rear silencer to support bracket
•   Install clamp
•   Install 2x bolt
– Do not tighten
8.   Secure support bracket to rear silencer
•   Install 2x new nut
9.   Secure flexible support for rear silencer to subframe
•   Secure 2x bolt
10.   Install intermediate pipe with heatshield to rear silencer and secondary catalytic convertor
•   Secure clamp
11.   Secure intermediate pipe with heatshield to secondary catalytic convertor
•   Secure mounting bracket for secondary catalytic convertor to stud on intermediate pipe
•   Install 3x new nut
12.   Install exhaust undertray
•   Install 12x bolt, 12x washer, 2x clip
13.   Lower vehicle
