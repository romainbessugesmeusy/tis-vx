Engine Vent Flange, Replace (Z20 LET)

  Remove

1.   Disconnect air intake hose (1) from mass air flow meter (2) and turbocharger (3)
•   Release 2x clip
•   Move air intake hose to one side



2.   Remove support bracket (1) with engine lifting eye from cylinder head
•   Remove bolt



3.   Remove engine vent hose (1) from cylinder head cover and engine vent flange
•   Release 2x clip (2)
•   Pull off



4.   Remove heat shield (1) from support bracket (2) on cylinder head
•   Remove 4x bolt, 4x washer



5.   Remove engine vent flange (1) from engine cylinder block
•   Remove 2x bolt (2)
– Remove, then discard gasket



  Install

6.   Install new engine vent flange to engine cylinder block
•   Clean mating surfaces
•   Use new gasket
Note:   When installing new gasket (1), raised side of channel (2) must face upwards.
•   Install 2x bolt 25 Nm



7.   Install heat shield to support bracket on cylinder head
•   Install 4x bolt, 4x washer
8.   Install engine vent hose to cylinder head cover and engine vent flange
•   Secure 2x clip
•   Push on
9.   Install support bracket with engine lifting eye to cylinder head
•   Install bolt
– Apply locking compound
10.   Connect air intake hose to mass air flow meter and turbocharger
•   Secure 2x clip
