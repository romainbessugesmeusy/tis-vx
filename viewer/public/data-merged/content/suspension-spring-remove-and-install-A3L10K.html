Suspension Spring, Remove and Install

  Remove

Shock absorber from front suspension – see operation "Shock Absorber, Remove and Install".

  Disassemble

Suspension spring from shock absorber – use suitable spring compressor, release retaining circlip for spring collar from upper groove.

  Assemble

Suspension spring to shock absorber – use suitable spring compressor, install retaining circlip for spring collar in upper groove.

  Install

Shock absorber to front suspension.
