Rear Clamshell, Remove and Install

  Remove

1.   Release washer fluid reservoir from bracket
•   Slide off
•   Position washer fluid reservoir to one side



2.   Disconnect battery negative lead
•   Release nut and bolt



3.   Remove soft top – see operation "Soft Top, Remove and Install" in group C
4.   Move driver seat forward
5.   Remove upper anchor bolt of seat belt for driver seat from B–pillar
•   Release cover, then remove bolt



6.   Release passenger seat
•   Remove 4x bolt (1)
7.   Remove lower anchor bolt (2) of seat belt
•   Remove bolt (2)
•   Remove front seat from vehicle



8.   Remove upper anchor bolt of seat belt for passenger seat from B–pillar
•   Release cover, then remove bolt
9.   Remove 2x loudspeaker from rear bulkhead trim
•   For early type, remove 2x cover, 6x screw
•   For later type (illustrated), remove 8x screw (1)
•   Remove 4x harness connector



10.   Remove rear bulkhead trim
•   Remove 6x fastener (1)
•   Disconnect harness connector from ultrasonic control unit for anti-theft warning system
•   Remove soundproof insulation



Important:   A four-post hoist must be used to raise the vehicle when removing the engine undertray. The vehicle can be moved to a two-post hoist for other operations if required.
11.   Remove engine undertray (1)
•   Raise vehicle
•   Remove 13x bolt, 13x washer
12.   Remove exhaust undertray (2)
•   Remove 12x bolt, 12x washer, 2x clip



13.   Lower vehicle to suitable working height
14.   Remove rear road wheel from right side
•   Remove security bolt
– Use special socket located under front bonnet
•   Remove 4x wheel bolt
15.   Remove right rear wheel arch liner
•   Remove 4x screw, 2x clip
16.   Release rear clamshell from right chassis
•   Remove insulation pad
•   Remove 2x bolt, 4x washer
17.   Install right rear road wheel
•   Hand tighten 4x bolt, security bolt
18.   Remove rear road wheel from left side
•   Remove security bolt
– Use special socket located under front bonnet
•   Remove 4x wheel bolt
19.   Remove left rear wheel arch liner
•   Remove 4x screw, 2x clip
20.   Release rear clamshell from left chassis
•   Remove 2x bolt, 4x washer
21.   Disconnect 3x harness connector from electronic control unit and air temperature sensor
•   Release 3x integral clip



22.   Release earth connector (2) from support bracket for electronic control unit
•   Remove nut (1)



23.   Disconnect harness connector (1) for contact switch of anti-theft warning system
•   Release integral clip
•   Remove, then discard cable tie (2)



24.   Install rear left road wheel
•   Hand tighten 4x bolt, security bolt
25.   Lower vehicle
26.   Remove engine cover from rear clamshell
•   Apply match marks
•   Remove 6x bolt (1), 6x washer and shims
Note:   Mark position of shims for installation.
– 2nd technician required



27.   Release latch for engine cover with contact switch of anti-theft warning system from rear clamshell, then move to one side
•   Apply match mark
•   Remove 2x screw, 2x washer
Note:   Release cable for latch not disconnected.



28.   Remove luggage compartment mat and soundproof insulation
29.   Disconnect negative and positive leads from auxiliary battery
•   Remove 2x screw (1)



30.   Disconnect backup-up power switch for central door locking
•   Disconnect harness connector (1)
•   Release harness from 3x clip



31.   Remove relay and fuse cover (1)
•   Remove 4x screw (2)



32.   Disconnect harness connector from compact disc changer (where fitted)
•   Release integral clip
•   Withdraw harness through aperture in right rear inner wheel arch into engine bay
33.   Release relay and fuse holder from rear clamshell
•   Remove 4x relay (1)
Note:   Mark position of relays for installation.
•   Remove 2x screw (2)



34.   Disconnect 3x harness connector (1) for rear lighting
•   Release 3x integral clip
•   Withdraw harness through aperture of left rear inner wheel arch into engine bay



35.   Release luggage compartment floor from rear subframe
•   Remove 3x bolt (1), (2), 3x washer



36.   Release rear clamshell from rear bulkhead
•   Remove 2x bolt, 2x washer (arrowed)
•   Remove 2x nut, 2x washer



37.   Release rear header panel from roll bar, then tilt forward
Note:   Do not remove the weatherseal from the panel. Care should be taken not to cut or damage the weatherseal.
•   Release 2x soft top cantrail bracket
•   Remove 2x bolt, 6x nut from 6x stud (1), 6x washer



38.   Remove fuel filler cap from fuel filler neck
39.   Remove trim plate from fuel filler neck
•   Remove 6x screw
40.   Install fuel filler cap to fuel filler neck



Important:   During handling, care should be taken not to damage the rear clamshell or adjacent fibreglass parts. Protect the clamshell from damage when removed, with particular attention to edges and corners.
41.   Remove rear clamshell (1)
Note:   Second technician required.
•   Slide clamshell rearward to clear exhaust pipe, then lift off



  Install

42.   Install rear clamshell
Note:   Second technician required.
Note:   Align rear clamshell to specification, see operation “Body Panel Alignment Values” in group A. Only the areas detailed in this section are adjustable. All further gaps align as a consequence of these adjustments as the body panels are formed from common moulds.
•   Loosely assemble all fixing points
43.   Remove fuel filler cap from fuel filler neck
44.   Install trim plate to fuel filler neck
•   Install, then secure 6x screw
•   Install fuel filler cap to fuel filler neck
45.   Secure rear header panel to roll bar
•   Install, then secure 6x nut, 6x washer
46.   Secure 2x cantrail bracket
•   Ensure dowel pin is installed in roll bar
•   Install bolt
– Apply locking compound to bolt thread
47.   Secure rear clamshell to rear bulkhead
•   Secure 2x bolt, 2x washer
•   Secure 2x nut, 2x washer
48.   Secure luggage compartment floor to chassis
•   Secure 3x bolt, 3x washer
49.   Install latch for engine cover with contact switch of anti-theft warning system to rear clamshell
•   Align match mark
•   Install, then secure 2x screw, 2x washer
50.   Connect 3x harness connector for rear lighting
Note:   With left drive and right drive vehicles the positions of the reverse light and rear fog light are transposed. When reconnecting the lighting harness take care of the following connections. For left drive vehicles connect the main harness labelled “FOG” to the rear harness labelled “L” and the main harness labelled “REV” to the rear harness labelled “R”. For right drive vehicles connect the main harness labelled “REV” to the rear harness labelled “L” and the main harness labelled “FOG” to the rear harness labelled “R”.
•   Insert harness through aperture of left rear inner wheel arch into luggage compartment
– Ensure correct sealing of harness grommet
•   Secure 3x integral clip
51.   Secure relay and fuse holder to rear clamshell
•   Install, then secure 2x screw
•   Install 4x relay
Note:   Position of relays marked on removal.
52.   Connect harness connector to compact disc changer (where fitted)
•   Secure integral clip
•   Insert harness through aperture of right rear inner wheel arch into luggage compartment
– Ensure correct sealing of harness grommet
53.   Install cover for fuse and relay assembly
•   Install, then secure 4x screw
54.   Connect backup-up power switch for central door locking
•   Connect harness connector
•   Install, then secure harness to 3x clip
55.   Connect negative and positive leads to auxiliary battery
•   Install, then secure 2x screw
56.   Install luggage compartment mat and soundproof insulation
57.   Raise vehicle to suitable working height
58.   Remove rear road wheel from left side
•   Remove security bolt
– Use special socket located under front bonnet
•   Remove 4x wheel bolt
59.   Secure earth lead to support bracket for electronic control unit
•   Install, then secure nut
60.   Connect 3x harness connector to electronic control unit and air temperature sensor
•   Secure 3x integral clip
61.   Connect harness connector for contact switch of anti-theft warning system
•   Secure integral clip
•   Install, then secure with new cable tie
62.   Secure rear clamshell to left chassis
•   Secure 2x bolt 4x washer
63.   Install left rear wheel arch liner
•   Install, then secure 4x screw, 2x clip
64.   Install rear road wheel to left side
•   Hand tighten 4x bolt, security bolt
65.   Remove rear road wheel from right side
•   Remove security bolt
– Use special socket located under front bonnet
•   Remove 4x wheel bolt
66.   Secure rear clamshell to right chassis
•   Secure 2x bolt, 3x washer
•   Install insulation pad
67.   Install right rear wheel arch liner
•   Install, then secure 4x screw, 2x clip
68.   Install rear road wheel to right side
•   Hand tighten 4x bolt, security bolt
69.   Install exhaust undertray
•   Install, then secure 12x bolt, 12x washer, 2x clip
70.   Install engine undertray
•   Install, then secure 13x bolt, 13x washer
71.   Lower vehicle
72.   Secure 2x rear road wheel
•   Tighten 8x bolt, 2x security bolt 90 Nm
– Use special socket located under front bonnet
73.   Install engine cover to rear clamshell
•   Align match marks
•   Align engine cover - see operation “Body Panel Alignment Values” in group A
•   Install shims
Note:   Position of shims marked on removal.
•   Install, then secure 6x bolt, 6x washer
– Second technician required
74.   Install rear bulkhead trim
•   Install soundproof insulation
•   Connect harness connector to ultrasonic control unit for anti-theft warning system
•   Install, then secure 6x fastener
75.   Install 2x loudspeaker to rear bulkhead trim
•   For early type, install 6x screw, 2x cover
•   For later type, install 8x screw
•   Install 4x harness connector
76.   Install upper anchor bolt of seat belt for driver seat to B–pillar
•   Install, then secure bolt 33 Nm
•   Secure cover
77.   Install upper anchor bolt of seat belt for passenger seat to B–pillar
•   Install, then secure bolt 33 Nm
•   Secure cover
78.   Position passenger seat, then install lower anchor bolt of seat belt
•   Install, then secure bolt 33 Nm
79.   Install passenger seat
•   Install, then secure 4x bolt 25 Nm
80.   Install soft top – see operation "Soft Top, Remove and Install"
81.   Connect battery negative lead
•   Secure nut and bolt
82.   Secure washer fluid reservoir to bracket
•   Slide on
83.   Program all volatile memories
