Road Wheel, Remove and Install

  Remove

1.   Chock road wheels
2.   Remove security bolt from road wheel
•   Use special socket located under front bonnet



3.   Raise and support vehicle
4.   Remove road wheel from vehicle
•   Remove 4x wheel bolt

  Install

5.   Install road wheel to vehicle
•   Install 4x wheel bolt 90 Nm
6.   Raise vehicle and remove support
7.   Lower vehicle
8.   Install security bolt 90 Nm to road wheel
•   Install special socket to holder
9.   Remove chocks from road wheels
