Engine Damping Block, Rear, Replace (Z20 LET)

  Remove

Important:   A four-post hoist must be used to raise the vehicle when removing the engine undertray and exhaust pipe undertray. The vehicle can be moved to a two-post hoist for other operations if required.
1.   Remove engine undertray (1)
•   Raise vehicle
•   Remove 13x bolt, 13x washer
2.   Remove exhaust pipe undertray (2)
•   Remove 12x bolt, 12x washer, 2x clip



3.   Install engine support to engine and transmission assembly
4.   Remove heatshield (1) from support bracket (2)
•   Remove 2x bolt, washer from support bracket
•   Remove bolt, washer from engine cylinder block



5.   Remove engine damping block with support bracket (1) from transmission and chassis
•   Remove bolt (2)
•   Remove centre bolt, nut (3), then discard



6.   Remove engine damping block (2) from support bracket (3)
•   Apply match marks (1)
•   Use hydraulic press



  Install

7.   Install new engine damping block to support bracket
•   Align match marks
– Transfer match marks to new engine damping block
•   Use hydraulic press
8.   Install engine damping block with support bracket to transmission and chassis
•   Install centre bolt, nut (3)
– Use new bolt, nut
– Do not tighten
•   Install bolt (2)
– Apply locking compound
– Do not tighten



9.   Install heatshield to support bracket
•   Install 2x bolt to support bracket
– Apply locking compound
– Do not tighten
•   Install bolt, washer to engine cylinder block
– Do not tighten
10.   Secure support bracket and heatshield to transmission
•   Secure 3x bolt, 2x washer 49 Nm
11.   Secure heatshield to engine cylinder block
•   Secure bolt
Important:   Do not support on oil pan.
12.   Remove engine support from engine and transmission assembly
13.   Secure centre bolt and nut for engine damping block 86 Nm
14.   Install engine undertray
•   Install 13x bolt, 13x washer
15.   Install exhaust pipe undertray
•   Install 12x bolt, 12x washer, 2x clip
16.   Lower vehicle
