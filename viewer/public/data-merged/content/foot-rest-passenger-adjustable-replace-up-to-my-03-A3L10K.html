Foot Rest, Passenger, Adjustable, Replace (up to MY 03)

  Remove

1.   Remove foot rest from passenger's foot well
•   Raise vehicle
•   Remove 6x screw from underside of vehicle



2.   Lower vehicle
3.   Remove 2x guide (1) from foot rest



  Install

4.   Install 2x guide (1) to foot rest



5.   Install foot rest to passenger's foot well
•   Raise vehicle
•   Install 6x screw to underside of vehicle
6.   Lower vehicle
