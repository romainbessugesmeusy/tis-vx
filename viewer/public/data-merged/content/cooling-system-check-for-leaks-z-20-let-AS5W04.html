Cooling System, Check for Leaks (Z 20 LET)

  Inspect

Important:   A four-post hoist must be used to raise the vehicle when removing the engine undertray. The vehicle can be moved to a two-post hoist for other operations if required.
1.   Remove engine undertray
•   Raise vehicle
•   Remove 13x bolt, 13x washer
2.   Remove front undertray
•   Remove 9x bolt
3.   Lower vehicle
Warning:   Ensure engine is cold before removing coolant reservoir cap.
4.   Remove coolant reservoir cap
5.   Apply pressure to cooling system
•   Install pressure testing equipment (commercially available) to coolant reservoir
•   Follow manufacturer's instructions



6.   Visually inspect for coolant leaks from above
7.   Raise vehicle
8.   Visually inspect for coolant leaks from below
9.   Install front undertray
•   Install 9x bolt
10.   Install engine undertray
•   Install 13x bolt, 13x washer
11.   Lower vehicle
12.   Remove pressure testing equipment from coolant reservoir
13.   Install coolant reservoir cap
