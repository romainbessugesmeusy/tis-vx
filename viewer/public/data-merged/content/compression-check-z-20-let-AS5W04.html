Compression, Check (Z 20 LET)

  Measure

1.   Disconnect harness connector (1) from ignition module
2.   Remove top trim cover (2) from cylinder head
•   Slide off



3.   Remove ignition module from cylinder head
•   Remove 2x bolt (1)



4.   Remove spark plug from cylinder head
5.   Remove fuel pump relay located in luggage compartment
•   Pull off
6.   Install compression gauge (1), (commercially available) into cylinder head



7.   Crank engine for approximately 4 seconds
•   Observe reading of compression gauge
8.   Remove compression gauge (1) from cylinder head



9.   Install spark plug to cylinder head 25 Nm
10.   Repeat process for remaining cylinders
11.   Install fuel pump relay located in luggage compartment
•   Push fit
12.   Install ignition module to cylinder head
•   Install 2x bolt (1) 8 Nm



13.   Install top trim cover (2) to cylinder head
•   Slide on
14.   Connect harness connector (1) to ignition module
