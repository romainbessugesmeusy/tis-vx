Folding Cover, Drive Shaft, Wheel Side, Replace

  Remove

1.   Remove relevant drive shaft from vehicle - see operation “Drive Shaft, Remove and Install”
2.   Secure drive shaft in soft jaw vice
3.   Using suitable tool, release 2x clip (1)
•   Remove, then discard 2x retaining clip (1) from folding cover
– Slide folding cover along shaft



4.   Remove constant velocity joint from drive shaft
•   Use soft head mallet to tap inner bearing race
•   Remove, then discard clip from drive shaft



5.   Remove, then discard folding cover from drive shaft
6.   Clean all components

  Install

7.   Install new folding cover to drive shaft
•   Position new retaining clip for folding cover to drive shaft
Important:   Use specified grease for constant velocity joint.
8.   Install constant velocity joint to drive shaft
•   Install new retaining clip to drive shaft
9.   Install folding cover to groove on drive shaft and groove on be constant velocity joint
•   Position 2x retaining clip to folding cover
– Check positioning of folding cover and 2x retaining clip
•   Secure 2x retaining clip to folding cover
– Use commercially available pliers
10.   Install drive shaft to vehicle - see operation “Drive Shaft, Remove and Install”
