Valves, remove

Special service tools required:

Assembly tool set KM-6215, installation/removal set MKM-6086

Note:   All attaching parts such as camshaft(s) and hydraulic valve lifters are removed.

  Install

Attach side mounts KM-6215-5 (1) for lever KM-6215-4 (2) to assembly device KM-6215-1. Attach lever KM-6215-4 (2).

See operation "Cylinder head, disassemble" , "Cylinder head, assemble".
