Subject:

Reprogrammable Electronic Control Units

 

       Some confusion still exists as to which electronic control units can be reprogrammed with the use of TECH2 and the Tech Information System (TIS)

 

       To clarify the situation a table has been produced which lists those electronic control units which are reprogrammable. The table is divided into Engine Management Systems and Automatic Transmission Systems and each lists:

 

       -      System

       -      Make and Options

       -      When the system was first introduced

       -      System Name

 

       Before embarking on lengthy diagnosis into fuel injection concerns it is always beneficial to check the Field Remedies on TIS first as new software may have been released to rectify the vehicles specific condition
