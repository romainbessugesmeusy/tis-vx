Spark Plugs, Replace (Z 20 LET)

  Remove

1.   Disconnect harness connector (1) from ignition module
2.   Remove top trim cover (2) from cylinder head
•   Slide off



3.   Remove ignition module from cylinder head
•   Remove 2x bolt (1)



4.   Remove 4x spark plug from cylinder head

  Install

5.   Install 4x spark plug to cylinder head 25 Nm
6.   Install ignition module to cylinder head
•   Install 2x bolt (1), 8 Nm



7.   Install top trim cover (2) to cylinder head
•   Slide on
8.   Connect harness connector (1) to ignition module
