Camshaft Timing Belt, Replace (Z 20 LET)

  Remove

Important:   A four-post hoist must be used to raise the vehicle when removing the undertray. The vehicle can be moved to a two-post hoist for other operations if required.
1.   Raise vehicle
2.   Remove engine undertray (1)
•   Remove 13x bolt, 13x washer



3.   Lower vehicle
4.   Release windscreen wash reservoir
•   Pull off, then move to one side
5.   Disconnect battery negative lead
•   Release nut, bolt
6.   Release intercooler hose (1) from throttle body (2)
•   Release clip



7.   Raise vehicle to suitable working height
8.   Remove rear road wheel from right side
•   Remove security bolt
– Use special socket located under front bonnet
•   Remove 4x wheel bolt
9.   Remove liner from wheel arch
•   Remove 6x screw
10.   Release intercooler hose (1) from intercooler (2)
•   Release clip
•   Remove, then discard 4x cable tie



11.   Remove upper cover of camshaft timing belt
•   Remove 2x bolt (1)
•   Pull off to release



12.   Remove ribbed V-belt
•   Release ribbed V-belt tension
– Use suitable spanner to rotate tensioner (1) clockwise, then install suitable locking pin (2)



13.   Remove crankshaft pulley
•   Apply match marks
•   Remove 4x bolt
– Use spanner to counter-hold centre nut on crankshaft sprocket
14.   Remove ribbed V-belt tensioner
•   Remove 2x bolt
15.   Remove lower cover of camshaft timing belt
•   Remove 2x bolt (1)



16.   Rotate crankshaft in normal direction of rotation to Top Dead Centre (TDC) on cylinder no. 1, until timing marks (3) and (1) are aligned
17.   Lock camshafts in position
•   Use Locking Tool KM-853 (2)
18.   Release tension from timing belt
•   Release bolt, then turn adjusting eccentric in direction of arrow (clockwise) until pointer (4) is located just before left stop
19.   Remove timing belt from pulleys
•   Position timing belt to one side



20.   Support engine
•   Use suitable support stand with KM-904 and KM-6054
21.   Remove engine mounting bracket from support bracket on cylinder head and damping block
•   Remove 3x bolt
•   Remove nut
22.   Remove timing belt from vehicle

  Install

23.   Install timing belt to vehicle
24.   Install engine mounting bracket to support bracket on cylinder head and damping block
•   Install, then secure 3x bolt 50 Nm
•   Install, then secure nut 50 Nm
25.   Remove support stand with KM-904 and KM-6054 from engine
26.   Check timing marks are aligned, then install timing belt to pulleys
27.   Pre-tension timing belt
•   Turn eccentric of tensioner in direction of arrow (anti-clockwise) until pointer is located just before right stop, then tighten tensioner bolt
•   Remove Locking Tool KM-853 from camshafts
•   Rotate crankshaft two complete turns in normal direction of rotation to Top Dead Centre (TDC) on cylinder no. 1, until timing marks (ll) and (lll) are aligned
•   Install Locking Tool KM-853 to camshafts



28.   Tension timing belt
•   Release tensioner bolt, then turn eccentric of tensioner indirection of arrow (clockwise) until pointer is located in correct position

Option I:

Pointer (1) for toothed belt tension roller aligns with the notched mark (2)

Option II:

Pointer (1) for toothed belt tension roller aligns with the "NEW" notched mark (2)

29.   Tighten tensioner
•   Secure bolt 20 Nm
•   Remove Locking Tool KM-853
30.   Rotate crankshaft two complete turns in normal direction of rotation to (TDC) on cylinder no. 1, until timing marks are aligned. If pointer of tensioner does not align with corresponding notch mark, then repeat adjustment procedure



31.   Install lower cover of camshaft timing belt
•   Install, then secure 2x bolt (1) 6 Nm
32.   Install ribbed V-belt tensioner
•   Install, then secure 2x bolt 25 Nm
33.   Install crankshaft pulley
•   Align match marks
•   Install, then secure 4x bolt 20 Nm
– Use spanner to counter-hold centre nut on crankshaft sprocket



34.   Install ribbed V-belt
•   Use suitable spanner to rotate tensioner (1) clockwise, then remove locking pin (2)



35.   Install upper cover of camshaft timing belt
•   Push on to secure
•   Install, then secure 2x bolt (1) 6 Nm



36.   Install intercooler hose (1) to intercooler (2)
•   Secure clip
•   Install, then secure 4x new cable tie



37.   Install liner to wheel arch
•   Install, then secure 6x screw
38.   Install rear road wheel to right side
•   Install 4x bolt, security bolt 90 Nm
39.   Lower vehicle
40.   Install intercooler hose (1) to throttle body (2)
•   Install, then secure clip



41.   Connect battery negative lead
•   Secure nut, bolt
42.   Install windscreen wash reservoir
•   Push fit
43.   Program all volatile memories
44.   Raise vehicle
45.   Install engine undertray
•   Install, then secure 13x bolt, 13x washer
46.   Lower vehicle
