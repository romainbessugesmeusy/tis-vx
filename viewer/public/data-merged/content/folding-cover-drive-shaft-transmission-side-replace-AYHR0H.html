Folding Cover, Drive Shaft, Transmission Side, Replace

  Remove

1.   Remove relevant drive shaft from vehicle - see operation “Drive Shaft, Remove and Install”
2.   Secure drive shaft in soft jaw vice
3.   Using suitable tool, release 2x clip (1)
•   Remove, then discard 2x retaining clip (1) from folding cover
– Slide folding cover along shaft



4.   Remove constant velocity joint from drive shaft
•   Using suitable tool, remove circlip (1) from outer race of constant velocity joint
•   Remove 6x ball bearing
•   Slide bearing cage clear of inner race
•   Remove, then discard circlip (2) from drive shaft
– Use commercially available tool
•   Remove bearing inner race and bearing cage from drive shaft



5.   Remove, then discard folding cover from drive shaft
6.   Clean all components

  Install

7.   Install new folding cover to drive shaft
•   Position new folding cover retaining clip to drive shaft
8.   Install bearing cage, then inner bearing race to drive shaft
•   Install new circlip to drive shaft
Note:   Install bearing cage with groove away from drive shaft.
Note:   Install bearing inner race with flattened spline towards driveshaft.
Important:   Use specified grease for constant velocity joint.
9.   Install 6x ball bearing to inner race and bearing cage
10.   Install outer race to 6x bearing
•   Install new circlip to outer race
11.   Install folding cover to groove on drive shaft and groove on constant velocity joint
•   Position 2x retaining clip to folding cover
– Check positioning of folding cover and 2x retaining clip
•   Secure 2x retaining clip to folding cover
– Use commercially available pliers
12.   Install drive shafts to vehicle - see operation “Drive Shaft, Remove and Install, Both Sides
