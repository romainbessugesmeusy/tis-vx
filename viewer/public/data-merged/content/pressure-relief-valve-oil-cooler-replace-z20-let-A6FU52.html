Pressure Relief Valve, Oil Cooler, Replace (Z20 LET)

  Remove

Important:   A four post hoist must be used to raise the vehicle when removing the engine undertray. The vehicle can be moved to a two post hoist for other operations if required.
1.   Remove engine undertray (1)
•   Raise vehicle
•   Remove 13x bolt, 13x washer



Important:   Observe relevant environmental regulations when disposing of used coolant and lubricants.
2.   Remove, then discard oil filter
•   Use Oil Filter Tool KM-726-A
– Position suitable collecting container
3.   Release oil feed pipe and oil return pipe from oil filter housing
•   Remove banjo bolt from oil feed pipe (1)
•   Remove banjo bolt from oil return pipe (2)
•   Remove 4x sealing washer, then discard
•   Remove clip



4.   Remove temperature controller housing for oil cooler
•   Remove adapter nut (1) using Remover KM-943
5.   Remove bolt (2)
•   Remove, then discard O ring



6.   Remove pressure relief valve
•   Cut thread into shutter disc using 10 mm tap (1)
•   Install 10 mm bolt, then extract pressure relief valve
– Pull out



  Install

7.   Install new pressure relief valve
•   Push fit using suitable tool
8.   Install temperature controller housing for oil cooler
•   Install bolt 23 Nm
•   Install new O ring
– Lubricate new O ring
•   Install adapter nut for oil filter 23 Nm
– Use Installer KM-943
9.   Secure oil feed pipe and oil return pipe to oil filter housing
•   Install banjo bolt for oil feed pipe 30 Nm
•   Install banjo bolt for oil return pipe 30 Nm
•   Install 4x new sealing washer
•   Install clip
10.   Install new oil filter
•   Use Oil Filter Tool KM-726-A
•   Screw fit 15 Nm
– Lubricate O ring seal
11.   Install engine undertray
•   Install 13x bolt, 13x washer
12.   Lower vehicle
13.   Run engine and check for leaks
•   Switch off engine when complete
14.   Check engine oil level, then top up if required
