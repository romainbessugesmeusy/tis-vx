Brake Fluid Reservoir, Replace

  Remove

1.   Open front bonnet
2.   Release cap from brake fluid reservoir



3.   Empty brake fluid reservoir of fluid
•   Use syringe to remove brake fluid
4.   Release clutch hose (1) from brake fluid reservoir



5.   Remove brake fluid reservoir from bracket
•   Remove cable tie



6.   Release 2x brake hose from brake fluid reservoir



7.   Remove 3x grommet from brake fluid reservoir



8.   Remove filter from brake fluid reservoir



  Install

9.   Install filter to new brake fluid reservoir



10.   Install 3x grommet to brake fluid reservoir
11.   Secure 2x brake hose to brake fluid reservoir
•   Push fit
12.   Install brake fluid reservoir to bracket
•   Install cable tie
13.   Secure clutch hose to brake fluid reservoir
•   Push fit
14.   Replenish brake fluid reservoir to correct level
•   Top up to MAX mark
– Use only DOT 4 brake fluid
15.   Secure cap to brake fluid reservoir
