Relief Valve, Fuel Pressure, Replace (Z20 LET)

  Remove

1.   Disconnect battery negative lead
•   Release windscreen wash reservoir, then move to one side
•   Release nut, bolt
Important:   Fuel may escape – observe the safety regulations and relevant national legislation.
2.   Release fuel pressure from fuel system
•   Connect test connection hose (2) to relief valve for fuel pressure on fuel rail
– Remove protective cap
•   Release fuel pressure using Pressure Tester KM-J-34730-91 (1)
– Collect any escaping fuel in a suitable container
3.   Remove Pressure Tester KM-J-34730-91 (1)
•   Disconnect test connection hose (2) from relief valve for fuel pressure on fuel rail



4.   Remove relief valve for fuel pressure (1) from fuel rail (2)
•   Remove, then discard washer



  Install

5.   Install new relief valve for fuel pressure to fuel rail
•   Install new washer
•   Install protective cap
6.   Connect battery negative lead
•   Secure nut, bolt
7.   Install windscreen wash reservoir
•   Push fit
8.   Program all volatile memories
