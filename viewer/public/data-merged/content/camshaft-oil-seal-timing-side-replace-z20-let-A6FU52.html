Camshaft Oil Seal, Timing Side, Replace (Z20 LET)

  Remove

Important:   A four-post hoist must be used to raise the vehicle when removing the engine undertray. The vehicle can be moved to a two-post hoist for other operations if required.
1.   Remove engine undertray
•   Raise vehicle
•   Remove 13x bolt, 13x washer
2.   Lower vehicle
3.   Disconnect battery negative lead
•   Release windscreen wash reservoir, then move to one side
•   Remove nut, bolt
4.   Disconnect intercooler hose (1) from throttle body (2)
•   Release clip



5.   Raise and support vehicle
6.   Remove rear road wheel from right side
•   Remove security bolt
– Use special socket located under front bonnet
•   Remove 4x wheel bolt
7.   Remove liner from wheel arch
•   Remove 4x screw, 2x clip
8.   Remove intercooler hose (1) from intercooler (2)
•   Release clip
•   Remove 4x cable tie, then discard



9.   Remove upper front cover (2) for timing belt
•   Remove 2x bolt (1)
•   Pull off to release



10.   Remove ribbed V-belt
•   Release ribbed V-belt tension
– Rotate tensioner (1) clockwise, then install suitable locking pin (2)



11.   Remove crankshaft pulley
•   Apply match marks to crankshaft pulley and crankshaft
•   Remove 4x bolt
– Counter-hold centre nut on crankshaft sprocket
12.   Remove ribbed V-belt tensioner
•   Remove 2x bolt
13.   Remove lower cover of camshaft timing belt
•   Remove 2x bolt (1)



14.   Rotate crankshaft clockwise to top dead centre (TDC) on cylinder No. 1, until 2x timing mark (3) and (1) are aligned
15.   Lock camshafts in position
•   Use Locking Tool KM-853 (2)
16.   Release tension from camshaft timing belt
•   Release bolt, then turn adjusting eccentric in direction of arrow (clockwise) until pointer (4) is located just before left stop
17.   Remove camshaft timing belt from pulleys, move to one side
•   Do not remove camshaft timing belt from vehicle



18.   Lower vehicle
19.   Disconnect harness connector (1) from ignition module
20.   Remove ignition module cover (2) from cylinder head
•   Slide off



21.   Remove ignition module from cylinder head
•   Remove 2x bolt (1)



22.   Disconnect harness connector from camshaft sensor
•   Release integral clip
•   Release harness from camshaft cover
– Release 3x integral clip
23.   Disconnect 3x breather hose (1) from camshaft cover
•   Release 3x clip
24.   Remove camshaft cover (2) from cylinder head
•   Remove 10x bolt
•   Remove 10x O ring, then discard
•   Remove gasket, then discard



Important:   2nd technician required to counter-hold camshaft.
25.   Remove camshaft sprocket
•   Remove bolt, then discard
26.   Remove camshaft oil seal (1)
•   Prise out using suitable tool



  Install

27.   Install new camshaft oil seal
•   Use Installer KM-422
– Lightly lubricate new seal
Important:   2nd technician required to counter-hold camshaft.
28.   Install camshaft sprocket
•   Install, then secure new bolt, washer 50 Nm + 60° + 15°
29.   Install camshaft cover
•   Install new gasket
•   Install 10x new O ring
•   Install 10x bolt 8 Nm
30.   Connect 3x breather hose to camshaft cover
•   Secure 3x clip
31.   Connect harness connector to camshaft sensor
•   Secure integral clip
32.   Secure harness to camshaft cover
•   Secure 3x integral clip
33.   Install ignition module to cylinder head
•   Install 2x bolt 8 Nm
34.   Install ignition module cover to cylinder head
•   Slide on
35.   Connect harness connector to ignition module
36.   Raise vehicle to suitable working height
37.   Rotate crankshaft clockwise to top dead centre (TDC) on cylinder No.1, until timing marks are aligned
38.   Lock camshafts in position
•   Use Locking Tool KM-853
39.   Install camshaft timing belt to pulleys
40.   Pre-tension camshaft timing belt
•   Turn eccentric of tensioner anti-clockwise until pointer is located just before right stop, then tighten tensioner bolt
•   Remove Locking Tool KM-853 from camshafts
•   Rotate crankshaft two complete turns (720°) clockwise to top dead centre (TDC) on cylinder No.1, until timing marks are aligned
•   Install Locking Tool KM-853 to camshafts
41.   Tension camshaft timing belt
•   Release tensioner bolt, then turn eccentric of tensioner clockwise until pointer is located in correct position (new or old camshaft timing belt)
42.   Secure tensioner
•   Tighten bolt 20 Nm
•   Remove Locking Tool KM-853
43.   Rotate crankshaft two complete turns (720°) clockwise to top dead centre (TDC) on cylinder No.1, until timing marks are aligned. If pointer of tensioner does not align with corresponding notch mark, then repeat adjustment procedure
44.   Install lower cover of camshaft timing belt
•   Install 2x bolt 6 Nm
45.   Install ribbed V-belt tensioner
•   Install 2x bolt 25 Nm
46.   Install crankshaft pulley
•   Align match marks
•   Install 4x bolt 20 Nm
– Using suitable tool, counter-hold centre nut on crankshaft sprocket
47.   Install ribbed V-belt
•   Using suitable tool, rotate tensioner clockwise, then remove locking pin
48.   Install upper cover of camshaft timing belt
•   Install 2x bolt 6 Nm
49.   Install intercooler hose to intercooler
•   Secure clip
•   Install 4x new cable tie
50.   Install liner to wheel arch
•   Install 4x screw, 2x clip
51.   Install rear road wheel
•   Install 4x bolt, security bolt
•   Do not tighten
52.   Lower vehicle
53.   Secure rear road wheel
•   Install 4x bolt, security bolt 90 Nm
54.   Connect intercooler hose to throttle body
•   Secure clip
55.   Connect battery negative lead
•   Install nut, bolt
•   Secure windscreen wash reservoir
56.   Program all volatile memories
57.   Raise vehicle
58.   Install engine undertray
•   Install 13x bolt, 13x washer
59.   Lower vehicle
