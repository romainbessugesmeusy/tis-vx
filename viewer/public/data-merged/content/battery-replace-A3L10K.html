Battery, Replace

1.   Open bonnet
2.   Remove windscreen wash reservoir from mounting bracket
3.   Release heater duct from bulkhead
•   Push fit
4.   Disconnect battery leads
•   Remove 2x nut



5.   Remove battery clamp from bulkhead
•   Remove 3x clamp bolt
6.   Remove battery from vehicle
7.   Remove leak hose from battery and install to new battery
•   Push fit
8.   Install battery to vehicle
9.   Install battery clamp to bulkhead
•   Install 3x clamp bolt
10.   Connect leads to battery
•   Install 2x nut
11.   Secure heater duct to bulkhead
•   Push fit
12.   Install windscreen wash reservoir to mounting bracket
13.   Close bonnet
