Oil Pan, Lower, Replace (Z 20 LET)

  Remove

Important:   A four-post hoist must be used to raise the vehicle when removing the engine undertray and exhaust pipe undertray. The vehicle can be moved to a two-post hoist for other operations if required.
1.   Remove engine undertray (1)
•   Raise vehicle
•   Remove 13x bolt, 13x washer
2.   Remove exhaust pipe undertray (2)
•   Remove 12x bolt, 12x washer, 2x clip



3.   Remove drain bolt (1)
•   Position collecting container
•   Remove, then discard O ring
4.   Release lower oil pan from upper oil pan
Note:   Support lower oil pan when removing to avoid damage to harness for oil level sensor.
•   Remove 16x bolt (2)
5.   Release oil level sensor from lower oil pan
•   Remove 2x bolt
•   Remove, then discard oil pan gasket



  Install

6.   Secure oil level sensor to lower oil pan
•   Install 2x bolt 8 Nm
7.   Install lower oil pan to upper oil pan
•   Install new oil pan gasket
•   Install 16x bolt 8 Nm
8.   Install drain bolt 10 Nm
•   Install new O ring
– Lubricate new O ring
9.   Lower vehicle
10.   Fill engine to correct oil level
•   Start and run engine until oil light goes out
•   Check oil level, adjust if necessary
11.   Raise vehicle
12.   Install engine undertray
•   Install 13x bolt, 13x washer
13.   Install exhaust pipe undertray (2)
•   Install 12x bolt, 12x washer, 2x clip
14.   Lower vehicle
