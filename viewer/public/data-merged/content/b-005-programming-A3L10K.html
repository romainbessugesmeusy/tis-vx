B-005 - PROGRAMMING

Test

Work Order Description

Nominal Value
T001 Tester Display Erase Remote Keys

This diagnostic tester function is used to delete all car key codes, which are programmed in the corresponding control unit. None of the keys will function after the erase procedure. All vehicle specific car keys supplied by the manufacturer to the customer must then be reprogrammed. In order for the reprogramming to work properly, all those keys must be available.

  • Ignition ON
  • Press corresponding key in the system main menu to call up Programming functions, select the desired test and confirm with ENTER . Follow the instructions in the diagnostic tester display.

If the following display appears during the test, programming has been completed successfully:

After programming successfully, the following trouble code is set: FC 14

Note:

Before the first programming is called up after communication has been established, you must enter the security code. The security code protects the control unit from unauthorised access. The valid code number is printed in the car pass.

Remote Keys Erasing Procedure Completed !
Concerned Terminals:

T002 Tester Display Program Remote Key

This mode can be used to read the car key codes of the car keys and store them in the memory of the control unit. This programming function must be repeated once for every car key.

The vehicle is supplied by the manufacturer with 2 programmed remote control keys

5 extra keys can be programmed in the control unit.

  • Ignition ON
  • Press corresponding key in the system main menu to call up Programming functions, select the desired test and confirm with ENTER . Follow the instructions in the diagnostic tester display.

If the following display appears during the test, programming has been completed successfully:

After programming of at least one car key the fault memory will be erased automatically.

Note:

Before the first programming is called up after communication has been established, you must enter the security code. The security code protects the control unit from unauthorised access. The valid code number is printed in the car pass.

Remote Key Successfully Programmed!
