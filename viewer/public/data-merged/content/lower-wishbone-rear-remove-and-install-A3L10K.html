Lower Wishbone, Rear, Remove and Install

  Remove

1.   Remove brake pads from caliper - see "Brake Pads, Rear, Replace" in group H
2.   Remove brake disc from wheel hub
•   Remove screw



3.   Remove shock absorber from lower wishbone
•   Remove nut (1), washer, bolt



4.   Release parking brake cable from lower wishbone
•   Remove nut, clip
5.   Remove lower wishbone from hub carrier
Note:   Illustration shows wheel hub and drive shaft removed for clarity
•   Remove nut



6.   Remove lower wishbone (1) from front chassis bracket
•   Remove bolt, tab washer
7.   Remove lower wishbone from rear chassis bracket
•   Remove nut, washer
•   Release track control arm (2) from chassis bracket and lower wishbone



  Install

8.   Install lower wishbone to front chassis bracket
•   Install new tab washer, bolt 45 Nm
9.   Install lower wishbone to hub carrier
•   Install new nut 45 Nm
Important:   End of track control arm locates into chassis. Ensure correct new locking nut is used. Early type have coarse thread, later type have fine thread.
10.   Install lower wishbone to rear chassis bracket
•   Install track control arm to chassis bracket and lower wishbone
•   Install washer, new nut 45 Nm
11.   Install shock absorber to lower wishbone
•   Install bolt, washer, new nut 45 Nm
12.   Connect parking brake cable to lower wishbone
•   Install clip, new nut
13.   Install brake disc to hub carrier
•   Install screw 12 Nm
14.   Install brake pads to caliper - see "Brake Pads, Rear, Replace" in group H
