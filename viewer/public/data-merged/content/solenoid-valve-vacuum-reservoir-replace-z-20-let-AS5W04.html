Solenoid Valve, Vacuum Reservoir, Replace (Z 20 LET)

  Remove

1.   Disconnect harness connector (2) from solenoid valve
2.   Disconnect 3x vacuum hose (3)
3.   Remove solenoid valve from support bracket on intake manifold
•   Remove 2x bolt (1)



  Install

4.   Install solenoid valve to support bracket on intake manifold
•   Install 2x bolt
5.   Connect 3x vacuum hose
6.   Connect harness connector to solenoid valve
