Ribbed V-belt Tensioner, Replace (Z 20 LET)

  Remove

Remove Ribbed V-belt, - see operation "Ribbed V-belt, Replace".

Remove 2x bolt, then remove ribbed V-belt tensioner (1) from alternator main bracket.


  Install

Attach ribbed V-belt tensioner to alternator support. Install 2x bolt 35 Nm

Install ribbed V-belt, - see operation "Ribbed V-belt, Replace".
