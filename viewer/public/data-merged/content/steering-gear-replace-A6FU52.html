Steering Gear, Replace

  Remove

1.   Remove hard top or soft top from vehicle
2.   Raise vehicle
3.   Raise and support front of vehicle from hoist
•   Install 2x chock to rear road wheel
4.   Lower vehicle to suitable working height
5.   Remove front road wheel, from left side
•   Remove security bolt
– Use special socket located under front bonnet
•   Remove 4x wheel bolt
6.   Remove left tie-rod end from left steering arm
•   Remove, then discard nut
Note:   Use commercially available joint separator.



7.   Remove front road wheel, from right side
•   Remove security bolt
– Use special socket located under front bonnet
•   Remove 4x wheel bolt
8.   Remove right tie-rod end from right steering arm
•   Remove, then discard nut
Note:   Use commercially available joint separator.



9.   Lower vehicle
10.   Release steering gear from steering column
•   Remove 2x nut, 2x bolt (1) from lower universal joint
•   Remove nut, bolt (2) from upper universal joint
Note:   Apply match mark to upper and lower universal joints.
11.   Release steering gear from bulkhead
•   Remove 2x lower bolt (4) for steering gear
•   Remove 2x upper bolt (3) for steering gear
12.   Remove lower universal joint and pinion shaft



13.   Remove pinion shaft grommet from bulkhead
14.   Raise vehicle to suitable working height
15.   Remove steering gear from vehicle
•   Lift both ends of steering gear, then move steering gear to right side of vehicle
Note:   2x technician required.
•   Lower steering gear, then rotate to remove pinion from bulkhead
Note:   2x technician required.
•   Withdraw steering gear from right side of vehicle
Note:   2x technician required.

  Install

16.   Install new steering gear to vehicle from right side
Note:   2x technician required.
•   Raise steering gear, then rotate to install pinion to bulkhead
Note:   2x technician required.
17.   Lower vehicle
18.   Install steering gear to bulkhead
•   Install 2x lower bolt to steering gear, loose fit
•   Install 2x upper bolt to steering gear, loose fit
Note:   Use recommended thread locking compound.
Note:   2x technician required.
19.   Install pinion shaft grommet to bulkhead
20.   Install lower universal joint and pinion shaft
•   Check straight ahead position, then align match marks
•   Install 2x nut, 2x bolt to lower universal joint 35 Nm
•   Install nut, bolt to upper universal joint 35 Nm
21.   Secure 2x upper bolt for steering gear 45 Nm
•   Secure 2x lower bolt for steering gear 22 Nm
22.   Raise vehicle to suitable working height
23.   Install right tie-rod end to right steering arm
•   Install new nut 30 Nm
24.   Install left tie-rod end to left steering arm
•   Install new nut 30 Nm
25.   Install front roadwheel to left side
•   Install 4x bolt 90 Nm
•   Install locking bolt 90 Nm
– Use special socket located under front bonnet
26.   Install front roadwheel to right side
•   Install 4x bolt 90 Nm
•   Install locking bolt 90 Nm
– Use special socket located under front bonnet
27.   Lower vehicle to hoist
•   Remove 2x chock from rear road wheel
28.   Lower vehicle
29.   Install hard top or soft top to vehicle
30.   Check steering wheel and front wheel alignment - see operation “Front Wheel Geometry” and “Straight Ahead Position, Check and Adjust”
