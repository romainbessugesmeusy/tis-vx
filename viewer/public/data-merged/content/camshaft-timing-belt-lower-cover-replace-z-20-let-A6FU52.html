Camshaft Timing Belt, Lower Cover, Replace (Z 20 LET)

  Remove

Important:   A four-post hoist must be used to raise the vehicle when removing the engine undertray. The vehicle can be moved to a two-post hoist for other operations if required.
1.   Remove engine undertray
•   Raise vehicle
•   Remove 13x bolt, 13x washer
2.   Remove 2x bolt (1) from upper cover of camshaft timing belt



3.   Lower vehicle
4.   Remove upper cover of camshaft timing belt
•   Pull off to release
5.   Raise vehicle
6.   Remove ribbed V-belt
•   Release ribbed V-belt tension
– Using suitable spanner rotate tensioner (1) clockwise, then Install suitable locking pin (2)



7.   Remove crankshaft pulley
•   Apply match marks
•   Remove 4x bolt, using spanner to counter-hold centre nut on crankshaft sprocket
8.   Remove ribbed V-belt tensioner
•   Remove 2x bolt
9.   Remove lower cover of camshaft timing belt
•   Remove 2x bolt (1)



  Install

10.   Install lower cover of camshaft timing belt
•   Install 2x bolt (1) 6 Nm
11.   Install ribbed V-belt tensioner
•   Install 2x bolt 25 Nm
12.   Install crankshaft pulley
•   Align match marks
•   Install 4x bolt 20 Nm using suitable spanner to counter-hold centre nut on crankshaft sprocket



13.   Install ribbed V-belt
•   Using suitable spanner rotate tensioner (1) clockwise, then remove locking pin (2)



14.   Lower vehicle
15.   Install upper cover of camshaft timing belt
•   Push fit to secure
16.   Raise vehicle
17.   Install 2x bolt (1) to upper cover of camshaft timing belt 6 Nm



18.   Install engine undertray
•   Install 13x bolt, 13x washer
19.   Lower vehicle
