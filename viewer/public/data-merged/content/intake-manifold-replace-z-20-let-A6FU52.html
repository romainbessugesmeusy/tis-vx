Intake Manifold, Replace (Z 20 LET)

  Remove

1.   Release screen wash reservoir from support bracket
•   Slide off
– Position screen wash reservoir to one side



2.   Disconnect battery negative lead
•   Release nut and bolt



3.   Disconnect intercooler hose (1) from connecting pipe for throttle body
•   Release clip (2)
– Position intercooler hose (1) to one side



4.   Disconnect vacuum hose (1) from cylinder head cover
•   Release clip (2)



5.   Disconnect vacuum hose (1) from fuel pressure regulator (2)
•   pull off



6.   Disconnect vacuum hose (1) for switchover valve from throttle body
•   pull off



7.   Disconnect 2x coolant hose (1) from throttle body
•   Install 2x clamp to 2x hose (1)
•   Position collecting container under 2x hose
•   Release 2x clip (2)



8.   Disconnect vacuum hose (1) for fuel purge valve from intake manifold
•   Release clip (2)
•   Release clip (3)



9.   Disconnect harness connector (1) from throttle position sensor
•   Release integral clip



10.   Remove throttle body (1) from intake manifold
•   Remove 4x bolt (2)



Important:   Fuel may escape – observe the safety regulations and relevant national legislation.
11.   Release fuel pressure from fuel system
•   Remove protective cap
•   Release fuel pressure using Pressure Tester KM-J-34730-91 (1)
•   Connect test connection hose (2) to fuel rail
– Collect any escaping fuel in a suitable container
12.   Remove Pressure Tester KM-J-34730-91 (1)
•   Disconnect test connection hose (2) from fuel rail
•   Install protective cap



13.   Disconnect breather hose (1) from cylinder head cover
•   Release clip (2)
14.   Release breather hose (1) from lifting eye
•   Remove, then discard cable tie (3)



15.   Release fuel rail assembly (3) from intake manifold
Remove 2x bolt (1)
16.   Remove cover (2) from housing for injector harness
•   Release 10x integral clip



17.   Remove 4x injector (2) from fuel rail assembly
•   Remove 4x clip (1)
•   Remove, then discard 8x O ring
•   Position fuel rail assembly to one side



18.   Remove upper support bracket (2) for alternator with support bracket for harness connector of crankshaft position sensor
•   Remove 3x bolt (1)



19.   Remove side support bracket (2) for alternator with support clip for vacuum hose of purge valve
•   Remove 2x bolt (1), washer



20.   Disconnect 2x earth connector (2) from intake manifold
•   Remove bolt (1)



21.   Disconnect vacuum pipe (1) for brake servo from intake manifold
•   Release integral clip



22.   Raise vehicle
23.   Remove exhaust undertray
•   Remove 12x bolt, 12x washer, 2x clip
24.   Release support bracket (2) for vacuum tank of switchover valve from intake manifold
•   Remove bolt (1)



25.   Lower vehicle
26.   Remove intake manifold from vehicle
•   Remove bolt (1)
•   Remove, then discard 7x nut (2) and gasket
•   Clean all mating surfaces



  Install

27.   Install intake manifold with new gasket to vehicle
•   Install, then secure bolt, 7x new nut 22 Nm
28.   Raise vehicle
29.   Secure support bracket for vacuum tank of switchover valve to intake manifold
•   Install, then secure bolt 25 Nm
30.   Install exhaust undertray
•   Install, then secure 12x bolt, 12x washer, 2x clip
31.   Lower vehicle
32.   Connect vacuum pipe for brake servo to intake manifold
•   Secure integral clip
33.   Connect 2x earth connector to intake manifold
•   Install, then secure bolt
34.   Install side support bracket for alternator with support clip for vacuum hose of purge valve
•   Install, then secure 2x bolt, washer
35.   Install upper support bracket for alternator with support bracket for harness connector of crankshaft position sensor
•   Install, then secure 3x bolt
36.   Install 8x new O ring to 4x injector
37.   Install 4x injector to fuel rail assembly
•   Install 4x clip
38.   Secure fuel rail assembly to intake manifold
•   Install, then secure 2x bolt 8 Nm
39.   Install cover to housing for injector harness
•   Secure 10x integral clip
40.   Connect breather hose to cylinder head cover
•   Secure clip
41.   Secure breather hose to lifting eye
•   Install new cable tie
42.   Install throttle body to intake manifold
•   Install 2x new sealing ring
•   Install, then secure 4x bolt 9 Nm
43.   Connect harness connector to throttle position sensor
•   Secure integral clip
44.   Connect vacuum hose for fuel purge valve to intake manifold
•   Secure 2x clip
45.   Connect 2x coolant hose to throttle body
•   Remove 2x clamp from 2x hose
•   Secure 2x clip
•   Remove collecting container from under 2x hose
46.   Connect vacuum hose for switchover valve to throttle body
•   push on
47.   Connect vacuum hose to fuel pressure regulator
•   push on
48.   Connect vacuum hose to cylinder head cover
•   Secure clip
49.   Connect intercooler hose to connecting pipe for throttle body
•   Secure clip
50.   Connect battery negative lead
•   Secure nut and bolt
51.   Secure screen wash reservoir to support bracket
•   Push on
52.   Check coolant level, then top up if required
53.   program all volatile memories
