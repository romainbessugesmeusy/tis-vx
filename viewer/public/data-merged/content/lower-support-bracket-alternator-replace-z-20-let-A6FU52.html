Lower Support Bracket, Alternator, Replace (Z 20 LET)

  Remove

1.   Disconnect battery negative lead
•   Release windscreen wash reservoir, then move to one side
•   Remove nut, bolt
2.   Disconnect intercooler hose (1) from throttle body
•   Release clip
3.   Release top support bracket from alternator
•   Remove bolt (3)
4.   Release side support bracket from alternator
•   Remove bolt (2)
•   Release bolt from intake manifold
Note:   Do not remove bolt.



Important:   A four-post hoist must be used to raise the vehicle when removing the engine undertray. The vehicle can be moved to a two-post hoist for other operations if required.
5.   Remove engine undertray
•   Raise vehicle
•   Remove 13x bolt, 13x washer
6.   Remove ribbed V-belt
•   Release ribbed V-belt tension
– Using suitable spanner rotate tensioner (1) clockwise, then Install suitable locking pin (2)



7.   Disconnect 2x harness connector from alternator
•   Remove 2x nut (1), 2x washer
8.   Disconnect harness connector (4) from oil pressure switch
•   Release integral clip
9.   Remove alternator with lower support bracket from engine
•   Remove 2x bolt (2)
•   Remove harness support bracket
– Release 2x oil pipe from harness support bracket
– Release harness for alternator from clip on harness support bracket
10.   Remove lower support bracket from alternator
•   Remove bolt (3), nut



  Install

11.   Install lower support bracket to alternator
•   Install nut, bolt (2)
•   Do not tighten
12.   Install alternator with lower support bracket to engine
•   Install 2x bolt (1) 35 Nm
•   Install harness support bracket
– Secure 2x oil pipe to harness support bracket



13.   Lower vehicle
14.   Install top support bracket to alternator
•   Install bolt (3)
•   Do not tighten
15.   install side support bracket to alternator
•   Install bolt (2)
•   Do not tighten
16.   Secure top support bracket to alternator
•   Secure bolt 25 Nm
17.   Secure side support bracket to alternator
•   Secure 2x bolt 25 Nm
18.   Connect intercooler hose (1) to throttle body
•   Secure clip



19.   Raise vehicle
20.   Secure lower support bracket to alternator
•   Secure nut, bolt (2) 35 Nm
21.   Connect 2x harness connector to alternator
•   Install 2x nut (1), 2x washer
22.   Connect harness connector (3) to oil pressure switch
•   Secure integral clip
•   Secure harness for alternator to clip on harness support bracket



23.   Install ribbed V-belt
•   Using suitable spanner rotate tensioner (1) clockwise, then remove locking pin (2)



24.   Install engine undertray
25.   Lower vehicle
26.   Connect battery negative lead
•   Install nut, bolt
•   Secure windscreen wash reservoir
27.   Program all volatile memories
