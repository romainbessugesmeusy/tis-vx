Shock Absorber, Remove and Install

  Remove

1.   Remove appropriate front road wheel
•   Remove 4x bolt, remove security bolt
– Use special socket located under front bonnet



2.   Position suitable support beneath lower wishbone
3.   Remove shock absorber from chassis bracket
•   Nut, bolt, 2x washer, 2x spacer



4.   Remove shock absorber from lower wishbone
•   Nut, bolt, 2x washer



  Install

5.   Install shock absorber to lower wishbone
•   New nut 45 Nm , bolt, 2x washer
6.   Install shock absorber to chassis bracket
•   New nut 45 Nm , bolt, 2x washer, 2x spacer
7.   Install upper shock absorber assembly to chassis bracket
•   New nut 45 Nm , bolt, 2x washer, 2x spacer
