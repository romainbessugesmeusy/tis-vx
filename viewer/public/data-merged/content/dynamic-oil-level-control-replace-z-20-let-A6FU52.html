Dynamic Oil Level Control, Replace (Z 20 LET)

  Remove

Important:   A four-post hoist must be used to raise the vehicle when removing exhaust pipe undertray. The vehicle can be moved to a two-post hoist for other operations if required.
1.   Remove engine undertray (1)
•   Raise vehicle
•   Remove 13x bolt, 13x washer



2.   Remove intercooler lower hose
•   Release 2x clip
– Remove, then discard cable tie
3.   Disconnect engine harness connector from oil level sensor
•   Release integral clip
Important:   Observe relevant environmental regulations when disposing of used coolant and lubricants.
4.   Drain engine oil
•   Position collecting container
•   Remove oil drain plug (1)
•   Remove, then discard sealing ring
5.   Remove lower oil pan
•   Release oil level sensor
– Remove clip
•   Remove 16x bolt (2)
– Remove, then discard gasket



6.   Remove oil level sensor from lower oil pan
•   Remove, then discard 2x bolt (1)



  Install

7.   Install oil level sensor to lower oil pan
•   Install 2x new bolt 8 Nm
8.   Install lower oil pan to engine
•   Install new gasket
•   Install oil level sensor
– Install clip
•   Install 16x bolt 8 Nm
Note:   2x technician required.
9.   Connect engine harness to oil level sensor
•   Secure integral clip
10.   Install intercooler lower hose
•   Install 2x clip
– Install new cable tie
11.   Install oil drain plug 10 Nm
•   Install new sealing ring
12.   Install engine undertray
•   Install 13x bolt, 13x washer
•   Lower vehicle
13.   Replenish engine oil
14.   Run engine until oil light goes out
•   Check oil level, top up if required
