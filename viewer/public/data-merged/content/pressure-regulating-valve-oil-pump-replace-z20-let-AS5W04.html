Pressure Regulating Valve, Oil Pump, Replace (Z20 LET)

  Remove

Important:   A four post hoist must be used to raise the vehicle when removing the engine undertray. The vehicle can be moved to a two post hoist for other operations if required.
1.   Remove engine undertray (1)
•   Raise vehicle
•   Remove 13x bolt, 13x washer



2.   Release turbo oil feed pipe from engine
•   Remove bolt
– Position collecting container
Important:   Observe relevant environmental regulations when disposing of used coolant and lubricants.
3.   Remove pressure regulating valve for oil pump (1)
Note:   Note position for installation.
•   Remove closure plug (5)
•   Remove, then discard sealing washer (4)
•   Remove spring (3)
•   Remove valve (2)



  Install

4.   Install pressure regulating valve for oil pump
Note:   Note position for installation.
•   Install valve
•   Install spring
•   Use new sealing washer
•   Install closure plug 50 Nm
5.   Secure turbo oil feed pipe to engine
•   Install bolt 25 Nm
– Remove collecting container
6.   Install engine undertray
•   Install 13x bolt, 13x washer
7.   Lower vehicle
8.   Run engine and check for leaks, stop engine
•   Check engine oil level, top up if required
