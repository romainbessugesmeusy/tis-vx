Track Control Arm, Remove and Install

  Remove

1.   Remove rear brake pads from caliper - see operation "Brake Pads, Rear, Replace" in group H



2.   Remove track control arm from hub carrier
Note:   Illustration shows wheel hub and drive shaft removed for clarity
•   Remove nut, conical spacer, spacer
3.   Release track control arm from lower wishbone
•   Remove nut



  Install

Important:   End of track control arm locates into chassis. Ensure correct new locking nut is used. Early type have coarse thread, later type have fine thread.
4.   Install new track control arm to lower wishbone
•   Install new nut 45 Nm
5.   Install track control arm to hub carrier
•   Install spacer, conical spacer, new nut 45 Nm
6.   Install rear brake pads to caliper - see operation "Brake Pads, Rear, Replace" in group H
