Immobiliser Control Unit, Replace

  Remove

1.   Disconnect battery negative lead
•   Release windscreen washer bottle (1), position to side
•   Release nut, bolt (2)



2.   Remove air bag unit from steering wheel
•   Remove 2x screw
•   Release locking clip
•   Disconnect harness connector
Note:   Illustration shows steering wheel removed for clarity



3.   Disconnect horn from combination switch assembly
•   Disconnect harness connector
4.   Remove steering wheel
•   Remove 4x bolt, 4x washer



5.   Remove upper cover from steering column
•   Remove 6x screw
6.   Remove lower cover from steering column
•   Remove 3x screw
•   Remove clip
•   Disconnect 2x harness connector
7.   Remove locking ring (2)
•   Pull off
8.   Remove bracket (3)
•   Slide off
9.   Remove immobiliser control unit (1) from steering column
•   Disconnect instrument harness connector
– Release integral clip



  Install

10.   Install immobiliser control unit to steering column
•   Connect instrument harness connector
– Secure integral clip
11.   Install bracket
•   Slide on
12.   Install locking ring
•   Push fit
13.   Install lower cover to steering column
•   Install 3x screw
•   Install clip
•   Connect 2x harness connector
14.   Install upper cover to steering column
•   Install 6x screw
15.   Install steering wheel
•   Install 4x bolt, 4x washer
– Apply locking compound
16.   Connect horn to combination switch assembly
•   Connect harness connector
17.   Install air bag unit
•   Install 2x screw 7 Nm
•   Connect harness connector
18.   Connect battery negative lead
•   Secure bolt, nut
•   Secure windscreen wash bottle
19.   Program all volatile memories
