Bowden Cable, Air Distribution Housing, Replace

  Remove

1.   Remove hard top or soft top
2.   Disconnect battery negative lead
•   Release windscreen wash reservoir, then move to one side
•   Release nut, bolt
3.   Remove instrument pad assembly – see operation ”Instrument Pad Assembly, Remove and Install”
4.   Remove wiper motor cover (1)
•   Remove 2x screw (2)



5.   Release harness connector (1) from wiper motor



6.   Disconnect bowden cable (1) from lever on air distribution housing
•   Prise off clip (2)



7.   Remove bowden cable (1)
•   Release screw (3)
•   Pull off from air direction control (4)
•   Release grommet (2)



  Install

8.   Transfer grommet to new bowden cable
9.   Install new bowden cable
•   Secure grommet
•   Push on to air direction control
•   Secure screw
10.   Adjust bowden cable
•   Connect inner bowden cable to lever on air distribution housing
•   Turn knob for air direction control fully clockwise (air flow directed to windscreen)
•   Ensure lever on air distribution housing is in fully downward position (flap allowing air flow to footwell is in fully closed position)
•   Secure outer bowden cable in position
– Push on clip
•   Check operation
11.   Secure harness connector to wiper motor
12.   Install wiper motor cover
•   Install 2x screw
13.   Install instrument pad assembly – see operation ”Instrument Pad Assembly, Remove and Install”
14.   Connect battery negative lead
•   Secure nut, bolt
15.   Install windscreen wash reservoir
•   Push fit
16.   Program all volatile memories
17.   Install hard top or soft top
