Cylinder Head Cover, Replace (Z 20 LET)

  Remove

1.   Disconnect harness connector (1) from ignition module
2.   Remove ignition module cover (2) from cylinder head
•   Slide off



3.   Remove ignition module from cylinder head
•   Remove 2x bolt (1)



4.   Disconnect harness connector from camshaft sensor (1)
5.   Disconnect 3x breather hose (2) from cylinder head cover (3)
6.   Remove upper cover of camshaft timing belt - see operation “Camshaft Timing Belt, Upper Cover, Replace (Z 20 LET)”
7.   Remove cylinder head cover
•   Remove 10x bolt
•   Remove, then discard 10x O ring
•   Remove, then discard gasket



  Install

8.   Install cylinder head cover
•   Install new gasket
•   Install 10x O ring
•   Install 10x bolt 8 Nm
9.   Install upper cover of camshaft timing belt - see operation “Camshaft Timing Belt, Upper Cover, Replace (Z 20 LET)”
10.   Connect 3x breather hose (2) to cylinder head cover (3)
11.   Connect harness connector to camshaft sensor (1)
12.   Install ignition module to cylinder head
•   Install 2x bolt 8 Nm
13.   Install ignition module cover to cylinder head
•   Slide on
14.   Connect harness connector to ignition module
