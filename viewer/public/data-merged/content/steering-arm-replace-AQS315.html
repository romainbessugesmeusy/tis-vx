Steering Arm, Replace

  Remove

1.   Chock both rear road wheels
2.   Raise and support vehicle
3.   Remove front road wheel
  • Remove 4x bolt, remove security bolt
    • Use special socket located under front bonnet
4.   Remove brake caliper assembly from hub carrier
•   Remove 2x bolt



5.   For earlier type wheel hub, remove brake disc
•   Remove screw



6.   For later type wheel hub, remove brake disc
•   Remove screw



7.   Release harness connector from wheel speed sensor
•   Release clip



8.   For earlier type wheel hub assembly, remove from hub carrier
•   Remove centre bolt, 4x washer, nut
•   Remove 3x bolt, 3x washer



9.   For later type wheel hub assembly, remove from hub carrier
•   Remove 3x bolt, 3x washer



10.   Release lower wishbone from hub carrier
•   Remove nut (1)



11.   Remove hub carrier and shims
•   Remove 2x bolt



  Install

12.   Install hub carrier and shims
•   Install 2x bolt 45 Nm
Note:   Clean threads and apply locking compound.
13.   Secure lower wishbone to hub carrier
•   Install new nut 45 Nm
14.   Install wheel hub assembly to hub carrier
•   Install 3x washer, 3x bolt 77 Nm
15.   For earlier type wheel hub, install centre bolt, 4x washer, nut 45 Nm
16.   Secure harness connector to wheel speed sensor
•   Secure clip
17.   Install brake disc to wheel hub
•   Install screw
18.   Install brake caliper assembly to hub carrier
•   Install 2x bolt 45 Nm
Note:   Clean threads and apply locking compound.
19.   Install front road wheel
•   Install 4x bolt, install security bolt 90 Nm
20.   Raise vehicle and remove support
21.   Lower vehicle
22.   Remove chocks from both rear road wheels
