Fuel Tank Sensor, Replace

1.   Release wash reservoir from support bracket, then move to one side



2.   Disconnect battery negative lead
•   Release nut



3.   Disconnect 2x ball joint (1) from 2x selector rod
4.   Release support bracket, then move to one side
•   Remove bolt (2)



5.   Remove engine undertray (1) from vehicle
•   Remove 13x bolt, 13x washer



6.   Remove 2x rear road wheel from vehicle
•   Remove 2x security bolt
•   Remove 8x bolt
7.   Release parking brake cable from compensator, then move parking brake cable to one side
•   Remove pin
– Remove clip (1)



8.   Remove guide bracket (1) from fuel tank lower panel
•   Remove 4x bolt



9.   Remove 2x lower bolt (1) from lower wishbone mounting



10.   Remove 2x bolt (1) from engine damping block



11.   Remove fuel tank lower panel (1)
•   Remove 5x bolt, 5x washer
•   Remove 3x nut, 3x washer, 3x bolt



12.   Release 3x fuel line from bulkhead
•   Remove 3x clip
13.   Release fuel filter from bulkhead
•   Remove bolt (1)



14.   Remove rear wheelarch liner from right side
•   Remove 6x screw



15.   Release fuel evaporation tank, then move to one side
•   Remove nut, bolt, cable tie



16.   Disconnect 3x hose (1) from fuel evaporation tank
Note:   Apply alignment marks to 3x hose to aid installation
•   Remove 3x clip
17.   Disconnect hose (2) from fuel filler pipe
•   Remove 2x clip



18.   Raise vehicle to provide increased access
19.   Support fuel tank to prevent damage to fuel pipes and wiring harness
20.   Remove fuel tank from vehicle
•   Disconnect 2x fuel pipe from fuel tank sensor
•   Disconnect harness connector from fuel tank sensor
•   Remove 2x fuel pipe plastic sleeve from fuel tank sensor
•   Remove 4x bolt, 4x lock washer
21.   Remove fuel tank sensor from fuel tank
•   Release circlip
•   Remove sealing ring



22.   Install fuel tank sensor to fuel tank
•   Install new sealing ring
•   Secure circlip
23.   Install fuel tank to vehicle
•   Connect 2x fuel pipe to fuel tank sensor
•   Connect harness connector to fuel tank sensor
•   Install 2x fuel pipe plastic sleeve to fuel tank sensor
•   Install 4x new bolt, 4x lock washer 25 Nm
24.   Connect 3x hose to fuel evaporation tank
•   Align 3x hose to position noted in step 16
•   Secure 3x clip
25.   Connect hose to fuel filler pipe
•   Secure 2x clip
26.   Install fuel evaporation tank
•   Secure new cable tie, bolt, nut
27.   Install rear wheelarch liner to right side
•   Secure 6x screw
28.   Install fuel filter to bulkhead
•   Secure bolt 9 Nm
•   Install 3x fuel line
•   Secure 3x clip
29.   Install fuel tank lower panel
•   Install 5x bolt, 5x washer 25 Nm
•   Install 3x bolt, 3x washer, 3x nut 45 Nm
30.   Install 2x bolt to engine damping block 25 Nm
31.   Install 2x bolt to lower wishbone mounting 45 Nm
32.   Install guide bracket for gearshift cables to fuel tank lower panel
•   Secure 4x bolt
33.   Connect parking brake cable to compensator
•   Install pin
– Install clip
34.   Install 2x rear road wheel
•   Secure 8x bolt, 2x security bolt 90 Nm
35.   Install engine undertray to vehicle
•   Secure 13x bolt, 13x washer
36.   Install support bracket for gearshift cable
•   Secure bolt
37.   Connect 2x ball joint of 2x gearshift cable to 2x selector rod
38.   Connect battery negative lead
•   Secure nut
39.   Secure wash reservoir to support bracket
40.   Program volatile memories
