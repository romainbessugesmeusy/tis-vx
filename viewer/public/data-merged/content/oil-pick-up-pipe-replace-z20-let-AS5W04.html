Oil Pick-up Pipe, Replace (Z20 LET)

1.   Remove upper oil pan - see operation “Oil Pan, Upper, Replace (Z 20 LET)”
2.   Remove oil pick-up pipe from engine
•   Remove 2x bolt from oil pump (1)
•   Remove 2x bolt from balancer shaft unit (2)
•   Remove, then discard O ring



3.   Install new oil pick-up pipe to engine
•   Install 2x new bolt to oil pump 8 Nm
•   Install 2x new bolt to balancer shaft unit 8 Nm
•   Install new O ring
4.   Install oil pan upper - see operation “Oil Pan, Upper, Replace (Z 20 LET)”
