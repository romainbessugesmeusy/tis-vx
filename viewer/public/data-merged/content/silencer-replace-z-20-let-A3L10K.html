Silencer, Replace (Z 20 LET)

  Remove

1.   Disconnect oxygen sensor for secondary catalytic converter
•   Disconnect engine harness connector (1)



Important:   A four-post hoist must be used to raise the vehicle when removing the engine undertray and exhaust pipe undertray. The vehicle can be moved to a two-post hoist for other operations if required.
2.   Remove engine undertray (1)
•   Raise vehicle
•   Remove 13x bolt, 13x washer
3.   Remove exhaust pipe undertray (2)
•   Remove 12x bolt, 12x washer, 2x clip



4.   Release harness for secondary catalytic converter
•   Release 2x clip from left subframe for oxygen sensor
•   Release clip from rear silencer heatshield
5.   Remove secondary catalytic converter (1) for exhaust
•   Remove 3x front nut (3), then discard
•   Remove 3x rear nut (2), then discard
– Release rubber mount bracket
•   Release rubber mount
•   Remove gasket (4) from front joint, then discard
Note:   Illustration shows exhaust removed for clarity.



6.   Remove intermediate pipe and heatshield from rear silencer
•   Release clamp (1)



7.   Release rear silencer, left support bracket from silencer
•   Remove 2x nut (1), then discard
8.   Remove rear silencer, left flexible support from subframe
•   Remove 2x bolt (2)
•   Remove clamp
9.   Release rear silencer, right support bracket from silencer
•   Remove 2x nut (1), then discard
10.   Remove rear silencer, right flexible support from subframe
•   Remove 2x bolt (2)
•   Remove clamp
Note:   Illustration shows left side only.



11.   Remove silencer from vehicle
•   Remove intermediate pipe clamp from silencer
– Pull off



  Install

12.   Install silencer to vehicle
•   Install intermediate pipe clamp to silencer
– Push fit
13.   Install rear silencer, right flexible support to subframe
•   Install clamp
•   Install 2x bolt
14.   Secure rear silencer, right support bracket to silencer
•   Install 2x new nut
15.   Install rear silencer, left flexible support to subframe
•   Install clamp
•   install 2x bolt
16.   Secure rear silencer, left support bracket to silencer
•   Install 2x new nut
17.   Install intermediate pipe and heatshield to rear silencer
18.   Install secondary catalytic converter for exhaust
•   Install new gasket to front joint
•   Install 3x new front nut
– Position bracket for rubber mount
•   Install 3x new rear nut
•   Position rubber mount
19.   Install harness to rear silencer heatshield for oxygen sensor installed to secondary catalytic converter
•   Secure clip to heatshield
•   Install harness to subframe
•   Secure 2x clip
20.   Secure clamp for intermediate pipe
21.   Install engine undertray
•   Remove 13x bolt, 13x washer
22.   Install exhaust pipe undertray
•   Install 12x bolt, 12x washer, 2x clip
23.   Lower vehicle
24.   Connect oxygen sensor for secondary catalytic converter to engine harness
•   Connect harness connector
