Programming okay?
Yes: E002 No: E003


Result

Cause Of Fault
E001 Result: Defective Component
  • Defective component:
    A17 Control Unit - Immobiliser

Important:

Reset concerned control unit (engine or immobiliser control unit) with diagnostic tester before replacing. Select immobiliser in the diagnostic tester and call up the corresponding test in the menu ADDITIONAL FUNCTIONS. Ensure that both control units are never reset and replaced at the same time.
Repair done
E002 Result: Programming
  • Previous programming was faulty

or

  • If programming/nominal value is okay and system is still faulty the following component is defective:
    A17 Control Unit - Immobiliser

Important:

Reset concerned control unit (engine or immobiliser control unit) with diagnostic tester before replacing. Select immobiliser in the diagnostic tester and call up the corresponding test in the menu ADDITIONAL FUNCTIONS. Ensure that both control units are never reset and replaced at the same time.
Repair done
E003 Result: Defective Component
  • Defective component:
    K117 Control Unit - Immobiliser
    or
    A4 Control Unit - Multec

Important:

Reset concerned control unit (engine or immobiliser control unit) with diagnostic tester before replacing. Select immobiliser in the diagnostic tester and call up the corresponding test in the menu ADDITIONAL FUNCTIONS. Ensure that both control units are never reset and replaced at the same time.
Repair done
