Covers, Steering Column, Replace

  Remove

1.   Open bonnet
2.   Remove windscreen wash reservoir from mounting bracket
3.   Disconnect battery negative lead
•   Remove nut
4.   Remove airbag assembly from steering wheel
•   Remove 2x bolt (1)
•   Release harness connector



5.   Disconnect harness connector for horn
6.   Remove steering wheel from steering column
•   Remove 4x bolt
7.   Remove upper cover from steering column
•   Remove 6x screw (1)



8.   Remove lower cover from steering column
•   Remove 3x screw (2), remove clip
•   Release harness connector

  Install

9.   Install lower cover to steering column
•   Secure harness connector
•   Install 3x screw, install clip
10.   Install upper cover to steering column
•   Install 6x screw
11.   Install steering wheel to steering column
•   Install 4x bolt
12.   Connect harness connector for horn
13.   Install air bag assembly to steering wheel
•   Secure harness connector
•   Install 2x bolt
14.   Connect battery negative lead
•   Install nut
15.   Install windscreen wash reservoir to mounting bracket
16.   Close bonnet
17.   Program all volatile memories
