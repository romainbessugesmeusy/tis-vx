Crankshaft Oil Seal, Flywheel Side, Replace (Z 20 LET)

  Remove

1.   Remove engine with transmission - see operation “Engine with Transmission, Remove and Install (Z 20 LET)”
2.   Remove turbocharger heat shield (1) from engine and transmission assembly
•   Remove 2x bolt
Important:   When removing transmission observe relevant safety measures and national legislation.
3.   Remove transmission (2) from engine assembly
•   Remove 11x bolt
Note:   Includes removal of support bracket for engine harness.



4.   Remove clutch plate with pressure plate (2) from flywheel
•   Immobilize flywheel
– Install Flywheel Holder KM-652 (1) to engine block
•   Remove 6x bolt, then discard



5.   Remove flywheel (1) from crankshaft
•   Remove 6x bolt, then discard



6.   Remove Flywheel Holder KM-652 from engine block
7.   Remove seal ring (1) from crankshaft
•   Prise out using suitable tool



  Install

8.   Install seal ring to crankshaft
•   Use Seal Ring Installer KM-635 with Needle Bearing Installer KM-535
•   Apply oil to seal lip
9.   Install flywheel to crankshaft
•   Install 6x new bolt
– Apply locking compound
– Do not tighten
10.   Immobilize flywheel
•   Install Flywheel Holder KM-652 to engine block
11.   Secure flywheel to crankshaft
•   Secure 6x bolt 65 Nm + 30° + 15° symmetrically
12.   Install clutch plate with pressure plate to flywheel
•   Install 6x new bolt 15 Nm symmetrically
13.   Remove Flywheel Holder KM-652 from engine block
Important:   When installing transmission observe safety measures and national legislation.
14.   Install transmission to engine
•   Install 7x bolt (1) 60 Nm symmetrically
Note:   Includes installation of support bracket for engine harness.
•   Install 4x bolt (2) 40 Nm symmetrically



15.   Install turbocharger heat shield to engine and transmission assembly
•   Install 2x bolt
16.   Check oil level and top-up if required
