Coolant Pump, Replace (Z 20 LET)

  Remove

Important:   A four-post hoist must be used to raise the vehicle when removing the engine undertray and exhaust pipe undertray. The vehicle can be moved to a two-post hoist for other operations if required.
1.   Remove engine undertray (1)
•   Raise vehicle
•   Remove 13x bolt, 13x washer



2.   Lower vehicle
3.   Disconnect battery negative lead
•   Release windscreen washer bottle (1), position to one side
•   Release nut, bolt (2)



4.   Disconnect intercooler hose (1) from throttle body
•   Release clip (2)



5.   Raise vehicle
Important:   Observe relevant environmental regulations when disposing of used coolant and lubricants.
6.   Drain cooling system
•   Position collecting container
•   Disconnect lower coolant hose from radiator feed pipe
•   Release clip (1)
– Remove, then discard cable tie (2)
– Remove collecting container when drained



7.   Lower vehicle to suitable working height
8.   Remove rear road wheel from right side
•   Remove security bolt
– Use special socket located under front bonnet
•   Remove 4x wheel bolt
9.   Remove wheel arch liner, right rear
•   Remove 4x screw, 2x clip
10.   Remove intercooler hose from intercooler
•   Release clip (1)
•   Remove, then discard 4x cable tie



11.   Remove ribbed V-belt - see operation “Ribbed V-belt, Replace (Z 20 LET)”
12.   Remove V-belt pulley from crankshaft
•   Remove 4x bolt
13.   Remove upper cover (1) for camshaft timing belt
•   Remove 2x bolt (2)
14.   Remove V-belt tensioner (4)
•   Remove 2x bolt
15.   Remove lower cover (3) for camshaft timing belt
•   Remove 2x bolt (5)



16.   Rotate crankshaft until pistons 1 and 4 are at TDC
•   Check camshaft alignment marks (1)
•   Check crankshaft alignment marks (2)
•   Install Camshaft Locking Tool KM-853



17.   Remove tensioner pulley for camshaft timing belt
•   Remove locking bolt (1)
•   Position camshaft timing belt to side
18.   Remove coolant pump
•   Remove 3x bolt (2)
•   Remove, discard O ring



  Install

Important:   Clean sealing surfaces, apply silicon grease
19.   Install coolant pump
•   Install 3x bolt 25 Nm
•   Install new O ring
20.   Install tensioner pulley for camshaft timing belt
•   Install bolt, hand tighten
•   Install camshaft timing belt to pulleys
21.   Adjust camshaft timing belt - see operation “Camshaft Timing Belt Tension, Check and Adjust (Z 20 LET)”
•   Remove Camshaft Locking Tool KM-853
22.   Check crankshaft and camshaft timing marks for alignment
•   Rotate engine two complete revolutions in direction of rotation to Top Dead Centre (TDC) cylinder No.1, then check timing marks are aligned. if timing marks are not aligned, then repeat adjustment procedure
•   Install Camshaft Locking Tool KM-853
23.   Secure adjustment pulley 20 Nm for camshaft timing belt
•   Remove Camshaft Locking Tool KM-853
24.   Install lower casing for camshaft timing belt
•   Install 2x bolt 6 Nm
25.   Install V-belt tensioner
•   Install 2x bolt 25 Nm
26.   Install crankshaft V-belt pulley
•   Install 4x bolt 20 Nm
27.   Install V-belt to pulleys
•   Release tension from tensioner
– Remove locking pin
28.   Install upper cover for camshaft timing belt
•   Install 2x bolt 6 Nm
29.   Install intercooler hose to intercooler
•   Install clip
•   Install 4x new cable tie
30.   Install inner wheel arch, right rear
•   Install 4x screw, 2x clip
31.   Install rear road wheel to right side
•   Install 4x bolt, security bolt
•   Do not tighten
32.   Raise vehicle
33.   Connect lower coolant hose to radiator feed pipe
•   Install clip
•   Install new cable tie
34.   Lower vehicle
35.   Secure rear road wheel
•   Tighten 4x bolt, security bolt 90 Nm
36.   Connect intercooler hose to throttle body
•   Install clip
37.   Connect battery negative lead
•   Secure nut, bolt
•   Secure windscreen wash bottle
38.   Replenish and bleed cooling system - see operation “Cooling system, Drain, Fill and Bleed (Z 20 LET)”
39.   Program all volatile memories
40.   Raise vehicle
41.   Install engine undertray
•   Install 13x bolt, 13x washer
42.   Lower vehicle
