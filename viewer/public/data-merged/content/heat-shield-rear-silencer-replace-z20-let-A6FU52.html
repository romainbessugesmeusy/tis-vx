Heat Shield, Rear Silencer, Replace (Z20 LET)

  Remove

1.   Remove rear clamshell - see operation “Rear Clamshell, Remove and Install”
2.   Release intermediate pipe from secondary catalytic converter
•   Remove 3x nut (1), then discard
•   Release mounting bracket



3.   Release heat shield for rear silencer from rear subframe
•   Remove 3x bolt, 3x spring washer, 3x flat washer, clip
4.   Remove rear silencer and heat shield from rear subframe
•   Remove 6x bolt (1)
•   Remove 4x bolt, 2x clamp (2)



5.   Remove intermediate pipe from rear silencer
•   Release clamp (1)



6.   Remove rear silencer from heat shield
•   Pull out

  Install

7.   Install rear silencer to new heat shield
•   Push fit
8.   Install intermediate pipe to rear silencer
•   Secure clamp
9.   Install rear silencer and heat shield to rear subframe
•   Install 6x bolt
•   Install 4x bolt, 2x clamp
10.   Secure heatshield for rear silencer to rear subframe
•   Install 3x bolt, 3x spring washer, 3x flat washer, clip
11.   Secure intermediate pipe to secondary catalytic converter
•   Secure mounting bracket
•   Install 3x new nut
12.   Install rear clamshell - see operation “Rear Clamshell, Remove and Install”
