Engine Control Unit, Replace (Z 20 LET)

  Remove

1.   Disconnect battery negative lead
•   Release windscreen wash reservoir, then move to one side
•   Remove nut, bolt
2.   Release, then disconnect 2x harness connector (1) from engine control unit (2)



3.   Remove engine control unit from support bracket
Note:   Engine control unit with support bracket shown removed from vehicle for clarity.
•   Remove nut, washer (1)
– Discard nut
•   Remove bolt, 2x washer (2)
•   Remove bolt, clamp (3)
•   Disconnect negative lead (4) from engine control unit
– Remove nut, washer
– Discard nut



  Install

4.   Install engine control unit to support bracket
•   Connect negative lead to engine control unit
– Install new nut, washer
•   Install bolt, clamp
•   Install bolt, 2x washer
•   Install new nut, washer
5.   Connect, then secure 2x harness connector to engine control unit
6.   Connect battery negative lead
•   Install nut, bolt
•   Secure windscreen wash reservoir
7.   Program all volatile memories
