Bowden Cable, Heater, Replace

  Remove

1.   Remove hard top or soft top
2.   Disconnect battery negative lead
•   Release windscreen wash reservoir, then move to one side
•   Release nut, bolt
3.   Remove air intake pipe (1) from heater assembly and air distribution housing
•   Remove mesh grille (if installed) from air distribution housing
•   Pull off



4.   Disconnect bowden cable (1) from lever on heater assembly
•   Prise off clip (2)



5.   Release bowden cable from bulkhead
•   Remove clip (1)
– Remove screw (2)



6.   Remove instrument pad assembly – see operation ”Instrument Pad Assembly, Remove and Install”
7.   Remove bowden cable (1)
•   Remove cable tie (2), then discard
•   Release screw (4)
•   Pull off from air temperature control (5)
•   Release grommet (3)



  Install

8.   Transfer grommet to new bowden cable
9.   Install bowden cable
•   Secure grommet
•   Push on to air temperature control
•   Secure screw
•   Install new cable tie
10.   Adjust bowden cable
•   Connect inner bowden cable to lever on heater assembly
•   Turn knob for air temperature control fully anti-clockwise (cold air position)
•   Ensure lever on heater assembly in fully closed position
•   Secure outer bowden cable in position
– Push on clip
•   Check operation
11.   Install instrument pad assembly – see operation ”Instrument Pad Assembly, Remove and Install”
12.   Secure bowden cable to bulkhead
•   Install clip
– Install screw
13.   Install air intake pipe to heater assembly and air distribution housing
•   Install mesh grille (if installed) to air distribution housing
•   Push on
14.   Connect battery negative lead
•   Secure nut, bolt
15.   Install windscreen wash reservoir
•   Push fit
16.   Program all volatile memories
17.   Install hard top or soft top
