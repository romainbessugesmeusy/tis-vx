Rear Wheel Arch Liner, Remove and Install

  Remove

1.   Secure both front wheels using chocks
2.   Raise and support the vehicle
3.   Remove the rear wheel
•   Remove 4 bolts, remove the locking bolt
– Use special socket under the front bonnet
4.   Remove wheel arch liner
•   Remove 6 screws

  Install

5.   Install wheel arch liner
•   Install 6 screws
6.   Install rear wheel
•   Install 4 bolts, install locking bolt 90 Nm
7.   Remove support and lower vehicle
8.   Remove chocks
