Radiator Grille, Replace

  Remove

1.   Remove front clamshell - see operation “Front Clamshell, Remove and Install”
2.   Remove front emblem from radiator grille - see operation “Emblem, Front, Replace”
3.   Roughly cut out radiator grille (1)
•   Use suitable tool
4.   Cut off radiator grille flange from front clamshell
5.   Cut off residues
6.   Trim down adhesive beads



  Install

7.   Install new radiator grille to clamshell
•   Use suitable bonding adhesive
8.   Install front emblem to radiator grille - see operation “Emblem, Front, Replace”
9.   Install front clamshell - see operation “Front Clamshell, Remove and Install”
