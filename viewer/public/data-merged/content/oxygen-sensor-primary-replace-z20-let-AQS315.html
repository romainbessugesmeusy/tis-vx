Oxygen Sensor, Primary, Replace (Z20 LET)

  Remove

1.   Disconnect harness connector for primary oxygen sensor (1) from engine harness
•   Release integral clip
•   Release cable tie, then discard



2.   Remove oxygen sensor (1) from primary catalytic converter
•   Use KM-6326



  Install

3.   Install oxygen sensor to primary catalytic converter
•   Use KM-6326
4.   Connect harness connector for primary oxygen sensor to engine harness
•   Secure integral clip
•   Secure new cable tie
5.   Program volatile memories
