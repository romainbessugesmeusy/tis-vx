Exhaust Heat Shield, Replace

1.   Raise vehicle
2.   Remove exhaust undertray (2) from rear clamshell and engine undertray (1)
•   Remove 8x bolt, 2x screw, 2x clip



3.   Release intermediate exhaust pipe from catalytic converter
•   Remove 3x bolt, 3x nut (1)



4.   Remove intermediate exhaust pipe from rear silencer
•   Remove U-clamp (1)



5.   Clean all mating surfaces
6.   Remove heat shield from intermediate exhaust pipe
•   Remove 4x bolt, 4x washer, 7x clip
7.   Install heat shield to intermediate exhaust pipe
•   Install 7x clip, 4x washer, 4x bolt
8.   Install intermediate exhaust pipe to rear silencer
•   Install U-clamp
9.   Install intermediate exhaust pipe to catalytic converter
•   Tighten 3x bolt, 3x nuts
10.   Install exhaust undertray to rear clamshell and engine undertray
•   Install 8x bolt, 2x screw 2x clip
11.   Lower vehicle
