Engine and Transmission, Remove and Install (Z20 LET)

  Remove

Important:   A four-post hoist must be used to raise the vehicle when removing the engine undertray and diffuser panel. The vehicle can be moved to a two-post hoist for other operations if required.
1.   Raise vehicle
2.   Remove engine undertray (1)
•   Remove 13x bolt, 13x washer
3.   Remove exhaust pipe undertray (2)
•   Remove 12x bolt, 12x washer, 2x clip



Important:   Observe relevant environmental regulations when disposing of used coolant and lubricants.
4.   Drain coolant system
•   Position collecting container
•   Remove lower hose from radiator feed hose
•   Remove, then discard cable tie (1)
– Release clip (2)
– Remove collecting container when drained



5.   Drain transmission fluid
•   Position collecting container
•   Remove drain plug
•   Install new sealing ring
•   Replace drain plug when drained 35 Nm
6.   Release support bracket for auxiliary cooling pump
•   Remove nut (1)



7.   Lower vehicle
8.   Disconnect battery negative lead
•   Release windscreen washer bottle (1) position to one side
•   Release bolt, nut (2)



9.   Remove engine cover
•   Apply match mark to engine cover 2x hinge
•   Remove 6x bolt, 6x washer, 2x shim
Note:   Shims will vary for cover alignment.
Note:   2x technician required.
10.   Disconnect harness connector (1) from mass air flow meter, temperature sensor

•   Release clip (2)

– Position hose to one side

•   Release clip

•   Pull off

•   Release clip

  Install

84.   Install engine and transmission to vehicle
85.   Install reaction mount bracket, to lower front chassis
•   Recut 2x bolt thread, use recommended locking compound if not microencapsulated
•   Install 2x lower bolt hand tight
Note:   Bracket can only be fitted one way.
86.   Install reaction mount, lower front, to between transmission housing and front chassis
•   Recut 2x bolt thread, use recommended locking compound if not microencapsulated
•   Install 2x bolt hand tight
•   Install new engine mount centre bolt and nut hand tight
87.   Install reaction mount, lower rear, to between transmission and rear chassis
•   Recut 3x bolt thread, use recommended locking compound if not microencapsulated
•   Install 3x bolt, washer hand tight
•   Install new engine mount centre bolt, nut hand tight
88.   Remove Engine and Transmission Support Tool KM-6054 (1) with Adapter KM-904 (2)
89.   Lower vehicle
90.   Install upper engine mount, left
•   Recut 3x bolt thread, use recommended locking compound if not microencapsulated
•   Install 3x bolt, hand tight
Note:   Use trolley jack to align 2x top mount to engine and engine mount bracket, lower front, 2x bolt.
