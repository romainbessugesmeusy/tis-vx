Hose, Intercooler to Throttle Valve Housing, Replace (Z20 LET)

  Remove

1.   Disconnect intercooler hose (1) from connecting pipe for throttle body
•   Release clip (2)
•   Pull off



2.   Raise vehicle to suitable working height
3.   Remove rear road wheel from right side
•   Remove security bolt
– Use special socket located under front bonnet
•   Remove 4x wheel bolt
4.   Remove right rear wheel arch liner
•   Remove 4x screw, 2x clip
5.   Remove intercooler hose (1) from intercooler
•   Release clip (2)
•   Release fuel vapour line from intercooler hose (1)
– Remove, then discard 4x cable tie (3)



  Install

6.   Install new intercooler hose to intercooler
•   Secure clip
•   Install 4x new cable tie
7.   Install liner to wheel arch
•   Install 4x screw, 2x clip
8.   Install rear road wheel to right side
•   Install 4x bolt, security bolt
•   Do not tighten
9.   Lower vehicle
10.   Secure rear road wheel right side
•   Tighten 4x bolt, security bolt 90 Nm
11.   Install intercooler hose to throttle body
•   Secure clip
