Sensor, Coolant Temperature, Replace (Z 20 LET)

  Remove

1.   Disconnect oxygen sensor for primary catalytic converter
•   Release integral clip (1)
– Release cable tie
2.   Remove bracket
•   Remove 4x bolt (2)



Important:   Observe relevant environmental regulations when disposing of used coolant and lubricants.
3.   Partially drain cooling system
•   Disconnect temperature sensor (1)
– Release integral clip
•   Remove expansion tank cap
•   Position collecting container
•   Remove drain plug (3)
4.   Remove temperature sensor (2) from distribution housing
•   Discard sensor, sealing washer
Note:   Clean sealant from drain plug threads.



  Install

5.   Install new temperature sensor 10 Nm
•   Install new sealing washer
6.   Connect temperature sensor
•   Secure integral clip
7.   Install drain plug
•   Use thread sealant
8.   Install bracket
•   Install 4x bolt
9.   Connect oxygen sensor for primary catalytic converter
•   Secure integral clip
– Install cable tie
10.   Fill cooling system
11.   Apply pressure to cooling system 1.0 bar to 1.5 bar
•   Use commercially available tool
12.   Bleed coolant system
•   Open engine bleed valve (1)
•   Open radiator bleed valve (2), left side of radiator



13.   Release pressure from cooling system
14.   Install coolant expansion tank cap
15.   Check coolant level
•   Top-up if necessary
16.   Run engine until operating temperature is reached
17.   Check coolant level
•   Top-up if necessary
