Mounting, Exhaust System, Replace

1.   Raise vehicle
2.   Remove exhaust undertray (2) from rear clamshell and engine undertray (1)
•   Remove 8x bolt, 2x screw, 2x clip



3.   Remove lower heat shield (1) from exhaust manifold
•   Remove 3x bolt and clip



4.   Remove oxygen sensor (1) from exhaust system



5.   Remove oxygen sensor (1) from exhaust system



6.   Remove exhaust down pipe/catalytic converter assembly from exhaust manifold
•   Remove 6x nut (1), 3x bolt, gasket
7.   Release mounting (2) from support bracket



8.   Install new mounting to support bracket
9.   Install exhaust down pipe/catalytic converter assembly to exhaust manifold
•   Install new gasket, 3x bolt, 6x nut
10.   Install 2x oxygen sensor to exhaust system
•   Apply special grease to the oxygen sensor thread
•   Tighten 2x sensor
11.   Install lower heat shield to exhaust manifold
•   Install 3x bolt and clip
12.   Install exhaust undertray to rear clamshell and engine undertray
•   Install 8x bolt, 2x screw 2x clip
13.   Lower vehicle
