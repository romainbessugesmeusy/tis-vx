Front Bonnet Hinge, Adjust

  Adjust

1.   Remove front bonnet – see operation "Front Bonnet, Remove and Install".
2.   Remove front clamshell – see operation "Front Clamshell, Remove and Install".
3.   Release hinge from radiator case
•   Mark alignment markings on hinge and radiator case
•   Slacken 3 nuts (1), 3 screws (2)
•   Adjust hinge
•   Tighten 3 screws (2), 3 nuts (1)



4.   Install front clamshell – see operation "Front Clamshell, Remove and Install".
5.   Install front bonnet – see operation "Front Bonnet, Remove and Install".
