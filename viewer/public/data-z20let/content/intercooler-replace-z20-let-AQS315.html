Intercooler, Replace (Z20 LET)

  Remove

Important:   A four-post hoist must be used to raise the vehicle when removing the engine undertray. The vehicle can be moved to a two-post hoist for other operations if required.
1.   Remove engine undertray
•   Raise vehicle
•   Remove 13x bolt, 13x washer
2.   Remove air intake hose (1) from intercooler
•   Release clip (2)
•   Remove, then discard cable tie (3)



3.   Lower vehicle to suitable working height
4.   Remove rear road wheel from right side
•   Remove security bolt
– Use special socket located under front bonnet
•   Remove 4x wheel bolt
5.   Remove right rear wheel arch liner
•   Remove 4x screw, 2x clip
6.   Remove insulation pad (1) from wheel arch
•   Pull off



7.   Release side pod from wheel arch
•   Remove, then discard 2x nut (1)



8.   Remove side pod from vehicle
•   Remove 2x screw (1)



9.   Remove air deflector (1) from intercooler and body
•   Remove 6x clip



10.   Remove intercooler hose (1) from intercooler (3)
•   Release clip (4)
11.   Disconnect harness connector (2) from charge pressure sensor
•   Release integral clip



12.   Remove intercooler from support bracket and chassis
•   Remove, then discard nut (1)
•   Remove 2x bolt (2)



13.   Remove intercooler from vehicle
Note:   Intercooler shown removed from vehicle for clarity.
•   Release support bracket for purge valve and charcoal canister (1) from chassis
•   Remove 3x bolt (2)
•   Position to one side



14.   Remove charge pressure sensor (1) from intercooler
•   Remove bolt (2)
•   Remove, then discard seal



  Install

15.   Install charge pressure sensor to new intercooler
•   Install bolt
•   Use new seal
16.   Install intercooler to vehicle
•   Install support bracket for purge valve and charcoal canister to chassis
•   Install 3x bolt
17.   Install intercooler to support bracket and chassis
•   Install new nut
•   Install 2x bolt
18.   Connect harness connector to charge pressure sensor
•   Secure integral clip
19.   Install intercooler hose to intercooler
•   Secure clip
20.   Install air deflector to intercooler and body
•   Install 6x clip
21.   Install side pod to vehicle
•   Install 2x screw
22.   Secure side pod to wheel arch
•   Install 2x new nut
23.   Install insulation pad to wheel arch
24.   Install liner to wheel arch
•   Install 4x screw, 2x clip
25.   Install rear road wheel to right side
•   Install 4x bolt, security bolt
•   Do not tighten
26.   Raise vehicle
27.   Intercooler air intake hose to intercooler
•   Secure clip
•   Install new cable tie
28.   Install engine undertray
•   Install 13x bolt, 13x washer
29.   Lower vehicle
30.   Secure rear road wheel right side
•   Tighten 4x bolt, security bolt 90 Nm
