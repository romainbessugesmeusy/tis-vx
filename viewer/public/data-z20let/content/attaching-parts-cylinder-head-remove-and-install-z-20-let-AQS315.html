Attaching Parts, Cylinder Head, Remove and Install (Z 20 LET)

  Remove

1.   Transfer cylinder head to work bench and support safely
•   Use commercially available overhaul stand
2.   Remove coolant distribution housing (1) from cylinder head
•   Remove 3x bolt (2)
•   Remove, then discard gasket
•   Clean mating surfaces



3.   Remove lifting eye (1) from cylinder head
•   Remove bolt (2)



4.   Remove thermostat housing (1) from cylinder head
•   Remove 2x bolt (2)
•   Remove, then discard O ring
•   Clean mating surfaces



5.   Remove camshaft position sensor (1) from cylinder head
•   Remove bolt (2)



6.   Remove coolant bypass housing (1) from cylinder head
•   Remove 2x bolt (2)
•   Remove, then discard O ring
•   Clean mating surfaces



7.   Remove 4x spark plug from cylinder head
8.   Remove 16x hydraulic valve lifter from cylinder head
•   Use KM-845
•   Position hydraulic 16x valve lifter to one side in installation position
•   Clean all removed parts



  Install

9.   Install 16x hydraulic valve lifter to cylinder head
•   Coat 16x hydraulic valve lifter with engine oil
10.   Install 4x spark plug to cylinder head
•   Secure 4x spark plug 25 Nm
11.   Install coolant bypass housing with new O ring to cylinder head
•   Install, secure 2x bolt 15 Nm
12.   Install camshaft position sensor to cylinder head
•   Install, secure bolt 6 Nm
13.   Install thermostat housing with new O ring to cylinder head
•   Install, secure 2x bolt 15 Nm
14.   Install lifting eye to cylinder head
•   Install, secure bolt 20 Nm
15.   Install coolant distribution housing with new gasket to cylinder head
•   Install, then secure 2x M8 bolt 20 Nm
•   Install, then secure M10 bolt 30 Nm
