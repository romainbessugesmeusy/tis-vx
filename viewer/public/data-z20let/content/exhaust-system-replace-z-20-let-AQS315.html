Exhaust System, Replace (Z 20 LET)

  Remove

1.   Disconnect oxygen sensor for secondary catalytic converter
•   Disconnect harness connector (1)
•   Release integral clip



Important:   A four-post hoist must be used to raise the vehicle when removing the engine undertray and exhaust pipe undertray. The vehicle can be moved to a two-post hoist for other operations if required.
2.   Remove engine undertray (1)
•   Raise vehicle
•   Remove 13x bolt, 13x washer
3.   Remove exhaust pipe undertray (2)
•   Remove 12x bolt, 12x washer, 2x clip



4.   Release harness for secondary catalytic converter
•   Release 2x clip from subframe
•   Release clip from rear silencer heatshield
5.   Remove secondary catalytic converter (1)
•   Remove 3x front nut (3), then discard
•   Remove 3x rear nut (2), then discard
– Release rubber mount bracket
•   Release rubber mount from bracket
•   Remove gasket (4) from front joint, then discard
Note:   Illustration shows exhaust removed for clarity.
6.   Remove oxygen sensor from secondary catalytic converter



7.   Remove intermediate pipe and heatshield from rear silencer
•   Release clamp (1)



8.   Remove heatshield from intermediate exhaust pipe
•   Remove 5x clip (1)
•   Remove 3x bolt, 3x washer (2)



9.   Release rear silencer, left support bracket from silencer
•   Remove 2x nut (1), then discard
10.   Remove rear silencer, left flexible support from subframe
•   Remove 2x bolt (2)
•   Remove clamp
11.   Release rear silencer, right support bracket from silencer
•   Remove 2x nut (1), then discard
12.   Remove rear silencer, right flexible support from subframe
•   Remove 2x bolt (2)
•   Remove clamp
Note:   Illustration shows right side only.



13.   Remove silencer from vehicle
•   Remove intermediate pipe clamp from silencer
– Pull off, then discard



  Install

14.   Install intermediate pipe clamp to silencer
•   Push fit, new clamp
15.   Install silencer to vehicle
16.   Install rear silencer, right flexible support to subframe
•   Install clamp
•   Install 2x bolt
17.   Secure rear silencer, right support bracket to silencer
•   Install 2x new nut
18.   Install rear silencer, left flexible support to subframe
•   Install clamp
•   install 2x bolt
19.   Secure rear silencer, left support bracket to silencer
•   Install 2x new nut
20.   Install heatshield to intermediate exhaust pipe
•   Install 3x bolt, 3x washer, 5x clip
21.   Install intermediate pipe and heatshield to rear silencer
•   Loose fit
22.   Install oxygen sensor to secondary catalytic converter
23.   Install secondary catalytic converter for exhaust
•   Install new gasket to front joint
•   Install 3x new front nut
– Position bracket for rubber mount
•   Install 3x new rear nut
•   Secure rubber mount to bracket
24.   Install harness to rear silencer heatshield for oxygen sensor, installed to secondary catalytic converter
•   Secure clip
•   Install harness to subframe
•   Secure 2x clip
25.   Secure clamp for intermediate pipe
26.   Install engine undertray
•   Remove 13x bolt, 13x washer
27.   Install exhaust pipe undertray
•   Install 12x bolt, 12x washer, 2x clip
28.   Lower vehicle
29.   Connect oxygen sensor for secondary catalytic converter
•   Connect harness connector
•   Secure integral clip
