Field Remedy: 2579

Subject: Washer system, pump - Without function
Models: Engines: Option:
All All
Complaint: Washer system without function.
Cause: Washer system fuse blown due to frozen washer fluid.
Production:

Remedy:

In case of customer complaint perform the following checks prior 
washer pump replacement:


 1. Check if the antifreezing concentrate is OK, respectively that 
    the washer fluid/system is not frozen.


    If the concentrate is insufficient/the washer fluid is frozen, 
    inform the customer about the correct mixture and/or refer to 
    the owner manual.


 2. If the washer fluid is frozen, defrost the washer system and 
    fill-up washer system with washer fluid in correct mixture.  


 3. Check the function of the pump and nozzles of the washer 
    system if necessary replace the washer system fuse.

Note: A frozen pump will cause the fuse to blow through again.

FunctionalGroup: N - Electrical Equipment
Complaint Group: 13 - No/Poor Function
