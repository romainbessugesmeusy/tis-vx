Cover, Instrument Housing, Replace

  Remove

1.   Open bonnet
2.   Remove windscreen wash reservoir from mounting bracket
3.   Disconnect battery negative lead
•   Remove nut
4.   Remove airbag assembly from steering wheel
•   Remove 2x bolt (1),
•   Release harness connector



5.   Disconnect harness connector for horn
6.   Remove steering wheel from steering column
•   Remove 4x bolt
7.   Remove upper cover from steering column
•   Remove 6x screw (1)
8.   Remove lower cover from steering column
•   Remove 3x screw (2), clip
•   Release harness connector



9.   Remove instrument housing from instrument panel
•   Remove 2x bolt, 2x washer
•   Release harness connector



10.   Remove cover from instrument housing
•   Remove 4x nut, 4x washer



  Install

11.   Install cover to instrument housing
•   Install 4x new nut, 4x washer
12.   Install instrument housing to instrument panel
•   Install 2x bolt, 2x washer
•   Secure harness connector
13.   Install lower cover to steering column
•   Install 3x screw, clip
•   Secure harness connector
14.   Install upper cover to steering column
•   Install 6x screw
15.   Install steering wheel to steering column
•   Install 4x bolt
16.   Connect harness connector for horn
17.   Install airbag assembly to steering wheel
•   Install 2x bolt
•   Secure harness connector
18.   Connect battery negative lead
•   Secure bolt
19.   Install windscreen wash reservoir to mounting bracket
20.   Close bonnet
21.   Program all volatile memories
