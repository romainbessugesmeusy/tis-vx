Balancer Shaft Unit, Replace (Z20 LET)

  Remove

Important:   A four post hoist must be used to raise the vehicle when removing the engine undertray and exhaust pipe undertray. The vehicle can be moved to a two post hoist for other operations if required.
1.   Remove engine undertray (1)
•   Raise vehicle
•   Remove 13x bolt, 13x washer
2.   Remove exhaust pipe undertray (2)
•   Remove 12x bolt, 12x washer, 2x clip



3.   Lower vehicle
4.   Disconnect battery negative lead
•   Release windscreen washer bottle (1), then position to one side
•   Release bolt, nut (2)



5.   Release intercooler hose (1) from throttle body
•   Release clip (2)



6.   Disconnect oxygen sensor harness from engine harness
•   Release integral clip
•   Remove, then discard cable tie
7.   Release harness from chassis
•   Release 2x clip
8.   Release harness from heatshield
•   Release clip
9.   Remove oil level dipstick
•   Pull off
10.   Release rear road wheel from right side
•   Release security bolt
– Use special socket located under front bonnet
•   Release 4x wheel bolt
11.   Raise vehicle
12.   Drain engine oil
•   Position collecting container
•   Remove drain plug
•   Remove, then discard O ring
13.   Lower vehicle to suitable working height
14.   Remove rear road wheel from right side
•   Remove 4x bolt, security bolt
15.   Remove right rear wheel arch liner
•   Remove 4x screw, 2x clip
16.   Remove intercooler hose from intercooler
•   Release (1)
•   Remove, then discard 4x cable tie



17.   Remove toothed belt upper cover (2)
•   Remove 2x bolt (1)



18.   Remove ribbed V-belt
•   Release ribbed V-belt tension
– Using suitable spanner rotate tensioner (1) clockwise, then Install suitable locking pin (2)



19.   Remove crankshaft pulley (1)
•   Apply match marks
•   Remove 4x bolt
– Use spanner to counter-hold centre nut on crankshaft sprocket



20.   Remove toothed belt lower cover
•   Remove 2x bolt
21.   Rotate crankshaft in normal direction of rotation to Top Dead Centre (TDC) on cylinder no. 1, until timing marks (1) and (2) are aligned



22.   Raise vehicle
23.   Remove secondary catalytic converter (1) for exhaust
•   Remove 3x front nut (3), then discard
•   Remove 3x rear nut (2), then discard
•   Release rubber mount bracket
•   Release rubber mount
•   Remove gasket (4) from front joint, then discard
Note:   Illustration shows exhaust removed for clarity.



24.   Install engine support to engine assembly
25.   Remove heat shield from rear engine mount
•   Remove 2x bolt from rear engine mount
•   Remove bolt from engine cylinder block
Important:   Loose install 2x bolt to rear engine mount before removing engine support from engine assembly.
26.   Remove engine support from engine assembly
27.   Release lower oil pan from upper oil pan
Note:   Support lower oil pan when removing to avoid damage to harness for oil level sensor.
•   Remove 16x bolt (2)
28.   Release oil level sensor from oil pan
•   Remove 2x bolt (1)
•   Remove, then discard oil pan gasket



29.   Disconnect engine harness from oil level sensor
•   Release integral clip
30.   Remove upper oil pan from vehicle
•   Remove 18x bolt
•   Remove, then discard gasket
31.   Remove oil pick up pipe from engine
•   Remove 2x bolt from oil pump (1)
•   Remove 2x bolt from balancer shaft unit (2)
•   Remove, then discard O ring



32.   Remove balancer shaft unit from engine
•   Remove 15x bolt (1)
•   Remove shim



  Install

Important:   Align flat surfaces of both balancer shafts in a downward position.
33.   Install balancer shaft unit to engine
•   Install 15x bolt 20 Nm + 45°
•   Install shim
– Clean mating surfaces



34.   Install oil pick up pipe to engine
•   Install 2x bolt to oil pump 8 Nm
•   Install 2x bolt to balancer shaft unit 8 Nm
•   Install new O ring
35.   Install upper oil pan to engine
•   Install 18x bolt 20 Nm
•   Apply bead of adhesive sealant compound to rear crankshaft bearing cap and oil pump
•   Install new gasket
36.   Connect engine harness to oil level sensor
•   Secure integral clip
37.   Secure oil level sensor to lower oil pan
•   Install 2x bolt 8 Nm
38.   Install lower oil pan to upper oil pan
•   Install new oil pan gasket
•   Install 16x bolt 8 Nm
39.   Install drain bolt 10 Nm
•   Install new O ring
– Lubricate new O ring
40.   Install engine support to engine assembly
41.   Remove 2x bolt from rear engine mount
42.   Install heat shield to rear engine mount
•   Install 2x bolt to rear engine mount
– Recut 2x bolt thread
– Apply locking compound
•   Secure bolt to engine cylinder block
43.   Remove engine support from engine assembly
44.   Secure rear engine mount
•   Secure 2x bolt 46 Nm
45.   Install secondary catalytic converter for exhaust
•   Install 3x new front nut
•   Install 3x new rear nut
•   Install rubber mount bracket to flange bolt
•   Secure rubber mount
•   Install gasket to front joint
46.   Install toothed belt lower cover
•   Install 2x bolt 6 Nm
47.   Install crankshaft pulley
•   Align match marks
•   Install 4x bolt 20 Nm
– Counter hold centre nut
48.   Install ribbed V-belt
•   Release ribbed V-belt tension
– Using suitable spanner rotate tensioner clockwise, then remove suitable locking pin
49.   Lower vehicle to suitable working height
50.   Install toothed belt upper cover (2)
•   Install 2x bolt 6 Nm
51.   Install intercooler hose to intercooler
•   Secure clip
•   Install new 4x cable tie
52.   Install right rear wheel arch liner
•   Install 4x screw, 2x clip
53.   Install rear road wheel to right side
•   Install 4x bolt, security bolt
•   Do not tighten
54.   Lower vehicle
55.   Secure rear road wheel
•   Tighten 4x bolt, security bolt 90 Nm
56.   Install oil level dipstick
•   Push fit
57.   Secure intercooler hose to throttle body
•   Secure clip
58.   Connect oxygen sensor harness from engine harness
•   Secure integral clip
•   Install new cable tie
59.   Secure harness to chassis
•   Secure 2x clip
60.   Secure harness to heatshield
•   Secure clip
61.   Connect battery negative lead
•   Secure nut, bolt
•   Secure windscreen washer bottle
62.   Program all volatile memories
63.   Fill engine to correct oil level
•   Start and run engine until oil light goes out
•   Check oil level, adjust if necessary
64.   Install engine undertray
•   Raise vehicle
•   Install 13x bolt, 13x washer
65.   Install exhaust pipe undertray (2)
•   Install 12x bolt, 12x washer, 2x clip
66.   Lower vehicle
