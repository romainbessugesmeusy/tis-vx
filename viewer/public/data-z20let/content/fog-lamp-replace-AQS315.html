Fog Lamp, Replace

1.   Chock rear road wheels
2.   Remove front road wheel from vehicle
•   Remove 4x bolt, security bolt
– Use special socket located under front bonnet
3.   For right fog lamp, remove fuse box from right wheelarch liner
•   Remove 4x screw
4.   Raise and support vehicle
5.   Remove front wheelarch liner from wheelarch
•   Remove 6x screw





6.   Remove headlamp (1) from front clamshell
•   Do not disturb splash shield (2) (if installed)
•   Remove nut, washer
•   Remove bracket
•   Disconnect harness connector

7.   Release harness connector (1) from fog lamp



8.   Remove dust cover from fog lamp
•   Push fit
9.   Remove support bracket with fog lamp from front clamshell
•   Remove 3x bolt (1), 3x washer



10.   Remove fog lamp from support bracket
•   Remove 2x screw, 2x spacer
11.   Install fog lamp to support bracket
•   Install 2x screw, 2x spacer
12.   Install support bracket with fog lamp to front clamshell
•   Install 3x bolt, 3x washer
13.   Install dust cover to fog lamp
•   Push fit
14.   Secure harness connector to fog lamp
15.   Install headlamp to front clamshell
•   Ensure that splash shield is not disturbed (if installed)
•   Connect harness connector
•   Install bracket
•   Install nut, washer
16.   Adjust headlamp and fog lamp
17.   Install front wheelarch liner to wheelarch
•   Install 5x screw, clip
18.   For right fog lamp, install fuse box to right wheelarch liner
•   Install 4x screw
19.   Install front road wheel to vehicle
•   Install 4x bolt, install security bolt 90 Nm
20.   Raise vehicle and remove support
21.   Remove chocks from rear road wheels
