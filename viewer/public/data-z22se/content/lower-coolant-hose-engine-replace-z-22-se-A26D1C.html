Lower Coolant Hose, Engine, Replace (Z 22 SE)

  Remove

1.   Drain cooling system - see operation “Cooling System, Drain, Fill and Bleed”
Note:   Includes releasing lower coolant hose from oil cooler.
•   Release clip
•   Remove, then discard cable tie
2.   Remove fuel tank - see operation “Fuel Tank, Remove and Install”
3.   Remove lower coolant hose (2) from coolant pipe
•   Release, then remove clip (1)



  Install

4.   Transfer 2x clip to new lower coolant hose
5.   Install new lower coolant hose to coolant pipe
•   Install clip
6.   Install fuel tank - see operation “Fuel Tank, Remove and Install”
7.   Secure lower coolant hose to oil cooler
•   Secure clip
•   Install cable tie
8.   Fill cooling system - see operation “Cooling System, Drain, Fill and Bleed”
