Exhaust System, Replace (Z22 SE)

1.   Raise vehicle
2.   Remove exhaust undertray (2) from rear clamshell and engine undertray (1)
•   Remove 8x bolt, 2x screw, 2x clip



3.   Remove lower heat shield (1) from exhaust manifold
•   Remove 3x bolt and clip



4.   Remove oxygen sensor (1) from exhaust system



5.   Remove oxygen sensor (1) from exhaust system



6.   Remove exhaust down pipe/catalytic converter assembly from exhaust manifold
•   Remove 6x nut (1), 3x bolt, gasket
•   Release rubber mounting (2)



7.   Remove intermediate exhaust pipe from rear silencer
•   Release U-clamp (1)
– Remove 2x nut, 2x washer



8.   Remove right outer support bracket (1) from rear silencer and chassis
•   Remove 2x bolt, 2x nut



9.   Release left support bracket (1) from rear silencer, then remove rear silencer
•   Remove 2x nut
10.   Clean all mating faces



11.   Install rear muffler to left outer support bracket
•   install 2x nut
Note:   To aid assembly, initially install right and left support bracket in a loose condition
12.   Install right outer support bracket to rear silencer and chassis
•   Install 2x bolt, 2x nut
13.   Secure right and left outer support bracket
•   Tighten 2x bolt, 4x nut
14.   Install intermediate exhaust pipe to rear silencer
•   Install U-clamp
– Install 2x washer, 2x nut
15.   Install exhaust down pipe/catalytic converter assembly to exhaust manifold
•   Install gasket, 3x bolt, 6x nut
•   Install mounting rubber
16.   Install 2x oxygen sensor to exhaust system
•   Secure 2x sensor
17.   Install lower heat shield to exhaust manifold
•   Install 3x bolt and clip
18.   Install exhaust undertray
•   Install 8x bolt, 2x screw 2x clip
19.   Lower vehicle
