Transmission Identification  

The F23 manual transmission is identified with a code, which is inscribed on the housing.  

 


 

 

 

 

1  

Housing type  

2  

Transmission code; first number = gear set, second number = final drive ratio  

3  

Production year  

4  

Serial number of relevant year  

5  

GETRAG internal modification index  

6  

Opel part number  

7  

Opel identification code  

8  

Barcode for production control
