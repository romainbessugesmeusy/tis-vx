Transmission Housing Cover, Replace (Transmission Installed)

  Remove

Important:   A four-post hoist must be used to raise the vehicle when removing the engine undertray. The vehicle can be moved to a two-post hoist for other operations if required.
1.   Remove engine undertray
•   Raise vehicle
•   Remove 13x bolt, 13x washer
Important:   Observe relevant environmental requirements when disposing of used oil.
2.   Release transmission housing cover (1) from transmission
•   Position suitable collecting container
•   Remove 8x bolt (arrowed)
3.   Remove transmission housing cover from transmission
•   Use KM-J-37228 (2)
Note:   Take care not to damage mating surfaces.



4.   Remove oil collection pipe (1) from transmission
Note:   Oil collection pipe removed to avoid damage when cleaning mating surfaces.
•   Pull off



  Install

5.   Clean mating surfaces
6.   Install oil collection pipe to transmission
•   Push on
7.   Install new transmission housing cover to transmission
•   Apply sealing compound to mating surface of transmission housing cover
Note:   Do not apply sealing compound to shaded area (1) as this could cause fluid guide pipe (2) to become blocked.
8.   Install 8x bolt 25 Nm



9.   Check transmission oil level and correct as necessary
10.   Install engine undertray
•   Install 13x bolt, 13x washer
11.   Lower vehicle
