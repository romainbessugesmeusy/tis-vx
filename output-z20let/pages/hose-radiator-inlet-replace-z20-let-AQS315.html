Hose, Radiator Inlet, Replace (Z20 LET)

  Remove

1.   Drain coolant system - see operation “Coolant System, Drain, Fill and Bleed”
2.   Disconnect radiator inlet hose from coolant pipe
•   Release, then remove clip (1)



3.   Lower vehicle
4.   Remove coolant hose from radiator inlet
•   Remove clip (1)



  Install

5.   Install coolant hose to radiator inlet
•   Install clip
6.   Raise vehicle
7.   Secure radiator inlet hose to coolant pipe
•   Secure clip
8.   Fill coolant system - see operation “Coolant System, Drain, Fill and Bleed”
