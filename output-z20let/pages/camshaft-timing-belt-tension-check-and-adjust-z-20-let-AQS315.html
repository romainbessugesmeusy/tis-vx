Camshaft Timing Belt Tension, Check and Adjust (Z 20 LET)

  Remove

Important:   A four-post hoist must be used to raise the vehicle when removing the undertray. The vehicle can be moved to a two-post hoist for other operations if required
1.   Raise vehicle
2.   Remove engine undertray (1)
•   Remove 13x bolt, 13x washer



3.   Lower vehicle
4.   Release windscreen wash reservoir
•   Pull off, then move to one side
5.   Disconnect battery negative lead
•   Release nut, bolt
6.  
Release intercooler hose (1) from throttle body (2)
•   Release clip
7.   Raise vehicle to suitable working height
8.   Remove rear road wheel from right side
•   Remove security bolt
– Use special socket located under front bonnet
•   Remove 4x wheel bolt
9.   Remove liner from wheel arch
•   Remove 6x screw
10.   Release intercooler hose (1) from intercooler (2)
•   Release clip
•   Remove, then discard 4x cable tie



11.   Remove upper cover of camshaft timing belt
•   Remove 2x bolt (1)
•   Pull off to release



12.   Remove ribbed V-belt
•   Release ribbed V-belt tension
– Use suitable spanner to rotate tensioner (1) clockwise, then install suitable locking pin (2)



13.   Remove crankshaft pulley
•   Apply match marks
•   Remove 4x bolt
– Use spanner to counter-hold centre nut on crankshaft sprocket
14.   Remove ribbed V-belt tensioner
•   Remove 2x bolt
15.   Remove lower cover of camshaft timing belt
•   Remove 2x bolt (1)



16.   Rotate crankshaft in normal direction of rotation to Top Dead Centre (TDC) on cylinder no. 1, until timing marks (1) and (2) are aligned



17.   Check timing belt tensioner for correct adjustment

Option I:

Pointer (4) for toothed belt tension roller is approximately 4 mm to left of notched mark (3)

Option II:

Pointer (6) for toothed belt tension roller aligns with the "Used" notched mark (5)
Note:   if pointer of tensioner does not align with corresponding notch mark, then carry out adjustment procedure
•   Check position of coolant pump
– Ensure lugs (1) and (2) are aligned



  Adjust

18.   Release tension from timing belt
•   Install Locking Tool KM-853 (1) to camshafts
•   Release bolt, then turn adjusting eccentric in direction of arrow (clockwise) until pointer (2) is located just before left stop



19.   Pre-tension timing belt
•   Turn eccentric of tensioner in direction of arrow (anti-clockwise) until pointer is located just before right stop, then tighten tensioner bolt
•   Remove Locking Tool KM-853 from camshafts
•   Rotate crankshaft two complete turns in normal direction of rotation to Top Dead Centre (TDC) on cylinder no. 1, until timing marks (ll) and (lll) are aligned
•   Install Locking Tool KM-853 to camshafts



20.   Tension timing belt
•   Release tensioner bolt, then turn eccentric of tensioner indirection of arrow (clockwise) until pointer is located in correct position

Option I:

Pointer (1) for toothed belt tension roller aligns with the notched mark (2)

Option II:

Pointer (1) for toothed belt tension roller aligns with the "NEW" notched mark (2)

21.   Tighten tensioner
•   Secure bolt 20 Nm
•   Remove Locking Tool KM-853
22.   Rotate crankshaft two complete turns in normal direction of rotation to (TDC) on cylinder no. 1, until timing marks are aligned. If pointer of tensioner does not align with corresponding notch mark, then repeat adjustment procedure



  Install

23.   Install lower cover of camshaft timing belt
•   Install, then secure 2x bolt (1) 6 Nm
24.   Install ribbed V-belt tensioner
•   Install, then secure 2x bolt 25 Nm
25.   Install crankshaft pulley
•   Align match marks
•   Install, then secure 4x bolt 20 Nm
– Use spanner to counter-hold centre nut on crankshaft sprocket



26.   Install ribbed V-belt
•   Use suitable spanner to rotate tensioner (1) clockwise, then remove locking pin (2)



27.   Install upper cover of camshaft timing belt
•   Push on to secure
•   Install, then secure 2x bolt (1) 6 Nm



28.   Install intercooler hose (1) to intercooler (2)
•   Secure clip
•   Install, then secure 4x new cable tie



29.   Install liner to wheel arch
•   Install, then secure 6x screw
30.   Install rear road wheel to right side
•   Install 4x bolt 90 Nm
•   Install locking bolt 90 Nm
– Use special socket located under front bonnet
31.   Lower vehicle
32.   Install intercooler hose (1) to throttle body (2)
•   Install, then secure clip



33.   Connect battery negative lead
•   Secure nut, bolt
34.   Install windscreen wash reservoir
•   Push fit
35.   Program all volatile memories
36.   Raise vehicle
37.   Install engine undertray (1)
•   Install, then secure 13x bolt, 13x washer
38.   Lower vehicle
