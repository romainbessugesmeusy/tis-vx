Starter Motor, Replace (Z 20 LET)

  Remove

1.   Disconnect battery negative lead
•   Release windscreen washer bottle (1), position to one side
•   Release nut, bolt (2)



2.   Remove upper bolt for starter motor
•   Remove bracket
Important:   A four-post hoist must be used to raise the vehicle when removing the engine undertray and exhaust pipe undertray. The vehicle can be moved to a two-post hoist for other operations if required.
3.   Remove engine undertray (1)
•   Raise vehicle
•   Remove 13x bolt, 13x washer



4.   Release vacuum cut-off valve bracket from engine
•   Disconnect body harness connector
•   Release harness clip (1)
•   Remove 2x bolt (2)
•   Position vacuum tank to one side



5.   Remove starter motor
•   Remove lower bolt
•   Disconnect harness earth from engine block (3)
– Remove bolt
•   Disconnect harness connector for starter motor (2)
– Remove nut, washer
•   Disconnect positive battery cable from starter motor (1)
– Remove nut, washer



  Install

Important:   Position starter motor before tightening bolt.
6.   Install starter motor
•   Install lower bolt 40 Nm
7.   Install positive battery cable to starter motor
•   Install washer, nut
•   Install harness connector to starter motor
– Install washer, nut
8.   Connect harness earth lead to engine block
•   Install bolt
9.   Install vacuum cut-off valve bracket to engine
•   Install 2x bolt 25 Nm
•   Install harness clip
•   Connect body harness connector
10.   Install engine undertray
•   Install 13x bolt, 13x washer
11.   Lower vehicle
12.   Install upper bolt for starter motor 45 Nm
•   Install bracket
13.   Connect battery negative lead
•   Secure bolt, nut
•   Secure windscreen washer bottle
14.   Program all volatile memories
