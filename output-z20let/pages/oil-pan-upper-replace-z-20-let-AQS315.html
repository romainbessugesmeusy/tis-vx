Oil Pan, Upper, Replace (Z 20 LET)

  Remove

Important:   A four post hoist must be used to raise the vehicle when removing the engine undertray and exhaust pipe undertray. The vehicle can be moved to a two post hoist for other operations if required.
1.   Remove engine undertray (1)
•   Raise vehicle
•   Remove 13x bolt, 13x washer
2.   Remove exhaust pipe undertray (2)
•   Remove 12x bolt, 12x washer, 2x clip



3.   Lower vehicle
4.   Remove oil level dipstick
•   Pull off
5.   Disconnect oxygen sensor harness from engine harness
•   Release integral clip
•   Remove, then discard cable tie
6.   Release harness from subframe
•   Release 2x clip
7.   Release harness from heatshield
•   Release clip
8.   Raise vehicle
9.   Remove drain bolt (1)
•   Position collecting container
•   Remove, then discard O ring
10.   Remove lower oil pan from upper oil pan
Note:   Support lower oil pan when removing to avoid damage to harness for oil level sensor.
•   Remove 16x bolt (2)
•   Release oil level sensor from lower oil pan
– Remove 2x bolt
•   Remove, then discard oil pan gasket



11.   Remove secondary catalytic converter (1) for exhaust
•   Remove 3x front nut (3), then discard
•   Remove 3x rear nut (2), then discard
•   Release rubber mount bracket from flange bolt
•   Release rubber mount
•   Remove gasket (4) from front joint, then discard
Note:   Illustration shows exhaust removed for clarity.



12.   Install engine support to engine assembly
13.   Remove heat shield from rear engine mount
•   Remove 2x bolt from rear engine mount
•   Remove bolt from cylinder block
14.   Loose install 2x bolt to rear engine mount
15.   Remove engine support from engine assembly
16.   Disconnect engine harness from oil level sensor
•   Release integral clip
17.   Remove upper oil pan from vehicle
•   Remove 14x bolt from cylinder block
•   Remove 4x bolt from transmission
•   Remove, then discard gasket

  Install

18.   Install upper oil pan to vehicle
•   Clean all mating surfaces
•   Apply a bead of adhesive compound to casing joints
•   Install new gasket
•   Install 14x bolt 20 Nm to cylinder block
•   Install 4x bolt 40 Nm to transmission
19.   Connect engine harness to oil level sensor
•   Secure integral clip
20.   Install engine support to engine assembly
21.   Remove 2x bolt from rear engine mount
22.   Install heat shield to rear engine mount
•   Install 2x bolt to engine mount 49 Nm
– Recut 2x bolt thread
– Use recommended locking compound
•   Secure bolt to engine block
23.   Remove engine support from engine assembly
24.   Install secondary catalytic converter for exhaust
•   Install 3x new front nut
•   Install 3x new rear nut
•   Install rubber mount bracket to flange bolt
•   Secure rubber mount
•   Install gasket to front joint
25.   Install lower oil pan to upper oil pan
Note:   Support lower oil pan when installing to avoid damage to harness for oil level sensor.
•   Secure oil level sensor to lower oil pan
– Install 2x bolt 8 Nm
•   Install new oil pan gasket
•   Install 16x bolt 8 Nm
26.   Install drain bolt 10 Nm
•   Install new O ring
– Lubricate new O ring
27.   Lower vehicle
28.   Connect oxygen sensor harness to engine harness
•   Secure integral clip
•   Install new cable tie
29.   Secure harness to subframe
•   Secure 2x clip
30.   Secure harness to heatshield
•   Secure clip
31.   Install oil level dipstick
•   Push fit
32.   Fill engine to correct oil level
•   Start and run engine until oil light goes out
•   Check oil level, adjust if necessary
33.   Raise vehicle
34.   Install engine undertray
•   Install 13x bolt, 13x washer
35.   Install exhaust pipe undertray (2)
•   Install 12x bolt, 12x washer, 2x clip
36.   Lower vehicle
