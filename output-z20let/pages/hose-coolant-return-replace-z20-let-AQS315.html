Hose, Coolant Return, Replace (Z20 LET)

  Remove

1.   Drain cooling system - see operation “Cooling System, Drain, Fill and Bleed”
2.   Remove clip from coolant return hose
3.   Disconnect engine harness from auxiliary coolant pump
•   Release integral connector
4.   Release coolant return hose from heater hose
•   Release, then discard cable tie
5.   Disconnect turbo-charger coolant hose from coolant return hose
•   Release clip
6.   Lower vehicle
7.   Remove, then discard coolant return hose from oil cooler housing
•   Remove clip (1)



  Install

8.   Connect coolant return hose to oil cooler housing
•   Secure clip
9.   Raise vehicle
10.   Connect turbo-charger coolant hose to coolant return hose
•   Secure clip
11.   Secure coolant return hose to heater hose
•   Install new cable tie
12.   Connect engine harness to auxiliary coolant pump
•   Secure integral connector
13.   Fill coolant system - see operation “Cooling System, Drain, Fill and Bleed”
