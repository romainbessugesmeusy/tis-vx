Exhaust Manifold with Turbocharger, Replace (Z 20 LET)

  Remove

1.   Remove engine cover - see operation “Engine Cover, Remove and Install” in group A
2.   Disconnect air intake hose (1) from mass air flow meter (2) and turbocharger (3)
•   Release 2x clip
•   Move air intake hose to one side



3.   Remove support bracket with engine lifting eye (1) from cylinder head
•   Remove bolt



4.   Remove heat shield (1) from support bracket (2) on cylinder head
•   Remove 4x bolt, 4x washer



5.   Remove support bracket (1) for heat shield from cylinder head
•   Remove bolt, 2x washer
•   Remove engine lifting eye (2)
– Remove bolt



6.   Disconnect harness connector (2) from switchover valve (1) on turbocharger
•   Release integral clip
7.   Disconnect vacuum hose (3) from turbocharger
•   Pull off
8.   Remove guide tube (4) for oil level dipstick from engine cylinder block
•   Remove oil level dipstick
•   Remove bolt
•   Remove 2x O ring, then discard



9.   Disconnect harness connector for primary oxygen sensor (1) from engine harness
•   Release integral clip
•   Release cable tie, then discard



10.   Disconnect harness connector (1) for secondary oxygen sensor from engine harness
•   Release integral clip
•   Lower harness through aperture in left rear inner wheel arch



11.   Release heat shield for turbocharger from transmission housing
•   Remove bolt (1)



Important:   A four-post hoist must be used to raise the vehicle when removing the engine undertray. The vehicle can be moved to a two-post hoist for other operations if required.
12.   Remove engine undertray (1)
•   Raise vehicle
•   Remove 13x bolt, 13x washer
13.   Remove exhaust undertray (2)
•   Remove 12x bolt, 12x washer, 2x clip



14.   Remove secondary catalytic converter (1)
•   Remove 6x nut, then discard
•   Remove gasket, then discard
•   Release harness for oxygen sensor from chassis
– Release 2x cable tie, then discard
– Release harness from clip



15.   Remove hose (3) from turbocharger (2) and intercooler (1)
•   Release 2x clip
•   Remove cable tie, then discard



16.   Release heat shield for turbocharger from engine cylinder block
•   Remove bolt, washer (1)



Important:   Observe relevant environmental requirements when disposing of used oil.
17.   Disconnect oil return pipe from engine cylinder block
•   Position suitable collecting container
•   Disconnect pipe union (1), 2x seal ring
– Discard 2x seal ring
– Install blanking plug



18.   Disconnect oil feed pipe from engine cylinder block
•   Remove bolt (1)
•   Disconnect pipe union (2)



19.   Lower vehicle
Important:   Observe relevant environmental requirements when disposing of used coolant (anti-freeze).
20.   Disconnect coolant feed hose (1) from turbocharger, then secure to one side
•   Position suitable collecting container
•   Disconnect pipe union, 2x seal ring
– Discard 2x seal ring



21.   Disconnect coolant return hose from turbocharger
•   Release clip (1)



22.   Remove exhaust manifold with turbocharger from cylinder head
•   Remove 10x nut, then discard
•   Remove gasket, then discard
23.   Remove washer (2) from exhaust manifold
•   Slide off
24.   Remove heat shield (5) from exhaust manifold
•   Remove bolt
25.   Remove catalytic converter (6) from exhaust manifold
•   Remove clamp (7)
Note:   Note position of catalytic converter when removing from exhaust manifold.
26.   Remove oil feed pipe (3) from turbocharger
•   Disconnect pipe union, 2x seal ring
– Discard 2x seal ring
27.   Remove oil return pipe (4) from turbocharger
•   Remove 2x bolt, gasket
– Discard gasket
28.   Remove coolant return pipe (1) from turbocharger
•   Disconnect pipe union, 2x seal ring, bolt
– Discard 2x seal ring



  Install

29.   Install coolant return pipe to turbocharger
•   Connect pipe union, 2x new seal ring, bolt
30.   Install oil return pipe to turbocharger
•   Install 2x bolt 8 Nm , new gasket
31.   Install oil feed pipe to turbocharger
•   Connect pipe union 35 Nm , 2x new seal ring
32.   Install catalytic converter to exhaust manifold
•   Install clamp 8 Nm
33.   Install heat shield to exhaust manifold
•   Install bolt
34.   Install washer to exhaust manifold
•   Slide on
35.   Install exhaust manifold with turbocharger to cylinder head
•   Install new gasket
•   Install 10x new nut
Note:   Tighten 10x nut 8 Nm in sequence shown. Wait 5 minutes, then re-tighten 10x nut 8 Nm in sequence shown.



36.   Install guide tube for oil level dipstick to engine cylinder block
•   Install 2x new O ring
•   Install bolt
•   Install oil level dipstick
37.   Connect vacuum hose to turbocharger
•   Push fit
38.   Connect harness connector to switchover valve on turbocharger
•   Secure integral clip
39.   Connect coolant feed hose to turbocharger
•   Connect pipe union, 2x new seal ring
40.   Install support bracket for heat shield to cylinder head
•   Install bolt, 2x washer
•   Install engine lifting eye
– Install bolt
41.   Install heat shield to support bracket on cylinder head
•   Install 4x bolt, 4x washer
42.   Secure heat shield for turbocharger to transmission housing
•   Install bolt
43.   Install support bracket with engine lifting eye to cylinder head
•   Install bolt
– Apply locking compound
44.   Connect coolant return hose to turbocharger
•   Secure clip
45.   Connect harness connector for primary oxygen sensor to engine harness
•   Secure integral clip
•   Secure new cable tie
46.   Connect air intake hose to mass air flow meter and turbocharger
•   Secure 2x clip
47.   Raise vehicle
48.   Connect oil feed pipe to engine cylinder block
•   Install bolt 35 Nm
•   Connect pipe union 20 Nm
49.   Connect oil return pipe to engine cylinder block
•   Connect pipe union 30 Nm , 2x new seal ring
– Remove blanking plug, then discard
50.   Secure heat shield for turbocharger to engine cylinder block
•   Install bolt, washer
51.   Install hose to turbocharger and intercooler
•   Secure 2x clip
•   Install new cable tie
52.   Install secondary catalytic converter
•   Install 6x new nut
•   Install new gasket
•   Secure harness for oxygen sensor to chassis
– Secure harness to clip
– Secure 2x new cable tie
•   Raise harness through aperture in left rear inner wheel arch
53.   Install engine undertray
•   Install 13x bolt, 13x washer
54.   Install exhaust undertray
•   Install 12x bolt, 12x washer, 2x clip
55.   Lower vehicle
56.   Connect harness connector for secondary oxygen sensor to engine harness
•   Secure integral clip
57.   Install engine cover - see operation “Engine Cover, Remove and Install” in group A
58.   Check coolant level, then top-up as required
59.   Check oil level, then top-up as required
