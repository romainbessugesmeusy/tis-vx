Oil Pump, Replace (Z20 LET)

  Remove

Important:   A four-post hoist must be used to raise the vehicle when removing the engine undertray. The vehicle can be moved to a two-post hoist for other operations if required.
1.   Remove engine undertray
•   Raise vehicle
•   Remove 13x bolt, 13x washer
2.   Remove exhaust undertray
•   Remove 12x bolt, 2x clip
3.   Lower vehicle
4.   Disconnect battery negative lead
•   Release windscreen wash reservoir, then move to one side
•   Release nut, bolt
5.   Disconnect intercooler hose (1) from connecting pipe for throttle body
•   Release clip (2)
•   Pull off



6.   Disconnect oxygen sensor harness from engine harness
•   Release integral clip
•   Remove, then discard cable tie
7.   Remove oil level dipstick
•   Pull off
8.   Raise vehicle
Important:   Observe relevant environmental requirements when disposing of used engine oil.
9.   Drain engine oil
•   Remove drain bolt from lower oil pan
•   Position collecting container
•   Remove, then discard O ring
10.   Remove oil filter, then discard
•   Use Oil Filter Wrench KM-726-A
11.   Disconnect engine harness from oil pressure switch
•   Release integral clip
12.   Release 2x oil line from thermostat housing for oil cooler
•   Remove banjo bolt from feed line (1)
•   Remove banjo bolt from return line (2)
•   Remove 4x sealing ring, then discard
•   Release clip



13.   Lower vehicle to suitable working height
14.   Remove rear road wheel from right side
•   Remove security bolt
– Use special socket located under front bonnet
•   Remove 4x wheel bolt
15.   Remove wheel arch liner, right rear
•   Remove 4x screw, 2x clip
16.   Remove intercooler hose (1) from intercooler
•   Release clip (2)
•   Release fuel vapour line from intercooler hose
– Remove, then discard 4x cable tie (3)



17.   Remove upper front cover (2) for timing belt
•   Remove 2x bolt (1)
•   Pull off



18.   Remove ribbed V-belt
•   Release ribbed V-belt tension
– Using suitable spanner rotate tensioner (1) clockwise, then Install suitable locking pin (2)



19.   Remove crankshaft pulley
•   Apply match marks to crankshaft pulley and crankshaft
•   Remove 4x bolt
– Counter-hold centre bolt on crankshaft sprocket
20.   Remove ribbed V-belt tensioner
•   Remove 2x bolt
21.   Remove lower cover for timing belt
•   Remove 2x bolt (1)



22.   Rotate crankshaft in normal direction of rotation to Top Dead Centre (TDC) on cylinder no. 1, until timing marks (2) and (1) are aligned
23.   Lock camshafts in position
•   Use Locking Tool KM-853



24.   Raise vehicle to suitable working height
25.   Release harness for oxygen sensor from subframe
•   Release 2x clip
26.   Release harness for oxygen sensor from heatshield
•   Release clip
27.   Remove secondary catalytic converter (1) for exhaust
•   Remove 3x front nut (3), then discard
•   Remove 3x rear nut (2), then discard
•   Release rubber mount bracket from flange bolt
•   Release rubber mount
•   Remove gasket (4) from front joint, then discard
Note:   Illustration shows exhaust removed for clarity.



28.   Install engine support to engine assembly
29.   Remove heatshield from rear engine mount
•   Remove 2x bolt from rear engine mount
•   Remove bolt from engine cylinder block
30.   Loose install 2x bolt to rear engine mount
Note:   To ensure engine safely secured
31.   Remove engine support from engine assembly
32.   Remove turbocharger oil feed pipe from engine cylinder block and turbocharger
•   Release 2x union
•   Remove bolt
33.   Install drain bolt (1) 10 Nm
•   Install new O ring
– Lubricate new O ring
34.   Remove lower oil pan from upper oil pan
Note:   Support lower oil pan when removing to avoid damage to harness for oil level sensor.
•   Remove 16x bolt (2)
•   Release oil level sensor from lower oil pan
– Remove 2x bolt
•   Remove, then discard oil pan gasket



35.   Disconnect engine harness from oil level sensor
•   Release integral clip
36.   Remove upper oil pan from engine cylinder block
•   Remove 14x bolt from cylinder block
•   Remove 4x bolt from transmission
•   Remove, then discard gasket
37.   Remove oil pick-up pipe from oil pump and balancer shafts
•   Remove 2x bolt from oil pump (1)
•   Remove 2x bolt from balancer shafts (2)
•   Remove, then discard O ring



38.   Lower vehicle to suitable working height
39.   Release tension from timing belt
•   Release bolt, then turn adjusting eccentric clockwise in direction until pointer is located just before left stop
40.   Remove timing belt from pulleys
•   Position timing belt to one side
41.   Remove crankshaft sprocket (1) from crankshaft
•   Remove bolt
– Counter-hold using KM-662-C



42.   Remove woodruff key and spacer from crankshaft
43.   Remove tensioner for timing belt (1)
•   Remove bolt



44.   Remove 2x guide roller for timing belt (1)
•   Remove 2x bolt



45.   Lower vehicle
46.   Disconnect harness connector (1) from ignition module
47.   Remove ignition module cover (2) from cylinder head
•   Slide off



48.   Remove ignition module from cylinder head
•   Remove 2x bolt (1)



49.   Disconnect harness connector from camshaft sensor (1)
50.   Release wiring harness from cylinder head cover
•   Release from 3x clip
51.   Disconnect 3x breather hose (2) from cylinder head cover (3)
52.   Remove cylinder head cover
•   Remove 10x bolt
•   Remove, then discard 10x 'O' ring
•   Remove, then discard gasket



53.   Remove 2x camshaft pulley (2) and (3) from camshaft
Note:   Note position of locating pin on pulley for exhaust camshaft.
•   Remove 2x bolt (1), 2x washer
– Counter-hold camshaft at hexagon with open end of spanner (4)
– Second technician required
– Discard 2x bolt



54.   Raise vehicle to suitable working height
55.   Install and set Engine Support Tool KM-6054 (1), use with Adapter KM-904 (2)
•   Support engine



56.   Remove engine mounting bracket from support bracket on engine cylinder block and damping block
•   Remove 3x bolt
•   Remove nut
57.   Remove timing belt from vehicle
58.   Remove support bracket from engine cylinder block
•   Remove 2x bolt
59.   Remove rear cover of camshaft timing belt (1) from engine cylinder block
•   Remove 3x screw, 2x bolt



60.   Remove thermostat housing for oil cooler (1) from oil pump
•   Remove threaded adaptor (2)
– Use KM-943
•   Remove bolt
– Use suitable tool (3)
•   Remove, then discard seal



61.   Remove oil pump (1) from engine cylinder block
•   Remove 6x bolt
– Remove, then discard gasket



  Install

62.   Install new oil pump to engine cylinder block
•   Install 6x bolt 8 Nm
•   Install new oil pump gasket
– Clean all mating surfaces
63.   Install new oil seal to oil pump
•   Lubricate sealing lip of oil seal
•   Use KM-693-A (1) with bolt and washer of crankshaft sprocket



64.   Install thermostat housing for oil cooler to oil pump
•   Use new seal
– Lubricate new seal
•   Install bolt 23 Nm
•   Install threaded adaptor 23 Nm
– Use KM-943
65.   Install rear cover of camshaft timing belt to engine cylinder block
•   Install 3x screw, 2x bolt 8 Nm
66.   Install support bracket to engine cylinder block
•   Install 2x bolt 45 Nm
67.   Install timing belt to vehicle
Note:   Note rotational direction of timing belt.
68.   Install engine mounting bracket to support bracket on engine cylinder block and damping block
•   Install 3x bolt 50 Nm
•   Install nut 50 Nm
69.   Raise vehicle to suitable working height
70.   Remove Engine Support Tool KM-6054 and Adapter KM-904 from engine
71.   Lower vehicle to suitable working height
72.   Install 2x camshaft pulley to 2x camshaft
Note:   Note position of locating pin on pulley for exhaust camshaft.
•   Install 2x new bolt, 2x washer 50 Nm + 60° + 15°
– Counter-hold camshaft at hexagon with open end of spanner
– Second technician required
73.   Install cylinder head cover to cylinder head
•   Install 10 x new 'O' ring, new gasket
•   Install 10x bolt 8 Nm
74.   Connect 3x breather hose to cylinder head cover
75.   Secure wiring harness to cylinder head cover
•   Secure to 3x clip
76.   Connect harness connector to camshaft sensor
77.   Install ignition module to cylinder head
•   Install 2x bolt 8 Nm
78.   Install ignition module cover to cylinder head
•   Slide on
79.   Connect harness connector to ignition module
80.   Raise vehicle to suitable working height
81.   Install 2x guide roller for timing belt
•   Install 2x new bolt 25 Nm
82.   Install tensioner for timing belt
•   Install bolt
Note:   Do not secure bolt.
83.   Install woodruff key and spacer to crankshaft
84.   Install crankshaft sprocket to crankshaft
•   Install new bolt 130 Nm + 45°
– Counter-hold using KM-662-C
85.   Check timing marks are aligned
86.   Lock camshafts in position
•   Use Locking Tool KM-853
87.   Install timing belt to pulleys
88.   Pre-tension timing belt
•   Turn eccentric of tensioner in direction of arrow (anti-clockwise) until pointer is located just before right stop, then secure tensioner bolt
•   Remove Locking Tool KM-853 from camshafts
•   Rotate crankshaft two complete turns in normal direction of rotation to Top Dead Centre (TDC) on cylinder no. 1, until timing marks (ll) and (lll) are aligned
•   Install Locking Tool KM-853 to camshafts



89.   Tension timing belt
•   Release tensioner bolt, then turn eccentric of tensioner in direction of arrow (clockwise) until pointer is located in correct position
Note:   For new belt, pointer (1) for timing belt tensioner aligns with notched mark (3).
Note:   For used belt, pointer (1) for timing belt tensioner aligns with notched mark (2).
90.   Secure tensioner
•   Secure bolt 20 Nm
•   Remove Locking Tool KM-853
91.   Rotate crankshaft two complete turns in normal direction of rotation to (TDC) on cylinder no. 1, until timing marks are aligned. If pointer of tensioner does not align with corresponding notch mark, then repeat adjustment procedure



92.   Install lower cover of camshaft timing belt
•   Install 2x bolt 6 Nm
93.   Install ribbed V-belt tensioner
•   Install 2x bolt 25 Nm
94.   Install crankshaft pulley
•   Align match marks
•   Install 4x bolt 20 Nm
– Counter-hold centre nut on crankshaft sprocket
95.   Install ribbed V-belt
•   Use suitable spanner to rotate tensioner clockwise, then remove locking pin
96.   Install upper cover of camshaft timing belt
•   Push fit
•   Install 2x bolt 6 Nm
97.   Install intercooler hose to intercooler
•   Secure clip
•   Install 4x new cable tie
98.   Raise vehicle to suitable working height
99.   Install oil pick-up pipe to oil pump and balancer shafts
•   Clean mating surfaces
•   Install new O ring
– Lubricate new O ring
•   Install 2x bolt 8 Nm to oil pump
•   Install 2x bolt 8 Nm to balancer shafts
100.   Install upper oil pan to engine cylinder block
•   Clean mating surfaces
•   Apply a bead of adhesive compound to casing joints
•   Install new gasket
•   Install 14x bolt 20 Nm to engine cylinder block
•   Install 4x bolt 40 Nm to transmission
101.   Connect engine harness to oil level sensor
•   Secure integral clip
102.   Install lower oil pan to upper oil pan
Note:   Support lower oil pan when installing to avoid damage to harness for oil level sensor.
•   Secure oil level sensor to lower oil pan
– Install 2x bolt 8 Nm
•   Install new oil pan gasket
•   Install 16x bolt 8 Nm
103.   Install turbocharger oil feed pipe to engine cylinder block and turbocharger
•   Secure 2x union
•   Install bolt
104.   Connect engine harness to oil pressure switch
•   Secure integral clip
105.   Secure 2x oil line to thermostat housing for oil cooler
•   Install banjo bolt, feed pipe to thermostat housing for oil cooler 30 Nm
•   Install banjo bolt, return pipe to thermostat housing for oil cooler 30 Nm
•   Install 4x new sealing ring
•   Secure clip
106.   Install new oil filter
•   Use Oil Filter Wrench KM-726-A
•   Tighten 15 Nm
– Lubricate O ring seal
107.   Install engine support to engine assembly
108.   Remove 2x bolt from rear engine mount
109.   Install heatshield to rear engine mount and engine cylinder block
•   Install 2x bolt to rear engine mount 49 Nm
•   Install bolt to engine cylinder block
110.   Remove engine support from engine assembly
111.   Install secondary catalytic converter for exhaust
•   Install 3x new front nut
•   Install 3x new rear nut
•   Secure rubber mount bracket to flange bolt
•   Secure rubber mount
•   Install new gasket to front joint
112.   Secure harness for oxygen sensor to subframe
•   Secure 2x clip
113.   Secure harness for oxygen sensor to heatshield
•   Secure clip
114.   Lower vehicle to suitable working height
115.   Install liner to wheel arch
•   Install 4x screw, 2x clip
116.   Install rear road wheel to right side
•   Install 4x bolt, security bolt
•   Do not tighten
117.   Lower vehicle
118.   Secure rear road wheel
•   Tighten 4x bolt, security bolt 90 Nm
119.   Install oil level dipstick
•   Push fit
120.   Install intercooler hose to throttle body
•   Secure clip
121.   Connect oxygen sensor harness to engine harness
•   Secure integral clip
•   Install cable tie
122.   Reconnect battery negative lead
•   Secure nut and bolt
123.   Install windscreen wash reservoir
•   Push fit
124.   Fill engine to correct oil level
•   Start and run engine until oil light goes out
•   Check and adjust oil level
125.   Program all volatile memories
126.   Raise vehicle
127.   Install engine undertray
•   Install 13x bolt, 13x washer
128.   Install exhaust undertray
•   Install 12x bolt, 2x clip
129.   Lower vehicle
