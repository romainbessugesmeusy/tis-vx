Loudspeakers, Replace

1.   Slide driver's seat forward
2.   Remove driver's side loudspeaker cover
•   Pull off
3.   Remove driver's side loudspeaker from trim panel
•   Remove 3x screw (1), release 2x harness connector



4.   Install driver's side loudspeaker to trim panel
•   Secure 2x harness connector, install 3x screw
5.   Install driver's side loudspeaker cover
•   Push fit
6.   Remove passenger's side loudspeaker cover
•   Pull off
Note:   Passenger's seat does not slide forward
7.   Remove passenger's side loudspeaker from trim panel
•   Remove 3x screw, release 2x harness connector (1)



8.   Install passenger's side loudspeaker to trim panel
•   Secure 2x harness connector, install 3x screw
9.   Install passenger's side loudspeaker cover
•   Push fit
