Coolant Hose, Return Flow, Oil Cooler, Replace (Z20 LET)

  Remove

1.   Drain cooling system - see operation “Cooling System, Drain, Fill and Bleed”
Note:   Includes releasing coolant hose, return flow from coolant return hose.
•   Release clip (1)
•   Remove, then discard cable tie (2)



2.   Remove fuel tank - see operation “Fuel Tank, Remove and Install”
3.   Remove coolant hose (2) from coolant pipe
•   Release, then remove clip (1)



  Install

4.   Transfer 2x clip and inner sleeve to new coolant hose, return flow
5.   Install coolant hose to coolant pipe
•   Install clip
6.   Install fuel tank - see operation “Fuel Tank, Remove and Install”
7.   Fill cooling system - see operation “Cooling System, Drain, Fill and Bleed”
