Oil Return Pipe, Turbocharger, Replace (Z20 LET)

  Remove

Important:   A four-post hoist must be used to raise the vehicle when removing the engine undertray. The vehicle can be moved to a two-post hoist for other operations if required.
1.   Remove engine undertray
•   Raise vehicle
•   Remove 13x bolt, 13x washer
2.   Remove hose (3) from turbocharger (2) and intercooler (1)
•   Release 2x clip
•   Remove, then discard cable tie



3.   Release heat shield for turbocharger from engine cylinder block
•   Remove bolt, washer (1)



Important:   Observe relevant environmental requirements when disposing of used oil.
4.   Disconnect oil return pipe for turbocharger from engine cylinder block
•   Position suitable collecting container
•   Remove pipe union (1), 2x seal ring
– Discard 2x seal ring



5.   Remove oil return pipe (2) from turbocharger
•   Remove 2x bolt (1), gasket
– Discard gasket



  Install

6.   Install new oil return pipe to turbocharger
•   Install 2x bolt 8 Nm , gasket
– Use new gasket
7.   Connect oil return pipe for turbocharger to engine cylinder block
•   Install pipe union 30 Nm , 2x seal ring
– Use 2x new seal ring
8.   Secure heat shield for turbocharger to engine cylinder block
•   Install bolt, washer
9.   Install hose to turbocharger and intercooler
•   Install new cable tie
•   Secure 2x clip
10.   Install engine undertray
•   Install 13x bolt, 13x washer
11.   Lower vehicle
12.   Check engine oil level and correct as necessary
