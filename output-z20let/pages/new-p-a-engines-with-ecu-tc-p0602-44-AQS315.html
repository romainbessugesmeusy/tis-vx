Field Remedy: 1678

Subject: New P&A engines with ECU - TC P0602 (44)
Models: Engines: Option:
All All None
Complaint: SVS on and TC P0602 (44) stored after programming of ECU
Cause: Engines supplied from P&A are being completed with a production ECM, which is not usable for P&A purposes, because they are built up for the assembly plants.
Production: Not affected

Remedy:

Please use the old ECM in case of an engine exchange. The new ECM coming with
the new engine has to be assembled to the old engine, which will be returned
to the warranty return center or the engine core return process.
FunctionalGroup: J - Engine
Complaint Group: 04 - Electrical failures
Trouble Code: TC P0602 (44)
