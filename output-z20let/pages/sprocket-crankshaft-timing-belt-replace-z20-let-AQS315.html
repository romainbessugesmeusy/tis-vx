Sprocket, Crankshaft, Timing Belt, Replace (Z20 LET)

  Remove

Important:   A four-post hoist must be used to raise the vehicle when removing the engine undertray. The vehicle can be moved to a two-post hoist for other operations if required.
1.   Remove engine undertray
•   Raise vehicle
•   Remove 13x bolt, 13x washer
2.   Lower vehicle
3.   Disconnect battery negative lead
•   Release windscreen wash reservoir, then move to one side
•   Release nut, bolt
4.   Disconnect intercooler hose (1) from connecting pipe for throttle body
•   Release clip (2)
•   Pull off



5.   Raise vehicle to suitable working height
6.   Remove rear road wheel from right side
•   Remove security bolt
– Use special socket located under front bonnet
•   Remove 4x wheel bolt
7.   Remove wheel arch liner, right rear
•   Remove 4x screw, 2x clip
8.   Remove intercooler hose (1) from intercooler
•   Release clip (2)
•   Release fuel vapour line from intercooler hose (1)
– Remove, then discard 4x cable tie (3)



9.   Remove upper front cover (2) for timing belt
•   Remove 2x bolt (1)
•   Pull off



10.   Remove ribbed V-belt
•   Release ribbed V-belt tension
– Using suitable spanner rotate tensioner (1) clockwise, then Install suitable locking pin (2)



11.   Remove crankshaft pulley
Note:   Note position of crankshaft pulley.
•   Remove 4x bolt
– Counter-hold centre bolt on crankshaft sprocket
12.   Remove ribbed V-belt tensioner
•   Remove 2x bolt
13.   Remove lower cover for timing belt
•   Remove 2x bolt (1)



14.   Rotate crankshaft in normal direction of rotation to Top Dead Centre (TDC) on cylinder no. 1, until timing marks (2) and (1) are aligned
15.   Lock camshafts in position
•   Use Locking Tool KM-853
16.   Release tension from timing belt
•   Release bolt, then turn adjusting eccentric clockwise in direction until pointer is located just before left stop
17.   Remove timing belt from pulleys
•   Position timing belt to one side



18.   Remove crankshaft sprocket (1) from crankshaft
•   Remove, then discard bolt
– Counter-hold using KM-662-C



  Install

19.   Install new crankshaft sprocket to crankshaft
•   Install new bolt 130 Nm + 45°
– Counter-hold using KM-662-C
20.   Check timing marks are aligned
21.   Install timing belt to pulleys
22.   Pre-tension timing belt
•   Turn eccentric of tensioner in direction of arrow (anti-clockwise) until pointer is located just before right stop, then secure tensioner bolt
•   Remove Locking Tool KM-853 from camshafts
•   Rotate crankshaft two complete turns in normal direction of rotation to Top Dead Centre (TDC) on cylinder no. 1, until timing marks (ll) and (lll) are aligned
•   Install Locking Tool KM-853 to camshafts



23.   Tension timing belt
•   Release tensioner bolt, then turn eccentric of tensioner in direction of arrow (clockwise) until pointer is located in correct position
Note:   For new belt, pointer (1) for timing belt tensioner aligns with notched mark (3).
Note:   For used belt, pointer (1) for timing belt tensioner aligns with notched mark (2).
24.   Secure tensioner
•   Secure bolt 20 Nm
•   Remove Locking Tool KM-853
25.   Rotate crankshaft two complete turns in normal direction of rotation to (TDC) on cylinder no. 1, until timing marks are aligned. If pointer of tensioner does not align with corresponding notch mark, then repeat adjustment procedure



26.   Install lower cover of camshaft timing belt
•   Install 2x bolt 6 Nm
27.   Install ribbed V-belt tensioner
•   Install 2x bolt 25 Nm
28.   Install crankshaft pulley
Note:   Note position of crankshaft pulley.
•   Install 4x bolt 20 Nm
– Counter-hold centre nut on crankshaft sprocket
29.   Install ribbed V-belt
•   Use suitable spanner to rotate tensioner clockwise, then remove locking pin
30.   Install upper cover of camshaft timing belt
•   Push on
•   Install 2x bolt 6 Nm
31.   Install intercooler hose to intercooler
•   Secure clip
•   Install 4x new cable tie
32.   Install liner to wheel arch
•   Install 4x screw, 2x clip
33.   Install rear road wheel to right side
•   Install 4x bolt, security bolt
•   Do not tighten
34.   Lower vehicle
35.   Secure rear road wheel right side
•   Tighten 4x bolt, security bolt 90 Nm
36.   Install intercooler hose to throttle body
•   Secure clip
37.   Connect battery negative lead
•   Secure nut and bolt
38.   Install windscreen wash reservoir
•   Push fit
39.   Program all volatile memories
40.   Raise vehicle
41.   Install engine undertray
•   Install 13x bolt, 13x washer
42.   Lower vehicle
