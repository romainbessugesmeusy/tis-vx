Sensor, Crankshaft Pulse, Replace (Z 20 LET)

  Remove

1.   Disconnect crankshaft pulse sensor from engine harness
•   Disconnect harness connector (1)
•   Release 3x clip



Important:   A four-post hoist must be used to raise the vehicle when removing the engine undertray. The vehicle can be moved to a two-post hoist for other operations if required.
2.   Remove engine undertray
•   Raise vehicle
•   Remove 13x bolt, 13x washer
3.   Remove ribbed V-belt
•   Release ribbed V-belt tension
– Using suitable spanner rotate tensioner (1) clockwise, then Install suitable locking pin (2)



4.   Remove hose (3) from turbocharger (2) and intercooler (1)
•   Release 2x clip
•   Remove, then discard cable tie



5.   Remove ribbed V-belt idler pulley (1) from accessory bracket (2)
•   Remove 3x bolt, washer
6.   Remove accessory bracket from engine cylinder block
•   Remove 3x bolt



7.   Remove crankshaft pulse sensor (1) from engine cylinder block
•   Remove bolt



  Install

8.   Install crankshaft pulse sensor to engine cylinder block
•   Install bolt 8 Nm
9.   Install accessory bracket to engine cylinder block
•   Install 4x bolt 35 Nm
10.   Install ribbed V-belt idler pulley to accessory bracket
•   Install 3x bolt, washer 20 Nm
11.   Install hose to turbocharger and intercooler
•   Install new cable tie
•   Secure 2x clip
•   Remove, then discard cable tie
12.   Install ribbed V-belt
•   Using suitable spanner rotate tensioner clockwise, then remove locking pin
13.   Install engine undertray
•   Install 13x bolt, 13x washer
14.   Lower vehicle
15.   Connect crankshaft pulse sensor to engine harness
•   Connect harness connector
•   Secure 3x clip
