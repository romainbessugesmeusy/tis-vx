Balancer Shaft Backlash, Check and Adjust (Z20 LET)

  Remove

1.   Remove upper oil pan, see operation “Oil Pan, Upper, Replace”
2.   Remove oil pick up pipe from engine
•   Remove 2x bolt from oil pump (1)
•   Remove 2x bolt from balancer shaft unit (2)
•   Remove, then discard O ring



3.   Remove front baffle plate from balancer shaft unit
•   Remove 2x bolt (1)



4.   Install Screw Measuring Device KM-949
•   Install long knurled bolt (1) with measuring arm (3) to balancer shaft unit
– Set measuring arm to 9 o'clock position
•   Install short knurled bolt (2) to balancer shaft unit
•   Install DTI gauge KM-798 (4) to measuring arm (3)
– Set dial gauge to zero
5.   Measure balancer shaft unit, tooth backlash
•   Toothed backlash must be measured in 4 different locations
•   Permissible backlash 0.02 mm to 0.06 mm
•   Use knurled bolt (2) to rotate shaft



6.   Remove balancer shaft unit from engine
•   Remove 15x bolt (1)
•   Remove shim



7.   Check identification number against table

•   One shim only to be used

  Install

8.   Install balancer shaft unit to engine
•   Install shim
•   Install 15x bolt 20 Nm + 45°
– Clean all mating surfaces
9.   Install Screw Measuring Device KM-949
•   Install long knurled bolt (1) with measuring arm (3) to balancer shaft unit
– Set measuring arm to 9 o'clock position
•   Install short knurled bolt (2) to balancer shaft unit
– Rotate shaft
•   Install DTI gauge KM-798 (4) to measuring arm (3)
– Set dial gauge to zero
10.   Measure balancer shaft unit, tooth backlash (5)
•   Toothed backlash must be measured in 4 different locations
•   Permissible backlash 0.02 mm to 0.06 mm



11.   Remove Screw Measuring Device KM-949 from balancer shaft unit
12.   Install oil pick up pipe to engine
•   Install 2x bolt to oil pump
•   Install 2x bolt to balancer shaft unit
•   Install new O ring
13.   Install front baffle plate to balancer shaft unit
•   Install 2x bolt 8 Nm
14.   Install upper oil pan, see operation “Oil Pan, Upper, Replace”
