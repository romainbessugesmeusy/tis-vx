Ignition Module, Replace (Z 20 LET)

  Remove

1.   Disconnect harness connector (1) from ignition module
2.   Remove top trim cover (2) from cylinder head
•   Slide off



3.   Remove ignition module from cylinder head
•   Remove 2x bolt (1)



  Install

4.   Install ignition module to cylinder head
•   Install 2x bolt (1), 8 Nm



5.   Install top trim cover (2) to cylinder head
•   Slide on
6.   Connect harness connector (1) to ignition module
