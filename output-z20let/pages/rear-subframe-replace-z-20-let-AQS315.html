Related Literature Units              

Rear Subframe, Replace (Z 20 LET)

  Remove

1.   Remove rear clamshell - see operation “Rear Clamshell, Remove and Install (Z 20 LET)”
2.   Disconnect oxygen sensor from engine harness
•   Release harness from chassis
•   Release 2x clip (2)
•   Release harness from heatshield
•   Release clip (1)



3.   Raise vehicle
4.   Remove secondary catalytic converter (1)
•   Remove 3x front nut (3), then discard
•   Remove 3x rear nut (2), then discard
– Release rubber mount bracket
•   Release rubber mount from bracket
•   Remove gasket (4) from front joint, then discard
Note:   Illustration shows exhaust removed for clarity.



5.   Install Engine Support Tool KM-6054 (1), use with Base Frame Adapter KM-904 (2)
•   Support engine weight



6.   Remove rear chassis heatshield
•   Remove 4x bolt
7.   Remove reaction mount, lower rear from between transmission and chassis
•   Remove centre bolt, nut (1)
•   Remove 3x bolt, washer (2)



8.   Remove engine support tool
9.   Remove rear silencer heatshield from subframe
•   Remove 3x bolt, 3x spring washer, 3x washer, clip
10.   Lower vehicle to suitable working height
11.   Remove rear silencer and heatshield assembly from subframe
•   Remove 6x bolt, 6x spring washer, 6x washer (1)
•   Remove 2x rubber mount (2)
•   Remove 4x bolt
•   Remove 2x clamp



12.   Remove 2x rear road wheel
•   Remove 2x security bolt
– Use special socket located under front bonnet
•   Remove 8x wheel bolt
13.   Remove split pin, hub nut, washer, for left rear wheel
14.   Release left drive shaft from hub, using soft head mallet
15.   Disconnect left ABS harness from ABS sensor
•   Release integral clip (2)
•   Release 2x clip (1)



Important:   Fluid leaks; observe relevant environmental regulations when disposing of used coolant and lubricants.
Important:   Install 2x blanking plug to brake hose and brake pipe end. Do not install previously used blanking plug.
16.   Release flexible hose for left brake caliper from between chassis bracket and brake pipe
•   Release pipe union (1) from flexible hose
•   Remove locking nut (2) from flexible hose (3)
•   Support flexible hose (3) at bracket
– Release flexible hose (3) from bracket
•   Release brake pipe from 2x subframe clip
17.   Release parking brake cable (5) from left brake caliper
18.   Release track control arm (4), left from chassis



19.   Remove rear suspension assembly, left
20.   Remove left front bolt and nut for lower wishbone
•   Release locking tag, then remove bolt, nut
21.   Remove left rear nut and bolt for upper wishbone
•   Release 2x locking tag, then remove bolt, nut
22.   Remove left front nut and bolt for upper wishbone
•   Release 2x locking tag, then remove bolt, nut
Important:   Leave upper bolt in shock absorber to support weight.
23.   Release upper nut and bolt for shock absorber
•   Remove nut
24.   Remove disc hub, shock absorber, track rod, lower wishbone assembly from vehicle
Note:   2x technician required.



25.   Remove left driveshaft from vehicle
26.   Remove right rear disc hub, shock absorber, track control arm, wishbone assembly
•   Follow sequence for left side
Note:   2x technician required.
27.   Lower vehicle
28.   Remove air intake pipe (3)
•   Remove 2x clip
•   Disconnect harness connector (1) from mass air flow meter (2)
•   Release 4x clip
•   Release engine breather pipe
– Release, discard cable tie



29.   Remove air filter cover
•   Release 2x outer clip (1)
•   Release 2x inner clip
30.   Remove air filter element



31.   Remove upper heatshield for exhaust manifold from engine
•   Remove 4x bolt, 4x washer (1)



32.   Remove upper heatshield backplate for exhaust manifold from engine
•   Remove bolt, washer (2)
•   Remove bolt, spring washer (1)
Note:   Includes remove and install right lifting eye.



33.   Disconnect fuel cut-off switch
•   Release harness connector
•   Release engine harness from subframe
•   Release 4x clip
– Position to engine
34.   Release 2x fuel pipe from subframe
•   Release 2x clip
35.   Release coolant expansion tank from support bracket
•   Release clip
– Position to one side
36.   Install engine lift, then support weight of engine
•   Secure 2x chain to 2x lifting eye



37.   Release left, then right roll bar stay from subframe
•   Remove 2x lower bolt, 2x washer (2)
38.   Remove right engine mount from engine
•   Remove 3x bolt (1)
•   Remove damping block locknut
Note:   Damping block locknut can be accessed through opening (2).



39.   Remove left engine mount from engine
•   Remove 3x bolt (1)
•   Remove damping block locknut
Note:   Damping block locknut can be accessed through opening (2).



40.   Remove left then right upper subframe 2x bolt
•   Remove upper 2x bolt, 2x washer
Note:   2x bolt can be accessed through opening (1) on either side of chassis.



41.   Remove subframe (1) from vehicle
•   Remove 2x bolt, 4x washer, 2x locking nut, 2x shim
Note:   Shim can only be fitted one way.
Note:   2x technician required.
42.   Remove fuel cut off switch (2) from rear subframe
•   Remove 2x screw, 2x washer
43.   Remove 5x clip for various pipe, harness and exhaust rubber, dampening clip, 2x bolt



  Install

44.   Install 5x clip for various pipe, harness and exhaust rubber, dampening clip, 2x bolt
45.   Install fuel cut-off switch to rear subframe
•   Install 2x screw, 2x washer
46.   Install subframe to vehicle
•   Install 2x bolt, 2x washer for roll bar stay to subframe, hand tight
•   Install lower 2x bolt, 4x washer, 2x new locking nut, 2x shim, hand tight
•   Install lower 2x bolt, 4x washer, 2x new locking nut, 2x shim, hand tight
•   Clean 2x bolt thread, use recommended locking compound if not micro-encapsulated
Note:   Shim can only be fitted one way.
Note:   2x technician required.
47.   Install 2x upper subframe bolt
•   Install upper 2x bolt, 2x washer, hand tight
Note:   Access for 2x bolt can be achieved through opening on either side of chassis.
•   Clean 2x bolt thread, use recommended locking compound if not micro-encapsulated
48.   Tighten 2x upper subframe bolt 85 Nm
49.   Tighten 2x lower subframe nut and bolt 85 Nm
•   Clean 2x bolt thread, use recommended locking compound if not micro-encapsulated
50.   Install left engine mount to chassis and engine
•   Install 3x bolt 49 Nm
•   Install new damping block nut 48 Nm
Note:   Access for damping block nut can be achieved through opening.
– Recut 3x bolt thread, use recommended locking compound if not micro-encapsulated
51.   Install right engine mount to chassis and engine
•   Recut 3x bolt thread, use recommended locking compound if not micro-encapsulated
•   Install 3x bolt (1) 49 Nm
•   Install new damping block nut 48 Nm
Note:   access for damping block nut can be achieved through opening.
– Recut 3x bolt thread, use recommended locking compound if not micro-encapsulated
52.   Remove engine lift from engine
•   Release 2x chain from 2x lifting eye
53.   Tighten 2x bolt for roll bar stay bolt 45 Nm
•   Clean 2x bolt thread, use recommended locking compound if not micro-encapsulated
54.   Lower vehicle
55.   Install coolant expansion tank to support bracket
•   Install clip
56.   Secure 2x fuel pipe to subframe
•   Secure to 2x clip
57.   Connect engine harness to subframe
•   Secure 4x clip
58.   Connect fuel cut off switch
•   Connect connector
59.   Install heatshield backplate to exhaust manifold
•   Install bolt, washer
•   Install bolt, spring washer
Note:   Includes remove and install right lifting eye.
60.   Install upper heatshield for exhaust manifold
•   Install 4x bolt, 4x washer
61.   Install air filter element
•   Install air filter cover
•   Install 2x outer clip
•   Install 2x inner clip
62.   Install air intake pipe
•   Secure 2x clip
•   Connect harness connector to mass air flow meter
•   Install engine breather
•   Install 4x clip
– Install new cable tie
63.   Raise vehicle to suitable working height
64.   Install left drive shaft
65.   Install left disc hub, shock absorber, track rod, lower wishbone assembly
Note:   2x technician required.
66.   Install new upper bolt, washer and nut 45 Nm for shock absorber
67.   Install front bolt and locking tag for left lower wishbone ( 65 Nm )
•   Position locking tag to bolt head
68.   Install new front bolt, locking tag and locking nut to upper wishbone 45 Nm
•   Position locking tag to bolt head
69.   Install new rear bolt, locking tag and locking nut to upper wishbone 45 Nm
•   Position locking tag to bolt head
Important:   Track control arm locates into chassis. Ensure correct new locking nut is used. Early type have coarse thread, later type have fine thread.
70.   Install track control arm to left chassis
•   Install new locking nut
71.   Secure track control arm 65 Nm
Important:   Fluid leaks; observe relevant environmental regulations when disposing of used coolant and lubricants.
Important:   Remove 2x blanking plug from brake hose and brake pipe.
72.   Install flexible brake hose for left brake caliper
•   Install flexible brake hose to bracket
•   Install flexible brake hose, bracket locking nut
•   Support flexible brake hose
•   Support flexible brake hose
•   Install pipe union
•   Install brake hose to chassis 2x clip
73.   Install brake cable for left caliper
74.   Connect left ABS harness connector to ABS sensor
•   Secure integral clip
•   Secure 2x clip
75.   Install right driveshaft, suspension, brake as per left sequence
76.   Install 2x new hub nut, washer for rear wheel
77.   Bleed brake system - see operation “Brake System, Bleed”
78.   Secure 2x hub nut 235 Nm for rear wheel
•   Install, then secure 2x new split pin
79.   Install 2x rear road wheel
•   Install 8x bolt 90 Nm
•   Install 2x locking bolt 90 Nm
– Use special socket located under front bonnet
80.   Install rear silencer heatshield to lower rear subframe
•   Install 3x bolt, 3x spring washer, 3x washer, clip
81.   Install rear silencer and heatshield assembly to subframe
•   Install 6x bolt, 6x spring washer, 6x washer
•   Install 2x rubber mount
•   Install 4x bolt
•   Install 2x clamp
82.   Install Engine Support Tool KM-6054 , use with Base Frame Adapter KM-904
•   Support engine weight
83.   Install reaction mount, lower rear to between transmission and chassis
•   Install new centre bolt, nut 86 Nm
•   Install 3x new bolt, washer 49 Nm
•   Clean 3x bolt thread, use recommended locking compound if 3x bolt are not micro-encapsulated
Note:   Upper bolt does not have a washer.
84.   Remove engine support tool from vehicle
85.   Check tightness of reaction mount, lower front between transmission and front chassis
•   Centre bolt, nut 86 Nm
•   2x reaction mount bolt, washer 49 Nm
86.   Install secondary catalytic converter
•   Install new gasket
•   Install 3x new nut, front
– Position bracket for rubber mount
•   Install 3x new nut, rear
•   Secure rubber mount to bracket
Note:   Exhaust removed for clarity.
87.   Lower vehicle
88.   Install engine harness to oxygen sensor
•   Install harness to rear silencer heatshield
•   Secure clip
•   Install harness to subframe
– Secure 2x clip
89.   Install clamshell - see operation “Rear Clamshell, Remove and Install (Z 20 LET)”
